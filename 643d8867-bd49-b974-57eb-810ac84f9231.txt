　　第715章
<br/>　　浩甲城不算甚大，却占了一片富饶土地。一条大河绕城而过，也算占了易守难攻四个字。
<br/>　　今日的浩甲城有些异样，街上一个行人也无，长街被打扫的干干净净，街道两旁林立披甲持矛的士兵。
<br/>　　浩甲四风楼，楼高三层，在这个年代搭起这么高的楼已属不易。站在三楼窗前，正好能看到一条大河浩浩荡荡而过。每逢旭日东升，落日西沉都能看到金光点点洒落河面，也算是浩甲城一景。
<br/>　　城中的达官贵人都愿意在此地招待客人，酒楼的大厨是特意从江南请来的，一道无刺鱼，是酒楼的拿手好菜。
<br/>　　今日的四风楼比平时显得清净，楼内打扫干净，外面围满兵丁。都是优中选优，精中选精挑出来的精锐。倾浩甲城一城之力，选出这百八十号精锐士兵也是不易。
<br/>　　在三楼坐着三位客人，整个四风楼今日都在为他们准备。
<br/>　　钱通玉留着两撇胡须，如果不看他身上穿的官服，看上去如一个大财主差不多。其实，钱家本来就是商户出身。往上数几代，钱家上下都是靠往北地贩茶叶丝绸为生。到钱通玉这一代，才算是由商入士，成了官场人物。
<br/>　　今日是他的主场，自然要拿足了气派，不能坠了威风。
<br/>　　“钱兄，为何今日聚会不在你的城主府，反而选在这四风楼？”
<br/>　　问话的是孙别山，他执掌金城，距离此地还有些距离。今日赴会，只带了几名得力的亲信，其实心里也打着鼓。
<br/>　　“喔呵呵。”钱通玉捋了捋胡须，道：“楼中落日，乃是此地一景，我是请几位过来赏景。况且那道无刺鱼，鲜而不腻，也是不能错过。”
<br/>　　武威的白空集心中腹诽，如果只是为了吃鱼和赏景，大家有必要来你这里么。
<br/>　　他看着窗外的滔滔河水，道：“他真的会来么，该不会拿大家耍子吧？”
<br/>　　钱通玉和孙别山都没有开口，心里也无法确认。毕竟来到这浩甲城，程大雷一条命也就不是他的了。
<br/>　　“到这个时间都还没来，说不得便不会来了。”
<br/>　　“哼，他若敢来，便让他知道我大刀的厉害。”白空集冷哼一声，他与宋伯康是结义兄弟，还颇有些交情在。
<br/>　　三人如今都在等，等程大雷的出现。至于程大雷出现后还要不要放他离开，其实心中也拿不准主意。不过，钱通玉已经提前布置下，除了这座四风楼外，周边几条街道都已布下精兵，只要钱通玉一声令下，程大雷便是插翅也难逃。
<br/>　　“那个……”孙别山顿了顿：“从凉州城赶到此地，就算从早上出发，怎么着也得四个时辰吧。算起来，他至少也得下午才能到？”
<br/>　　钱通玉和白空集一顿，二人心中千思百虑，倒是疏忽了这一节。一时之间，都有些尴尬，坐在那里老神自在，闭口不言。
<br/>　　三人百无聊赖的等着，时间一点一滴过去，差不多天尽黄昏时候，远方大河上飘来一叶孤舟，一人摇橹，一人立在船头。
<br/>　　孙别山眼睛一亮：“来了。”
<br/>　　三人同时凑到窗前，极目远眺，钱通玉难以置信道：“当真是一个来的。”
<br/>　　“该不会有诈吧。”白空集心底有些打鼓。
<br/>　　“我常听人说姓程的是位狂人，嗨，今天算是见识了，可真够狂的。”孙别山。
<br/>　　小舟接近港口，三人下楼迎接，离得近了才看清楚。程大雷一身白衣，腰间悬着佩剑，假若捂住脸，倒真有几分出尘气质。他身后跟着一名表情冷淡的少女，作书童打扮。
<br/>　　钱通玉顿了顿，便立刻回过神拱手道：“程当家，久闻大名，今日方才目睹庐山真面目，在下钱通玉，向程当家见礼。”
<br/>　　白空集和孙别山也向程大雷拱了拱手，只是没有钱通玉显得热情。
<br/>　　程大雷双手负在身后，道：“都来了，很好，很好。安排在那里了，过去看看吧。”
<br/>　　说着，程大雷登上码头，与三人擦肩而过。
<br/>　　三人对视一眼，面面相觑，这程大雷根本没有同三人还礼，一副长官的态度，也忒不把三人看在眼里了吧。
<br/>　　压下心中的不快，钱通玉快走几步，在程大雷身前引路。一直来到四风楼前，程大雷停住了脚步，举目四望，楼前围满兵丁。
<br/>　　钱通玉目光闪烁，程大雷该不会起了疑心，不愿上楼吧。
<br/>　　“早听闻四风楼的厨子是从江南请来的名家，今日我算是有口福了。”
<br/>　　钱通玉悬着的心放下来，伸手道：“程当家请，早已安排准备了。”
<br/>　　程大雷迈步上楼，银眸跟在身后，一直到三楼。程大雷自顾自便在主座上坐下，跟在后面的孙别山和白空集看到这一幕，心中都有几分不快。
<br/>　　这是干脆把自己当主人，完全不将自己三人放在眼里啊。
<br/>　　钱通玉悄悄冲二人打了个眼色，二人在才落座。
<br/>　　小二开始络绎不绝上菜，菜过三巡，酒过五味，程大雷偶尔评点饭菜的口味，偶尔侧目赏景，单单不提此行的目的。
<br/>　　孙别山是苦出身，摸爬滚打一步一步走到今天的位置，他耐不住性子，道：“程当家此来，不单单是为了喝酒吃肉吧？”
<br/>　　程大雷这才停住筷子，道：“请讲？”
<br/>　　孙别山深吸一口气，道：“程当家的来意我们都知道，某家是个直性子，也不藏着掖着了。让俺们和你蛤蟆寨互不侵犯可以，但若说让我归附蛤蟆寨万万不能。”
<br/>　　说罢，孙别山盯着程大雷的双目，程大雷目光不闪不必，最后反倒是孙别山悄悄侧过头去。
<br/>　　“我相信这话也是白城主与钱城主的意思？”
<br/>　　二人没有否认，大概便是等于承认。
<br/>　　程大雷长长叹口气，道：“我们同在凉州混饭吃，也算是邻居，俗话说远亲不如近邻，我好心好意指给三位一条生路，可三位明显是没把我当自己人呐？”
<br/>　　“不知此话怎讲？”钱通玉。
<br/>　　“哼，装神弄鬼，真以为我们是吓大的么？”白空集冷哼一声，不动神色的摁住刀柄。
<br/>(本章完)
