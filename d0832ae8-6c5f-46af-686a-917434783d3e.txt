　　第150章
<br/>　　裴喜之下意识后退一步，是不是有什么奇怪的东西混进来，周围气氛突然变得怪异起来。
<br/>　　“阁下是？”裴喜之盯着面前的猥琐男人，感觉有些不舒服。
<br/>　　“兄弟我是程将军座下徐神机，当然都是为幽州王办事。”
<br/>　　徐神机这个名字裴喜之是听说过的，来之前他已经对蛤蟆寨做过一定调查。据说此人老奸巨猾，诡计多端，乃是程大雷的左膀右臂，程大雷一多半坏主意都是他出的。
<br/>　　可自己并没有看出他足智多谋在何处，弓腰，垂首，脸上堆满笑容，眼神闪烁里面藏着的应该是……谄媚。
<br/>　　对了，就是谄媚，甚至还有小小的嫉妒。毕竟大家都是靠脑袋吃饭，为何自己在幽州王座下光宗耀祖，你却只能跟着一个山贼混，有些羡慕嫉妒恨也是理所应当的。
<br/>　　如此，裴喜之自觉判断出蛤蟆寨对自己的态度，之前的担忧完全是杞人忧天。
<br/>　　他轻咳一声，昂首挺胸道：“头前带路吧。”
<br/>　　路上，虽然徐神机极尽谄媚之能事，但裴喜之都刻意扮上一副冷漠面孔，与对方保持着距离。纵然，这次要示好蛤蟆寨，可若对蛤蟆寨太过客气，对方往往会得寸进尺。对于这帮山贼，还是打一棒子给个甜枣好。
<br/>　　对方既然是这样态度，看来这次任务便不会有太多难度。也不知到了山寨，对方会怎样招待自己。说不得为了让自己替他们在幽州王面前美言几句，他们还会送上些银子、女人什么的。
<br/>　　如此，这次任务，可能还有些油水捞。
<br/>　　徐神机将他们引到飞虎寨。
<br/>　　“我这次来，是尊幽州王的命令，来看看蛤蟆寨的情况。当然，是好是坏，也就是兄弟一句话。”
<br/>　　“明白，明白。”徐神机道：“裴大人一路辛苦，肯定还没有用饭吧，为了招待裴大人，饭菜我们都已经准备好了。”
<br/>　　裴喜之微微颌首，话已经点给你们了，接下来该怎么做，就看你们懂不懂事了。
<br/>　　长条桌上，裴喜之等人坐在主位，徐神机以及十七八个山贼陪座。
<br/>　　看着摆在自己面前的饭菜，裴喜之陷入久久的无语中。那盆飘着树根的东西是叫做汤么，还有面前这黑不溜秋石头状的东西，是主食么。
<br/>　　“咱们就吃这个？”裴喜之难以置信道。想象中的鸡鸭鱼肉呢，想象中有美女陪着唱小曲，一边喝酒一边欣赏歌舞呢。
<br/>　　“哈，裴大人有口福了，这些平时我们都吃不到，喂，你们几个今天也沾裴大人光，都给我注意点，不要吃得狼吞虎咽。”
<br/>　　“……”裴喜之。
<br/>　　好吧，好吧，吃饭是小事，看来蛤蟆寨是真够穷的，他们没有把人肉端上来自己已经该庆幸了不是么，尝尝他们平日吃什么也是好的嘛。
<br/>　　“住手！”
<br/>　　裴喜之刚想拿齐那野菜、草面混合在一起的食物时，被徐神机喊住。裴喜之吓了一跳，怎么就连这也不给吃。
<br/>　　“裴大人不要着急，还没有祷告呢。”
<br/>　　“祷告……是个什么东东？”
<br/>　　就见徐神机几个双手合十，虔诚面对桌上的食物，嘴里嘀嘀咕咕说着什么。
<br/>　　“感谢幽州王赐予我们的食物，幽州王的光辉普照世间。”
<br/>　　“徐军师，你们这是？”
<br/>　　“哈，这是我们每天饭前必做的，我们寨主十分崇拜幽州王，在卧室都挂着幽州王都画像，早晚三叩首，晨起先上香。怎么，裴大人饭前不用祷告么？”
<br/>　　“我们那里……倒是没有这个规矩。”
<br/>　　一顿饭不知怎么吃完了，如果说自己吃的是猪食，裴喜之都觉得是对猪食的侮辱。
<br/>　　徐神机打了个饱嗝，道：“大家现在虽然吃得不好，但这是成功者的必须经历的，那句话怎么说来着，不吃苦吃苦一辈子，肯吃苦吃苦一阵子。遥想当年的幽州王也是从底层一点点干起，最后成为我们崇拜的幽州王。”
<br/>　　徐神机言必提幽州王，语气眼神都极其恭敬。而裴喜之发现，只要听到幽州王这三个字时，其他人不管在做什么，都会停下手中事，抬头挺胸，目视前方。
<br/>　　搞得裴喜之很不好意思。
<br/>　　“今天有新弟兄加入我们这个大家庭，大家鼓鼓掌，请我们的新弟兄讲讲幽州王的光荣往事。”
<br/>　　众人目光落在裴喜之身上，裴喜之睁大眼睛：“有这个必要么？”
<br/>　　“难道裴大人不尊敬幽州王，难道不是发自肺腑的崇拜他，难道不是把他当作心中的光明？”
<br/>　　“这个……”好像自己无法说不。
<br/>　　“看来裴大人还有些不好意思，大人你要勇于表现自己啊。这样，就让裴大人先准备一下，那个兄弟先说一下。”
<br/>　　“我先来！”一个山贼腾地站起来了，其他人显得特别遗憾：怎么被他抢了先。
<br/>　　只见这个山贼满脸激动的从怀里拿出一张纸，用充满深情的咏叹调朗诵道：“啊，我有一个梦想，就是成为幽州王那样的男人。遥想当年，是谁金戈铁马中杀敌如麻，是谁废寝忘食爱民如子，是谁夙兴夜寐，不忘家园……啊，是幽州王，是幽州王啊！”
<br/>　　“像离群的羊羔向往羊群，像迷途的航船渴望归乡，向初生的婴儿渴望母亲的**，啊，幽州王您就是羊群，您就是故乡，您就是**……”
<br/>　　裴喜之几个人坐在那里，一脸错愕，只见一个个山贼站起身，用令人肉麻的语调朗诵着这些东西，直让人浑身起鸡皮疙瘩。
<br/>　　“裴大人，准备一下，下个该你了。”徐神机低声道。
<br/>　　“我也要来……”
<br/>　　“嗯，我们这里是五千字起步的。”
<br/>　　“五千字！”
<br/>　　……
<br/>　　山谷内，蛤蟆寨。
<br/>　　程大雷站在高处查看周围的地形，刘悲站在他旁边。
<br/>　　“你真的决定用这种法子对付幽州派来的人？”刘悲。
<br/>　　“我有什么法子，现在还不是和幽州王撕破脸皮的时候。”程大雷。
<br/>　　“可我觉得瞒不住幽州王的。”
<br/>　　“能瞒一日是一日吧，终有互相呲出牙齿的时候，谁让我们现在的牙齿还没幽州王硬。”
<br/>　　程大雷极目望去，山谷内的情形一览无遗。
