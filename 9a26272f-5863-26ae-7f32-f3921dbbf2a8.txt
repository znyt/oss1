        怪不得她会被酒吧里这群“败家子”给盯上。
<br/>
<br/>        “大哥，你刚刚去哪儿呢？让我好找啊！”女孩仰着脑袋，一脸真诚而委屈地看着傅夜沉，随之紧紧地挽住了他的胳膊，不愿放手。
<br/>
<br/>        傅夜沉神情冷凛地注视着她。
<br/>
<br/>        怎么搞的？他为什么无动于衷？
<br/>
<br/>        “大哥，你这是怎么呢？”女孩焦虑地皱起了眉头。
<br/>
<br/>        傅夜沉微微垂眸，冷淡地看了那双紧攀着他的纤白小手一眼，像是听不懂她求救的低语般，默不吭声。
<br/>
<br/>        他正用一种很冷漠的神情看着她。
<br/>
<br/>        外出找艳遇、到酒吧钓凯子的女人，他碰过不少。
<br/>
<br/>        五花八门的搭讪方式，更是层出不穷。
<br/>
<br/>        这个女孩十有八九，亦是如此吧！
<br/>
<br/>        刹那间，女孩的表情僵住，美丽的眼睛闪过一丝诧异，忐忑不安地接着说道：“大哥，我真的一直在找你，爸爸妈妈说，让我找到你后，让你跟我一起回家。”
<br/>
<br/>        “……”傅夜沉无语了。
<br/>
<br/>        “大哥？”女孩焦急地又唤了声。
<br/>
<br/>        不会吧？他是个哑巴？还是他仍然听不懂她的意思？完了，这下真的完了！
<br/>
<br/>        傅夜沉觑着她，眼底有一种意味不明的淡漠。
<br/>
<br/>        “请问我们认识吗？”他终于开口了。
<br/>
<br/>        尽管他的口气是那么的冷漠，大而化之的女孩还是因他有了反应而喜形于色。
<br/>
<br/>        她凑到他跟前，双手不自觉地紧紧握住他的手臂：“大哥，你不要再生我的气了，好不好？”
<br/>
<br/>        傅夜沉轻轻地瞥了她一眼，她是长得很漂亮，言行举止，也确实很像筱颖，但是他真的对筱颖以外的女人不感兴趣。
<br/>
<br/>        “对不起，我没兴趣管别人的闲事。”
<br/>
<br/>        女孩瞪大眼，愣了一下后，只见她美丽的脸蛋扭曲变形，且跳了起来。
<br/>
<br/>        “你……你居然见死不救？你是一个胆小鬼，你怕事、懦弱、没有同情心，我告诉你，过了今晚，你会一辈子遭受良心的谴责！”
<br/>
<br/>        听着她噼哩啪啦地滔滔不绝，傅夜沉一脸不悦地挑了挑眉。
<br/>
<br/>        他从始至终都在怀疑，这个女孩是否对他别有用心。
<br/>
<br/>        “美女，你在摩蹭什么？”一个醉醺醺的男人冷不防地，凑到女孩背后，“陪我们去乐一乐！”
<br/>
<br/>        他粗鲁地搂住女孩，将女孩从傅夜沉的跟前给强行拉走。
<br/>
<br/>        女孩头皮一阵发麻，开始惊叫着挣扎起来：“放开我！快点放开我，你们干什么啦，快点放开！”
<br/>
<br/>        醉醺醺的男人压根就不把她的话当回事，只当她是玩欲迎还拒的把戏，在外人面前做做样子罢了。
<br/>
<br/>        “好啦，不要装了，这实在太浪费时间了，走吧，我保证待会绝对让你销魂！”几个男人放声大笑。
<br/>
<br/>        “放手啊！救命！”女孩尖叫，回头频望着傅夜沉并尖锐地喊，“大哥，求求你救救我！大哥，求求你，你若不救我，我就自杀给你看，做鬼也要向你讨命！”
<br/>
<br/>        傅夜沉身心一怔，不自觉地抬眸看着被那几个男人拉走的她，他看她的表情不像在说谎……
<br/>
<br/>        终究，他还是挺身而出，往抓住那女孩的那个男人跟前走去，手指头在他的肩上点了点。
<br/>
<br/>        “干什么？！”
<br/>
<br/>        男人不耐烦地回头，但瞧清楚背后所站的人后，立即吓得马上噤声，就连其他看热闹的酒客都安静了下来。
<br/>
<br/>        “我劝你不要多管闲事！”其中一个微胖的男人提着胆子呵斥道。
<br/>
<br/>        “她是我妹妹。”傅夜沉伸手握住女孩的手，将她由那个男人的怀中拉了出来，用力带往自己身边靠近。
<br/>
<br/>        酒吧里顿时响起一阵暧昧叫嚣。
<br/>
<br/>        之前抓住女孩的那个男人眼睁睁的看着到手的肥肉又被人抢走，不禁有些恼火起来。
<br/>
<br/>        女孩视线由下往上瞧，心脏怦怦地跳。
<br/>
<br/>        哇塞！这个男人好高！站在一群坏男人中，还高出了一些，她目测他至少有一九○公分。
<br/>
<br/>        女孩盯着傅夜沉看的眼瞳发亮，她站直了身体还不及他的肩膀，所以她只能看到他刚毅有型的下巴，以及听到他浑厚的嗓音。
<br/>
<br/>        她发现他真的好帅，除了迷人如黑钻般的眼瞳外，他那头闪闪发亮的黑发也很迷人，而且更不用提他高挺的鼻梁和紧抿的薄唇，虽然有点儿霸气侧漏，却让人无法忽视。
<br/>
<br/>        “走吧！我们回家！”傅夜沉连看都不看她一眼，就拖着女孩往外走。
<br/>
<br/>        那几个男人似乎不甘心自己看中的猎物，就被这么一个来历不明的男人给截胡了，于是随手操起一旁圆桌上的啤酒瓶，气势汹汹地尾随傅夜沉而去。
<br/>
<br/>        “啊——大哥，小心！”女孩忽然尖叫了起来。
