　　第536章
<br/>　　天亮的时候，黄家已成了一片废墟。
<br/>　　黄家上下八十余口，没有一个能活过昨夜。断壁残垣，大屋倒塌，地上倒着一具具尸体。
<br/>　　奇怪的是，他们的脑袋都被砍去，院子里搁着的，是一具具没有头颅的腔子。
<br/>　　脑袋是方石隆砍走的，黄家勾结巨贼程大雷，满门被杀，报上去也是一份不小的功劳。谎报战功这种事，在各地都不算稀罕事，何况这次杀的，的确有鹰嘴岭的山贼。
<br/>　　黄土镇昨天死的人不少，除了罪有应得的黄家外，其他还有一些平民卷入这场杀戮中。
<br/>　　天亮了，官兵和土匪都消失了，他们才敢冒出头，埋葬亲人的尸体，收拾坍塌的房屋。
<br/>　　远远的，有两个女人缓缓行过来。二人都穿着粗布衣服，打扮不算精致。其中一个年级稍长些，露出些丰腴仪态，手中拖着拂尘，看着像个道姑。
<br/>　　另外一个身穿黑色袍子，脸上淡淡愁容。
<br/>　　相同点是，二人都很漂亮。
<br/>　　这就透着些奇怪，在这世道，女人，尤其是年轻漂亮的女子轻易不出门，出门就代表着危险，如果不得不抛头露面，也会故意扮丑。
<br/>　　道士和尚女人小孩，轻易不会出现在江湖上。可一旦这四类人在江湖上行走，多少便有些自保的本事。
<br/>　　但黄土镇不是大地方，这里的人也未必有多少见识。有几个闲汉混混蹲在墙角晒太阳，窥见这一幕大咧咧的吹着口哨，有几个胆子大的一开始风言风语。
<br/>　　道姑女人出手几枚铜钱，隔着十步远敲在他们嘴上，有人被击落门牙，其中一个人，一只眼睛自己被打瞎。
<br/>　　几个人惨叫连连，疼地在地上打滚，但这一幕，没能让道姑皱皱眉头，她仍旧愣着一张脸，用手托着拂尘。
<br/>　　这二人自然就是从蛤蟆城出来的李婉儿和李莫愁。
<br/>　　李婉儿轻轻撇撇嘴，却也没有说什么。
<br/>　　继续向前走着，便走到黄家的宅子外，看着面前的废墟，她下意识颦起眉头。
<br/>　　一路从蛤蟆城来到这里，走了帝国很多地方，也见过许多惨象。她本该见怪不怪，但今日的情景……的确忒惨了些。
<br/>　　“请问老伯，这里是怎么会事？”
<br/>　　李婉儿问一个想从黄家废墟中拾一些盆碟来用的老汉。
<br/>　　“唉，别提了，昨天晚上先来了山贼，又来了官兵，最后黄家人都死绝了。唉，可怜黄老爷一辈子行善，是这十里八乡有名的善人。”老汉说着将以个白瓷盘子擦净了，揣在怀里。
<br/>　　李婉儿看着眼前的惨状，呐呐叹了一句：“为何会这样？”
<br/>　　自从离了蛤蟆城后，她便见过太多不好的事。家破人亡，流离失所，卖儿卖女……
<br/>　　从京州到草原，从草原到蛤蟆城，李婉儿也算是吃过一些苦。可当真正走出蛤蟆城，她才明白普通人的生活不是苦，而是惨。
<br/>　　原来在李家的统治下，普通人过的是如此生活。
<br/>　　如此思来，天下人人不满李家的统治，却也不是没有理由。
<br/>　　“还是将他们安葬了好，总不能死了还不得安生。”李婉儿已挽起袖子。
<br/>　　李莫愁不是很愿意搭理她，如果她不是奉了程大雷的命令，早就弃李婉儿而去。
<br/>　　“好啊，这里有几十具尸体，看你要忙活多久？”
<br/>　　李婉儿没有太多反映，挽起袖子，捡了一柄铁锹，开始清理黄宅废墟上的尸体。
<br/>　　李莫愁半点没有帮忙的意思，但李婉儿这样做下去，还不知要耽搁多久。
<br/>　　“如果你真想做些事，是不是死人比活人更加重要，何必在死人身上耽搁时间。”
<br/>　　李婉儿依旧固执的做着自己事，动作不快，当然也快不起来。
<br/>　　李莫愁颇有些无可奈何：“你想要做事，但总要想想这事有什么意义。你一无财，二无武艺，手下也没人。想要做事，总有有些自己的本钱。喂，听说你六哥就在附近，我们是不是该去看看她。”
<br/>　　这些话从李莫愁口中说出，似乎接下来就被风吹散，李婉儿仍旧没有停下手中的动作。
<br/>　　“喂，小妮子，这铁锹是我先找到的，你得还给我。”刚才的老汉道。
<br/>　　李婉儿手中的铁锹被要走了，李莫愁下意识笑出声来：看你两手空空，还能做什么。
<br/>　　李婉儿无奈的笑笑，从怀里掏出一锭银子。
<br/>　　“这些给你，你帮我埋了他们，之后我再给你已锭。”
<br/>　　……
<br/>　　徐州沿海，一场战斗已接近尾声。
<br/>　　“逃啊！”
<br/>　　“老李，你等一等我，拉兄弟一把呀。”
<br/>　　“快点逃兵吧，官兵太凶了。”
<br/>　　老疤是海上一伙海盗的头目，势力并不大，约莫只有五六十人。过往的大型商船他们不会劫，当然也劫不了。也就是上岸抢一下平民富户，在官兵没发现前洗劫一番。
<br/>　　落凤城本来相当于他们后花园般的存在，落凤城没有什么兵，他们想来就来，想走就走。
<br/>　　但近些日子，落凤城有些不一样了。城里来了个了不得的人物，见到海盗就杀，附近几伙海盗都遭了他们毒手。
<br/>　　今日老疤不小心，也与落凤城的兵碰上，结果就只有撒丫子跑一条路。
<br/>　　身后的李行哉提着大枪，深一脚浅一脚的往前冲。
<br/>　　“杀，弟兄们与我杀。”他大吼着：“杀他娘，抢他娘，海盗来了咱们才有饭吃。”
<br/>　　老疤心头直骂娘：“这谁才是海盗呐！”
<br/>　　老疤一伙人被围在浅滩上，带出来的弟兄死了大半。看着身边几个伤痕累累的弟兄，老疤哇呀大叫一声，拎着刀冲了过来。
<br/>　　“我和你们拼了。”
<br/>　　一记画龙点睛，长枪直直刺出，串了老疤的喉咙。手下兵丁一拥而上，将残存的海盗赶尽杀绝。
<br/>　　“杀，杀了他们。看他们身上有没有什么值钱的东西。”李行哉雄姿英发，这种抢东西的感觉委实不错。
<br/>　　“王上！”
<br/>　　一匹快马奔来，戚继光从马上跃下来，将手中一封书信递过去。
<br/>　　“这是刚刚得到的消息，王上请过目。”
<br/>　　李行哉打开信封，粗略扫了一眼，嘴角轻轻翘起。
<br/>　　“来了，他来了。”
<br/>(本章完)
