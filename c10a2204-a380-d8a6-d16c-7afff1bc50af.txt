　　足足愣了好一会，叶尘还是没有想明白其中的缘由，
<br/>　　这女人，之前明明还好好的，怎么说翻脸就翻脸？
<br/>　　难道是因为我之前出手，让她丢了颜面，所以就给我脸色看？
<br/>　　就在叶尘百思不得其解之时，云破天又恨恨的望了叶尘一眼，最终乖乖来到那中央位置，从剩下的几个修炼道场之中，挑选了一个位置，落在其中。
<br/>　　时间在一点点流逝，很快就又有人陆续来到第七重星域。
<br/>　　第五个来到此地的，赫然是乐无忌。
<br/>　　当乐无忌看到，叶尘坐在最顶端的位置，显然也颇感意外，不过他似乎并没有要和叶尘争夺位置的意思，反而冲着叶尘露出一脸温和的笑容，
<br/>　　“叶兄弟，之前多谢你将冰甲猛犸象的妖丹让给我，等出了迷罗星海，兄弟我一定另有重谢！”
<br/>　　叶尘对这个乐无忌的印象，倒也算不错，于是微微一笑道：
<br/>　　“乐兄客气了！咱们是公平交易，无需言谢！”
<br/>　　乐无忌哈哈一笑，道：
<br/>　　“叶兄弟明明将妖丹让给我了，却还能在我之前赶到第七重星域，真是令人佩服直至！”
<br/>　　其余几人听到两人的对话，除了洛冰玄知道事情的真相之外，另外三人全都纷纷露出一抹震惊之色。
<br/>　　听乐无忌这话的意思，之前叶尘竟然将妖丹卖给了乐无忌！
<br/>　　而且因为这个原因，应该耽误了不少时间，没想到最终却还是走到了乐无忌的前头，甚至仅仅只比龙辰落后了一小段时间而已！
<br/>　　这就不得不令人，重新去思考这小子的能力了！
<br/>　　尤其是一直神情都无比淡然的龙辰，也眼眸深处也不由得露出一抹浓浓的凝重之色。
<br/>　　虽然叶尘的表现，有些出乎他的意料之外，可在他看来，应该还在可控的范围之内。
<br/>　　可是如今乐无忌的一番话，再加上之前云破天和叶尘、洛冰玄之前的恩怨纠葛，他大致可以得出一个结论。
<br/>　　叶尘在第四重星域，应该耽搁了很长一段时间，倘若他当初没有搭救洛冰玄，没有将冰甲猛犸象的妖丹卖给乐无忌，那他岂不是要比自己的闯关速度还要快很多？
<br/>　　一念及此，一直胸有成竹的龙辰，顿时也有些不淡定了，甚至在眼眸深处，隐隐浮现出一抹杀意，不过很快便一闪即逝。
<br/>　　乐无忌才刚刚落座之后不久，
<br/>　　轰隆！
<br/>　　轰隆！
<br/>　　居然一连两次巨震，几乎同时从远处传来。
<br/>　　很显然，
<br/>　　这一次，居然同时来了两个人！
<br/>　　很快，
<br/>　　那两道人影，便出现在了众人跟前。
<br/>　　这是一男一女，男的身形魁梧，身上煞气冲天，女人妖娆多姿，身上透着妩媚之气。
<br/>　　这一男一女，赫然正是那十大天骄中的另外两位，绝无敌和风华舞！
<br/>　　至此，这次迷罗星海开启，十大天骄中来了七位，此时已经全部聚齐。
<br/>　　等到两人来到跟前，看到龙辰、洛冰玄等几人，虽然感觉有些挫败，不过却也并不怎么惊讶。
<br/>　　毕竟眼前的这些人，排名本来就在他们之上，走在他们两个的前面，倒也不是不能接受。
<br/>　　可是当两人看到，与龙辰、洛冰玄并驾齐驱的，不是云破天或者燕狂生，反而是名不见经传的叶尘，两人顿时全都愣住了。
<br/>　　尤其是绝无敌，两眼死死盯着叶尘片刻，几乎都恨不得要把他生吞活剥了！
<br/>　　“小子！就凭你，也配和龙老大、冰玄仙子并列？”
<br/>　　叶尘瞥了一眼绝无敌，冷冷道：
<br/>　　“我不配，难道你就配吗？”
<br/>　　其实叶尘心底，是期盼着绝无敌挑战自己的，如此一来，他便可以趁机将此人一举斩杀，或者将他赶出迷罗星海！
<br/>　　不料，绝无敌眉头微微一皱，望了一眼下方的云破天和燕狂生，似乎想到了什么，最终竟然没有再开口反驳，反而乖乖落在了那第二等级的修炼道场之中，显然是直接放弃了。
<br/>　　这个结果，倒是有些出乎叶尘的意料之外，让人微微有些失望。
<br/>　　绝无敌若是主动挑战他，他也不好自己对此人出手，否则以龙辰的尿性，多半又要出手多管闲事。
<br/>　　而且，他之前先是得罪了燕狂生和云破天，甚至还有那龙辰，叶尘也搞不清楚，此人的真实想法。
<br/>　　若是他不顾一切，悍然对绝无敌出手，难保这些人，不会联合起来，先解决掉自己这个黑马！
<br/>　　所以，经过一番思量之后，叶尘决定还是先按兵不动。
<br/>　　随着绝无敌和风华舞到来，之后众人一连等了足足两天，终于在第三天的时候，才陆陆续续有人继续感到第七重星域。
<br/>　　由此也可以从侧面看出，那些寻常的闯关者们，虽然也都是各大门派的天才人物，可是和十大天骄相比，终究还是有着不可逾越的巨大差距！
<br/>　　一转眼，时间已经过去了十三天！
<br/>　　叶尘此时的修为，已经恢复到了元婴境六重，和龙辰旗鼓相当。
<br/>　　而就在第三十三人，踏入这第七重星域之时，九天之上突然响起一道浑厚淡漠的声音，
<br/>　　“恭喜诸位！本次所有的闯关者，已经全部到齐，通天塔马上即将开启，请诸位到通天塔前集合，马上宣布本关的通关规则！”
<br/>　　唰唰唰！
<br/>　　随着这道声音响起，原本高台上正在修炼的那些人，顿时全都纷纷跳了下来，迅速来到那座雄伟的高塔之前。
<br/>　　最初进入迷罗星海的，一共是整整一百人，到现在便只剩下只有三十多人，其余的六十多人，显然不是死了，就是捏碎玉简，已经离开了迷罗星海。
<br/>　　众人在通天塔前集合完毕之后，那道淡漠的声音，顿时便再次响起，
<br/>　　“诸位，眼前这座通天塔，一共是十八层，每一层都会有一个守关者，闯过的关卡越多，成绩也就越高，而你们这些人里面，最终只有前八名，可以进入第八重星域……”
<br/>　　随着这道声音响起，周围那些普通天才们，顿时便开始低声议论起来，
<br/>　　“这么说来，我们岂不是都没有机会了？凭我们，拿什么和十大天骄去争？”
<br/>　　这时，那道淡漠的声音再次响起，
<br/>　　“诸位！每一层的守关者，会根据你们当下的修为，将自身的实力，压缩到一定的比例，所以，当前修为越高的，反而越吃亏！”
