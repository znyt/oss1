　　大食水军在短短的瞬间就被打懵了，看着自方的战船不知怎么就炸成碎片的感觉让所有大食人全都有些胆战心惊。
<br/>　　曾经他们也跟大唐水师打过一次，不过那一次大唐水师虽然胜了，但自身也有不小的损失。
<br/>　　这一次大食人多长了一个心眼儿，特地弄了以前两倍的战船，而且也将床弩什么的东西装到了船上，打算给大唐来个措手不及。
<br/>　　结果没想到，大唐又改变了作战方式，不单单是战舰换成了钢铁战舰，而且船上还带着那种让人一听到声音觉得大难临头的武器。
<br/>　　是的，以数年前，大唐水师并没有装备太多的火药武器，甚至就算是‘海狼’也只装备了一些简单的火箭弹。毕竟那时候的火药武器全都是黑火药，战船也都是木质战船，水师根本没有能力保证那些火药不受潮。
<br/>　　可是现在不同了，一体式的弹药被发明出来之后，起到了有效的防潮作用，重新设计过的钢铁战舰也准备了专门的防潮室用来储存弹药。
<br/>　　这样一来，大唐水师等于全部换装，从战舰到武器没有任何一样是大食人所熟悉的装备，吃亏也就成了再正常不过的事情。
<br/>　　而反观大唐南海水师，高高的船舷对那些矮小的大食战船足以形成俯视的角度，这使得那些大食人进攻变的困难了许多，大多数的攻击都被高高的船舷挡住，根本无法对钢铁战舰上的水师官兵形成威胁。
<br/>　　而另一方面，战舰上的火炮或者说舰炮却可以居高临下的一个又一个定点清除那些漂在海面上的大食战船，只要一炮砸中了，立刻就会使大食人的战船分崩离析，变成漂在海上的破木头。
<br/>　　甚至尉迟宝琪在看了两轮齐射之后，觉得没什么意思，突发奇想搞起了大练兵。
<br/>　　五艘战舰互相比赛，看准打的准，看准打的多。
<br/>　　于是乎五舰钢铁战舰并驾齐驱，向着大食人的船队碾压了过去，对着他们的战船展开了定点清除。
<br/>　　一炮打出去，歪了？不要紧，调整角度总结经验，弄好了之后再一炮！每艘战舰上十余门火炮，你方唱罢我登场轮流向大食的战船发起进攻。
<br/>　　大食人的将军此时也有些懵圈，他万万没想到大唐的战船竟然是铁制的，他的攻击落到上面根本起不到任何一点作用。
<br/>　　至于说常用的跳帮作战……，别提了，人家大唐的船比自己的船不知高出多少倍，就算是已方的战船能够顶着那莫名其妙的武器冲上去，到了人家船边上照样傻眼。
<br/>　　而且最重要的要还是靠不过去，大唐人不知道使用的是什么武器，尤其是那个担着白烟到处乱飞的那种，简直就是大食战船的噩梦，凡是靠近到大唐战船附近的大食船只，就没有不被那东西打爆的。
<br/>　　远程攻击没有用，跳帮作战又没有办法靠近大唐的战船，办法想了无数，自己这边的船却越打越少。
<br/>　　眼看着再这样下去，就只有全军覆没一条路可以走，大食水军的将军果断下令——撤！
<br/>　　是的，撤！老子打不过你，跑还不行么？
<br/>　　可是让他十分绝望的是，逃跑对于他来说同样是一种奢望，大唐的钢铁战舰似乎并不像他想象中的那么慢，形成追逐之后，他就发现自己似乎一直就处在被动挨打的位置。
<br/>　　那些可恶的大唐战船一没有船桨，二没有风帆，可偏偏跑的比谁都快，甚至他还可以看到大唐战舰破开海浪将海水劈开的样子。
<br/>　　那种不知道名字的特殊武器声音依旧从身后不断传来，听上去距离不远又不近，完全就是一副赶尽杀绝的意思。
<br/>　　跑不掉，躲不开，打不动，大食人这次是真的要绝望了，拼了老命的让战船加速，可是毛用没有，自方的战船依旧像是在被点名一样一艘艘被打爆。
<br/>　　再看南海水师，这帮犊子似乎是越打越顺手了，那炮打的极准，几乎两发炮弹必然有一发会命中，然后他们就会调转炮口去找下一个目标。
<br/>　　一群大食战船数百艘，却被大唐五艘战舰像撵鸭子一样，追的满世界乱窜。
<br/>　　还有一些大食商人，本来以为自己的军队人多势众指望着他们能够打开海峡，然后就可以跟在军队的后面去大唐做一回生意。
<br/>　　结果没想到自己国家的军队不争气，几乎刚一接触就被打的溃不成军，还不如英勇一些的海盗。
<br/>　　这还是当初那支彪悍的军队么？这还是那只就算是面对战神也敢举起手中战刀的军队么？他们的勇气在哪里？他们的决心在哪里？
<br/>　　败了，大食的水军再一次败在了唐人的手中，大唐就像是一个无法打破的梦魇，使得大食人无论如何都无法战胜。
<br/>　　不过这个想法才刚刚产生，这些大食人商人就意识到，自己似乎犯了一个很大的错误，既然那些战船都敌不过大唐战舰，那么自己这些商船还留在这里干什么？等死么？
<br/>　　想通了这个问题，大批的商船开始掉头。
<br/>　　可是商船到底是商船，和战船是没办法比拟的，他们几乎是等于帮着大唐拦住了大食水军的退路，直接将回去的航道全部堵死了。
<br/>　　大食水军的将军气的几乎想要杀人，可杀人又能怎么样？根本解决不了实际问题，就算是把那些商人都杀光了，航道该堵还是堵着的。
<br/>　　什么？你说绕开？绕开的确可以，可是大唐的战舰明显速度比他们要快，本来就跑不过人家，如果再绕一点路，那岂不是跑的更慢？
<br/>　　前无去路，后有追兵，绝望的大食将军终于吃不住劲儿了，命人打起白旗，宣布投降！
<br/>　　可就算是这样，尉迟宝琪依旧是又干掉了他们五十来艘战船才算是彻底罢休。
<br/>　　一场拖了长达半年之久的战争，就这样迅速的开始，又诡异的结束。
<br/>　　大食人损失了近百艘战船，被俘虏了数万人，以全军覆没为结果，完全成了他们最后的使命！
