　　半个时辰后，在易老的带领下，三百人，往进行第三个考核的地方而去——武道碑林。
<br/>　　武道碑林，位于裂天剑派最深处，平日里乃是剑派禁地，也是剑派最大的底蕴之一。
<br/>　　论重要程度，甚至在寒冰池之上。
<br/>　　不多时，众人来到武道碑林。
<br/>　　映入眼帘的，是上百个巨大的青色石碑。
<br/>　　有大有小，大的，高数十米，小的，只有数米。
<br/>　　靠近些，一股古朴而威严的气势扑面而来，让人神色一凛。
<br/>　　同一时间，十道身影，忽然凭空而现，站到了众人面前。
<br/>　　这是十个中年人，尽皆气势不凡。
<br/>　　观他们的气息，怕是最弱都在洞玄八重以上。
<br/>　　“易老。”十人出现后，恭敬地对着易老行了一礼。
<br/>　　“嗯。”易老点点头，而后转过身，对众人说道，“这十位，乃是剑派的内门执事。”
<br/>　　“尽皆修为不俗，武道知识丰富，同时也是内门中负责教导弟子武道修炼的前辈。”
<br/>　　“你们是否有机会在日后接受他们的指导，全凭这最后一轮考核。”
<br/>　　易老说罢，三百武者同时对着十位内门执事鞠了一躬，齐声道，“参见执事。”
<br/>　　“免礼。”十人点点头，打量了众人一眼，而后都露出了满意的笑容。
<br/>　　“嗯，今年参加考核的武者，质量比以往好多了。”
<br/>　　“不错，观他们的修为和年龄，天赋确实比过往几年的弟子好上许多。”
<br/>　　十人笑着说道。
<br/>　　这时，易老转过身来，面对三百武者，沉声道，“第三轮考核，名为石碑悟道，考的是你们的天赋。”
<br/>　　“决定的，将是你们日后在剑派内的地位。是成为普通的外门弟子，还是进入内门十堂，成为内门弟子，全看自身天赋。”
<br/>　　裂天剑派，分外门弟子与内门弟子，地位和待遇，天差地别。
<br/>　　外门弟子，剑派每月给予一定的修炼资源，由外门执事指导修炼。
<br/>　　内门弟子，剑派每月给予充足的修炼资源，进入内门十堂，由内门执事指导修炼。
<br/>　　先不说修炼资源上的差别，单单武道老师，就差距够大的了。
<br/>　　外门执事，都是些负责剑派内琐事的普通执事，修为普遍在洞玄三、四重左右。
<br/>　　而内门执事，修为最弱的都是洞玄八重。
<br/>　　至于内门十堂，分别为药堂、剑堂、刀堂、火堂、寒冰堂、万木堂.....
<br/>　　剑派弟子根据自身武魂，修习的功法，战斗手段等等，分别进入不同的分堂。
<br/>　　简单来说，便是精修，可以得到更精准的修炼指导。
<br/>　　同时，内门十堂各有一位长老，精通该堂修炼之道。
<br/>　　裂天剑派，十大长老，皆是破玄境修为，随便一个都是名震北山郡的强大武者。
<br/>　　像药堂长老，便是一位破玄七重武者，兼五品巅峰炼药师。
<br/>　　加入药堂的内门弟子，除了修炼武道，更兼学习炼药知识。
<br/>　　剑堂长老，公认的北山郡第一剑道武者。
<br/>　　万木堂长老，本身就是植物系武魂的武者，传闻他对世间任何植物都了如指掌，极为强大。
<br/>　　.....
<br/>　　十大长老，平日极少管事，潜心武道修炼。
<br/>　　教导弟子之事，一般都由内门执事负责。
<br/>　　但，作为内门弟子，终归是有机会能接触到长老，并求教一下的。
<br/>　　单单这一点，已经可以看出内门弟子与外门弟子的巨大差距。
<br/>　　.....
<br/>　　言归正传。
<br/>　　裂天剑派，外门弟子过万；内门弟子，则每堂只有寥寥数十人，十堂加起来不过数百人。
<br/>　　可想而知，要成为内门弟子，非常困难。
<br/>　　能够加入其中的，无一不是天才中的天才。
<br/>　　这时，易老指了指那上百个青色石碑，解释道，“此乃玄界碑，由裂天剑派历代长老所刻。”
<br/>　　“每一块，便是一位长老一生的修炼知识。”
<br/>　　“或武技功法，或修炼心得。”
<br/>　　“这里共有一百一十八个青色石碑，也就是一百一十八位先人留下来的武道财富。”
<br/>　　“非天资卓绝者，难以参悟玄界碑上的内容。”
<br/>　　“而就算天资过人，但每一块玄界碑上的内容都不同，能否选择到适合自己的，并从中有所收获，就看你们自己的机遇了。”
<br/>　　说罢，易老摆了摆手，道，“话已至此，随我入碑林吧。”
<br/>　　裂天剑派的考核，每一个都有其目的性。
<br/>　　第一个考应变能力；第二个考个人实力；第三个考天赋。
<br/>　　应变能力，别说是武者，就是普通人，都该具备，故放在第一项考核，分量最轻。
<br/>　　个人实力，自不必多说，放在第二项考核，分量次之。
<br/>　　天赋，作为第三个考核，分量最重，也是裂天剑派执事们最看中的地方。
<br/>　　毕竟，来参加考核的，都是全郡各地的天才，年龄都不大。
<br/>　　应变能力不强，实力不足，都可以接受。
<br/>　　但唯独天赋，裂天剑派只收天才，不收庸才。
<br/>　　将三百武者带入武道碑林后，易老便转身离去了，没有多说一句话，也没有和十位内门执事打招呼，一个瞬身便消失在原地。
<br/>　　而三百武者，则已经一个个都迫不及待地开始观看玄界碑上的内容。
<br/>　　开玩笑，每一块玄界碑可就是一个破玄境武者一生的修炼财富阿。
<br/>　　萧逸也不例外，同时他也在暗暗惊叹，不愧是北山郡的武道圣地，第一门派，一百一十八块玄界碑，可就是一百一十八个破玄境武者。
<br/>　　裂天剑派的历史，非常久远，听说炎武王国立国之时，它便存在了。
<br/>　　漫长的岁月里，能出现一百一十八个破玄境武者，足可看出它的底蕴有多么深厚。
<br/>　　正当众人都在满怀期待地参悟着石碑时....
<br/>　　忽然，一位武者开始剧烈呕吐起来。
<br/>　　不多时，甚至有人直接晕厥在地。
<br/>　　“怎么回事？”林劲一惊。
<br/>　　“我感觉有些头晕眼花阿。”秦飞扬疑惑地说道。
<br/>　　“小心。”萧逸皱起了眉头，沉声道，“石碑上有禁制，能影响我们的心神和思绪。”
<br/>　　这可是考核，怎么会这么轻易让众人参悟玄界碑呢。
<br/>　　每一块玄界碑上，都烙有禁制。武者直视石碑上的内容，就会头晕目眩，心神不定，难以专注地去参悟。
<br/>　　武道碑林外的十位内门执事淡淡地笑了笑。
<br/>　　“玄界碑，岂是那么容易参悟的。”
<br/>　　“在禁制的影响下，任何人都会心神不稳。想要成功参悟，必须要有过人的忍耐力、心智、还有领悟力。”
<br/>　　“这才是真正考验天赋的时候阿。”
<br/>　　“按照以往的惯例，顶多只有十五人能通过考核，不知今年如何呢。”
<br/>　　武道碑林内，越来越多的武者开始感觉到不适。
<br/>　　只要他们的眼睛盯向玄界碑，就会头晕目眩。
<br/>　　所有人都在硬撑着。
<br/>　　其结果，便是从头晕目眩变成了头痛欲裂，承受着非人的折磨。
<br/>　　萧逸等人，都变得脸色凝重起来。
<br/>　　“小心点。”萧逸沉声道，“如果受不了，不要硬撑，不参悟这些石碑就是了。”
<br/>　　“嗯。”林劲四人点点头，而后各自分散开，寻找适合自己的玄界碑。
<br/>　　如果是不适合自己的，便不需要浪费时间去看了。
<br/>　　萧逸快不走着，同时目光扫过一块块玄界碑。
<br/>　　“嗯？”萧逸忽然停了下来，在一块高达数十米的玄界碑下停了下来。
<br/>　　玄界碑有大有小，越大，证明记录的这位长老修为越强，修炼知识越多；越小，则反之。
<br/>　　“这是一位擅用掌法的长老，修为达到破玄九重。可惜，不适合我。”萧逸摇了摇头，继续行走着。
<br/>　　如果不适合自己，一来是浪费时间；二来，自己本就不精通掌法，难以参悟。
<br/>　　一般来说，如果自己的武魂，和这些玄界碑上的知识契合，就会容易参悟得多。
<br/>　　而且，武魂阶品越高，就越是容易参悟。
<br/>　　半晌，萧逸再次停了下来，眼中竟是喜意。
<br/>　　面前的玄界碑，记录的是一位擅使剑的长老的知识。
<br/>　　这位长老，当年是破玄九重修为，剑术以迅猛见长。
<br/>　　手中之剑，快如闪电，却又猛如惊雷；特别是那瞬间的爆发力，如火山迸发，威力惊人。
<br/>　　一剑刺出，转瞬间已达敌人咽喉；而后瞬间爆发的力量，甚至比速度更为恐怖。
<br/>　　曾一剑秒杀一位同境界的破玄九重武者。
<br/>　　“厉害，就是你了。”萧逸眼前一亮，开始参悟这块玄界碑。
<br/>(本章完)
