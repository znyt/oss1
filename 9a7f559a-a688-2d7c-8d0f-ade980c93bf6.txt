母女俩正开心的时候，夏振国从楼梯上下来，急匆匆的准备出门。
<br/>
<br/>        “爸，你要去哪里？”夏妍妍问。
<br/>
<br/>        夏振国道，“我要出门一趟，刚才夕绾打电话给我，说是老爷子已经苏醒了！”
<br/>
<br/>        什么？
<br/>
<br/>        夏妍妍和李玉兰同时一僵，两个人对了一个眼神，怎么可能？
<br/>
<br/>        她们已经给老爷子下毒了，按照毒性，老爷子应该归西了吧？
<br/>
<br/>        “爸，你说真的，爷爷苏醒了？”
<br/>
<br/>        “当然是真的，我现在就去医院，你们要不要跟我一起去看老爷子？”夏振国力邀这对母女俩。
<br/>
<br/>        夏妍妍的脸色变得很古怪，她拒绝道，“我今天就不去了，爸你去看看情况，妈还在坐月子，我留在家里陪妈。”
<br/>
<br/>        “好，那我先走了。”夏振国快速的消失了。
<br/>
<br/>        夏振国一走，李玉兰直接瘫在了沙发上，她紧张兮兮的看向夏妍妍，“妍妍，这究竟是怎么回事，老爷子怎么可能苏醒呢，老爷子千万不能苏醒，要不然十一年前我们设计推他下楼，甚至还有给他下毒的事情都会爆出来，到时我们就完了。”
<br/>
<br/>        夏妍妍当然也知道这其中的厉害关系，现在陆寒霆和夏夕绾离婚了，她离陆太太就差了这么一步，她是绝对不会允许意外发生的。
<br/>
<br/>        “妈，先不要慌，我觉得这件事有蹊跷，我偷偷去医院看一下这事是真是假。”
<br/>
<br/>        “好。”
<br/>
<br/>        ……
<br/>
<br/>        夏妍妍赶到了医院，她戴着鸭舌帽和墨镜，很怕夏夕绾她们认出来，问了前台，她找到了夏老爷子的病房。
<br/>
<br/>        站在病房门口，她就听到里面传来了夏振国和夏夕绾惊喜的声音，“爷爷，你终于醒了，我们好想你啊。”
<br/>
<br/>        很快里面又传来了一道苍老粗嘎的声音，像是老爷子的，“水…水…”
<br/>
<br/>        夏妍妍透过门上的小玻璃窗向里面看去，只见病床上躺着一道年迈的身影，还有一只苍老的手伸了出来，被夏振国和夏夕绾紧紧的握住。
<br/>
<br/>        夏妍妍心里都凉了，她不知道哪个环节出了错，她只知道老爷子是真的苏醒了。
<br/>
<br/>        夏妍妍迅速返回了夏家，将这件事告诉了李玉兰。
<br/>
<br/>        李玉兰脸色煞白，已经慌了神了，“妍妍，我们现在该怎么办，完了完了，那可是故意杀人罪，我们会被判刑的。”
<br/>
<br/>        夏妍妍脸色一阴，“妈，你慌什么，犯法的事情我们已经做了，现在只能一不做二不休了，我看爷爷现在意识还没有完全清醒，说话都不利索，我们还有机会，今天晚上妈你偷偷潜进病房，给爷爷打一针，这里面的药水无色无味，绝对不会被发现了。”
<br/>
<br/>        李玉兰震惊的看着夏妍妍，“妍妍，你要我去？”
<br/>
<br/>        夏妍妍握住了李玉兰的手，“妈，这件事只能你去，难道你不愿意吗，只要你去了，爷爷死了，十一年前的真相就没人知道了，你还是夏夫人，到时我成为陆太太，我们美好的人生才刚开始！”
<br/>
<br/>        李玉兰想了一下，目光里也露出了阴狠，她点头，“好，我去，永绝后患！”
<br/>
<br/>        ……
<br/>
<br/>        夜晚降临，整个医院都变的静悄悄的了，李玉兰来了，她乔装成了一个护士，戴着口罩，推开了老爷子的病房门，她手里拿着夏妍妍给的针管。
