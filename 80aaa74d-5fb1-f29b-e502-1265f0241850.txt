　　后土部落。
<br/>　　自从后土身化轮回之后，后羿便成了部落的首领，他带领着部落的人训练，带着部落的人杀敌。
<br/>　　此次十日横空，部落遭受大难，夸父部落便是后土部落下的一个小部落，像这样的部落已经被十日毁去了无数个。后羿身为部落首领，应该对此负责。
<br/>　　这日，后羿坐在家中，看着天空之上的十日，看着被炙烤得开裂的大地，看着无数湖泊干涸，无数生灵死亡，他感受到了自己的使命。
<br/>　　抚摸着手中的一张水晶般的玄晶大弓，感受着上面弥散出来的无尽寒气，那股仿佛能够冰冻灵魂的寒气，让人颤栗。
<br/>　　这张大弓只有弓，没有箭，上面刻有雕花刻纹，神鸟翱翔，冰龙在天，栩栩如生。它不像是一件武器，反而宛若一件美轮美奂的艺术品。
<br/>　　“那****在天山得到你时，你还只是一块冰石，但到了我手中后，你却变成了一张大弓。我一直不明白你为何名为‘射日’，如今看来，一切早已命中注定。”
<br/>　　后羿双手抚摸着大弓，就仿佛是在抚摸自己的恋人一般，良久，他对着大弓轻声道：
<br/>　　“也许，这就是你的使命，也是我的使命。射日，射日……也罢，既然上天都选择了你我，那么，你我更应该对得起上天才是。就让你我射了这横贯苍穹的十日吧。”
<br/>　　“嗡！”
<br/>　　后羿的话音刚落，他手中的冰晶大弓便颤动一声，发出嗡嗡响声，仿佛是在兴奋。
<br/>　　“好，既然你也这么认为，那我们便走吧。”
<br/>　　后羿微微一笑，感受到手中冰晶大弓的的兴奋情绪，他仿佛也被感染了。
<br/>　　“后羿，你……你真的要走吗？”
<br/>　　就在此时，门外那一直站着的女子开口说道，话语之中，带着浓浓的担心以及不舍。这是一位身穿银装，面容姣好，气质如仙的女神，她就仿佛九天之上的月神，那如月亮一般的清净，让人看后不由自主的心神舒畅。
<br/>　　她，便是后羿的妻子，嫦娥。
<br/>　　后羿站了起来，看着眼前的如同神女一般的嫦娥，他的眼中闪过一分眷恋，一分甜蜜，但更多的是不舍。
<br/>　　突然，后羿上前，用他那冰冷的双手，抚摸着嫦娥的脸庞，轻声道：
<br/>　　“我的族人在等着我去拯救，娘娘既然已经将部落交于我管理，我便不能让它毁灭。这是我的使命，我不得不走。放心，我会安全回来的。”
<br/>　　嫦娥的眼中闪过一丝泪花，她知道，后羿此行非常危险，或有生命之危。感受着后羿的情绪，她知道既然后羿做出决定，便不会轻易更改。做为他的妻子，她选择相信他，她不再缠着他，而是轻声说道：
<br/>　　“我相信你，你……一定要回来。”
<br/>　　“我一定。”
<br/>　　后羿用手温柔的抹去嫦娥眼中的泪花，他的眼神柔和，看着嫦娥。
<br/>　　突然，轻轻地低下头，吻了一下嫦娥的额头，之后，他毫不犹豫地转身，走出了大门，向着十大金乌奔去。他不敢回头，他怕自己一旦回头，便狠不下心来离开。
<br/>　　门框外，嫦娥看着后羿那坚定的背影，眼中闪过浓浓不舍，她心中有种不祥的预感，这是自己最后一次见后羿了。
<br/>　　……
<br/>　　九天之上，风云震动，十日横空，大日炎炎，苍穹通红。
<br/>　　自从十大金乌灭了大巫夸父之后，他们就变得更加嚣张，疯狂了。它们到处引燃真火，让洪荒大地变成一片火海。
<br/>　　“一箭风云变！”
<br/>　　就在此时，一道冷漠的声音从火海之中传出，声音无情若冰，淡漠如水，不见丝毫情绪波动，但却又好像火山爆发的前奏。
<br/>　　正在兴奋玩耍的十大金乌纷纷感觉一阵心惊肉跳，全身冰冷，就好似有巨大的危机笼罩其身，让人看不到丝毫生机。
<br/>　　“咻！”
<br/>　　一阵破空之音响起，风云变色，乾坤震动，虚空摇晃，好像经受不起某种压迫一般。肃杀，冰冷、无情、寒冰，生死……充斥着苍穹。
<br/>　　一道黑点闪过，携着风云之力，席卷而来，仿佛洪荒猛兽觉醒，盯住了某人，让人躲无可躲，闪无可闪。
<br/>　　“轰！”
<br/>　　一声爆炸声响起，只见一个巨大的火球从天而落，带着无尽火光与硝烟，深深的砸落在地。
<br/>　　“二箭苍穹哭！”
<br/>　　“咻！咻——”
<br/>　　又是两道箭光闪过，乌云涌动，汇聚而来，苍穹变得漆黑一片，仿佛一片黑色天幕，遮蔽了苍穹，只剩下九个太阳，还在闪闪发光，这一刻，九日变得十分明显。
<br/>　　“轰！”
<br/>　　又是两道巨大的火球坠落下来，前后相差不到一息时间，就有三只金乌陨落。其它金乌还没有从这突如其来的变故中惊醒，就又有同伴死亡，顿时，他们纷纷惊恐。
<br/>　　“啊！不好，他那弓箭正好克制我们，快逃！”
<br/>　　感受到那股强烈的锁定他们的箭意，那股冰冷到灵魂深处的寒意，丰富能够将惊魂冰冻一般，其余金乌纷纷脸色大变，惊叫起来。
<br/>　　“小三、小六，小七，你们死的好惨，啊！”
<br/>　　“畜生，此时才逃，你们不觉得晚了吗？”
<br/>　　后羿冷笑一声，他想起死去的无数巫族，以及他的好朋友夸父，眼中杀意顿时就如同狂风暴雨一般，止不住的狂涌而来。
<br/>　　同时，又是一道冷漠的声音响起，就仿佛催命符一般令人心头发寒。
<br/>　　“三箭雷霆震！”
<br/>　　“轰隆隆！”
<br/>　　九天之上，乌云涌动，翻滚不休，相互撞击，电闪雷鸣，雷霆狂暴。三道雷霆宛若化成三道雷箭，对着剩下的金乌射去。
<br/>　　不动如山，动如雷震！
<br/>　　雷霆快速闪动，让人不抓不到丝毫痕迹，划破漆黑天际，三道闪电闪过，瞬息之间，剩下七大金乌才刚刚有所行动，便被三道狂暴的雷箭射中。
<br/>　　“啊！不！”
<br/>　　“轰隆隆！”
<br/>　　雷箭炸开，又是三道巨大的火球从天而降，惊天动地，如同流星一般，划破天际，向着大地砸来。
<br/>　　“嘶！”
<br/>　　“这是……大巫后羿，竟如此了得，恐怕就连准圣面对这一箭，都不能闪过。”
<br/>　　“巫族，真是不可小觑！”
<br/>　　“洪荒之中从没有见过如此了得的箭术。之前便一直听说了巫族后羿的箭术，如今一见，果然名不虚传。后羿，当为洪荒第一神箭手！”
<br/>　　“厉害了，此箭可杀准圣！”
<br/>　　“……”
<br/>　　洪荒众大神见到这一幕，纷纷惊骇不已，面对后羿的箭术，他们只觉得无法闪躲，心头一阵冰凉。他们想到，若是后羿用这样的箭术对自己，到时该怎么办？
<br/>　　同时，他们也确定了，后羿堪称洪荒第一神箭手！
<br/>　　“四箭山河碎！”
