<br/>    李振邦三人说话并没有压低声音，所以商队的人和佣兵们自然都听到了，出声叫住李振邦三人的正是车队中的一名白胡子老头。<br/>　　<br/>　　    “嗯？老大爷，你是在叫我吗？”李振邦纳闷的看向老头，他很确定自己并不认识他。<br/>　　<br/>　　    “是的，小兄弟，我刚才无意中听到你说的话，什么五庄观，什么人参果，不知道小兄弟是从哪里听说的？老夫活了这么久，还从没有听说过这么个地方，也没有听说过这东西，所以出声询问一下。”<br/>　　<br/>　　    “呃……”李振邦思索了一下，他有些不知道该怎么说。如果他说自己是在讲故事，恐怕这老头也不一定会相信，还会认为自己是敷衍他。<br/>　　<br/>　　    “小兄弟，这是我的一点意思，不成敬意，还希望你笑纳啊！”老头误会了李振邦的意思，以为他想要一些好处，毕竟很多消息都是需要花钱买的。所以他拿出来一个钱袋，取出来两枚银币，递给了李振邦。<br/>　　<br/>　　    李振邦犹豫了一下，还是接了过来。看到李振邦接过银币，老头不但没有生气，反而开心的笑了起来。<br/>　　<br/>　　    “这个是我无意间听说的，好像是从什么上古书籍中看到的，据说是什么上古遗迹中可能还存在吧！”<br/>　　<br/>　　    白胡子老头缕着白胡子点了点头，他在听到李振邦话的时候，脑海中第一个跳出来的就是上古时期的事情。<br/>　　<br/>　　    因为只有上古时期才会有各种不可思议的东西，现在发现的很多上古遗迹中都发现了不少了不得的宝贝，李振邦的话让他更坚定了自己的想法。<br/>　　<br/>　　    “那你可知道那五庄观所在何处？”白胡子老头有些激动的问道。<br/>　　<br/>　　    “呃……老大爷，你开玩笑呢吧？我怎么可能知道五庄观在什么地方啊！我要是知道，我早就去了！”李振邦无奈的摇了摇头。心中暗笑：这老头是不是想长寿想疯了。<br/>　　<br/>　　    “小兄弟，你再仔细的想一想，看看能不能想起来什么？”<br/>　　<br/>　　    “好吧！我再想想啊！五庄观好像在一座山上，好像叫什么万寿山。对了！五庄观前有一座石碑，碑上刻着十个字！”李振邦停顿了一下，既然对方这么想了解，自己就逗逗他好了。<br/>　　<br/>　　    看到李振邦停了下来，白胡子老头又拿出来两枚银币，递给了李振邦。<br/>　　<br/>　　    李振邦毫不犹豫的收下了，就当他听书的门票了。<br/>　　<br/>　　    “十个字是……万寿山福地，五庄观洞天。对，就是这十个字。”李振邦思索了一下说道，同时点了点头，确定了一下自己的话。<br/>　　<br/>　　    龙淼淼和欧米伽两人看着李振邦一本正经的样子强忍着没有笑出来，不过都憋的很痛苦。龙淼淼毕竟还年轻，忍耐力有限，脸都已经憋得有些扭曲了，幸好她带着斗笠，没有人能看到。<br/>　　<br/>　　    白胡子老头眼睛中放出一道精光，他没想到自己竟然这么轻松就得到了一条关于上古遗迹这么重要的消息。<br/>　　<br/>　　    这也不能怪白胡子老头，以前的上古遗迹都是从一些小孩或者普通百姓口中寻到的一些蛛丝马迹发现的。有的是一些神话故事，有的是一些恐怖故事等等。<br/>　　<br/>　　    他倒不相信人参果会万年成熟，也不相信人参果有这么大的功效，但是延年益寿想必还是会有的。如果自己发现了人参果，自己就会名载史册的，甚至有机会成为一名圣级强者。就算是将这个消息卖出去，自己也能挣得盆满钵满啊！<br/>　　<br/>　　    让李振邦万万没有想到的是，这个消息后来真的是满世界传扬，兴起了一番寻找寻五庄观的热潮，甚至有一些人跳出来说自己是五庄观的传人，满世界进行诈骗。更有甚者最后竟然真的发现了一座五庄观，当然这都是后话了。<br/>　　<br/>　　    “还有事情吗？没有的话我们就走了！”<br/>　　<br/>　　    “等一下，这个给你，不过你要保证不会再将这个消息告诉其他人。”白胡子老头竟然拿出来一枚金币递给了李振邦。<br/>　　<br/>　　    李振邦接过金币，眼冒金光的看向白胡子老头，忙不迭的点头保证，三人这才离开。<br/>　　<br/>　　    走了很远，确定已经甩开了后面的商队，三人再也忍不住了，哈哈大笑起来。<br/>　　<br/>　　    “振邦，没想到你这骗人还是一把好手啊！哈哈！逗死我了！你看到那个老头紧张兮兮的样子没有？太好笑了！”欧米伽哈哈大笑了起来。<br/>　　<br/>　　    “就是，就是！振邦哥，你说如果那老头知道自己被骗了，会不会拎着大刀把你剁成肉泥！”龙淼淼咯咯直笑，笑的花枝乱颤。<br/>　　<br/>　　    “切，跟我有什么关系。我只是在讲故事，结果他想花钱听，我就卖给他好了！”李振邦颠了颠手中的金币和银币，笑呵呵的说道。<br/>　　<br/>　　    “振邦哥，继续讲，继续讲，他们师徒四人到了五庄观以后怎么样了？”龙淼淼笑够了以后，急忙催促着李振邦继续讲。<br/>　　<br/>　　    “你就让我歇一会吧！这里可是森林，周围的景色多美啊，空气多么新鲜啊！我们出来是历练的，顺便了解一下卡罗帝国的风土人情。人不止要读万卷书，还要行万里路啊！”李振邦敲了一下龙淼淼的斗笠，宠溺的说道。<br/>　　<br/>　　    龙淼淼哼了一声，不再理会李振邦，抬着头朝前方走去。欧米伽看了看李振邦，伸了个懒腰，也晃晃悠悠的朝着前面走去，反而留下李振邦一个人在原地发愣……<br/>　　<br/>　　    让李振邦感到意外的是，整整一个下午，龙淼淼和欧米伽都没有再让自己讲故事。同时俩人也都约定好了一般，谁也没有理会自己，反而两个人倒是有说有笑，看看花看看草的。<br/>　　<br/>　　    幸亏李振邦及时发现三人偏离了预定的路线，否则他们三人说不定就跑到哪里去了，即便如此，他们也绕远了不少路程。等三人回到预定路线的时候，天都已经黑了。<br/>　　<br/>　　    本来李振邦这次出来就是历练的，多走一些路，多看一些风景本就是无可厚非的事情。而且这一路上他也采集了不少药材，否则他也不至于在家里面待几天就出来了。<br/>　　<br/>　　    “前面有火光，大家小心一点。”李振邦急忙提醒道。<br/><br/><br/>
