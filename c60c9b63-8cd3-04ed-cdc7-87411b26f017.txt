　　“砰！砰！砰！……”
<br/>　　元始的神海之上，传来一阵阵撞击的声音，声音越来越宏大，到后面，完全不亚于雷闪电鸣之声。
<br/>　　元始的身体在剧烈颤动，显然，他在承受着非一般的痛苦，他额头之上，出现了点滴冷汗，但他并没有放弃，依旧用神念引导着太阴本源不停地撞击神窍之门。
<br/>　　元始心念一动，将太阴本源化为一道细针，猛然刺向穴窍之门。
<br/>　　“嗖！”
<br/>　　细针毫无阻拦地刺破金色大门，之后，源源不断地太阴之力如同大坝崩溃一般，涌进元始的神窍大门。
<br/>　　“轰隆！”
<br/>　　一声大响，金门被洞开，神窍之门化为无数金色碎片，随着至阴本源涌进神窍之中。这一刻，神窍空间随着太阴本源的涌入，渐渐地被扩大。
<br/>　　神窍在闪闪发光，银亮色的光芒散发着阴寒的气息，一部分太阴本源沿着元始那如同大江大河般宽阔的经脉流入到他全身，慢慢地增强者着他的肉体强度。
<br/>　　霎时间，元始的整个身体银光绽放，元始就仿佛化身太阴星，不由自主地吸引着星空之中的星光的朝拜。
<br/>　　星光入体，增强他的法力，锻炼他的肉体，他的力量在一点一滴地增强。
<br/>　　这种状态不知持续了多久，也许是一瞬，也许是数百年。
<br/>　　直至良久，元始才睁开双眼，双眼之中，一个银色的星球在缓缓旋转，他的眼中冒出阴冷刺骨的寒气。
<br/>　　“呼——”
<br/>　　元始呼出的一口气，瞬间便结冻成冰，散落在地。
<br/>　　元始站了起来，感受一下自己的身体强度，极为满意。现在的他，仅仅只靠肉体就能虐杀准圣圆满境界的强者。
<br/>　　……
<br/>　　秘境之中，月宫门前，两道绝色女子正坐在一座花亭下，吃着灵果。
<br/>　　突然，较小的女子恨恨的说道：
<br/>　　“老师，你说那个混蛋死了没有？哼！敢动太阴本源，最好让他死掉。”
<br/>　　另一个女子笑了笑，抓起一个灵果，吃了一口，苦笑道：
<br/>　　“没那么容易的，我看他不像是鲁莽之人，若是如此容易死的话，他就不会去吸收那太阴本源了。”
<br/>　　“哼！他最好死掉，竟然敢封印我们！”女子说来就十分生气，自己都没有惹过对方，无缘无故就被封印了。
<br/>　　那女子没有说话，她一样被封印了，她如今感受不到自己身上丝毫的法力波动，只能任人宰割。这种感觉，好不爽啊！
<br/>　　想到那个男子，她就有种恨恨的冲动，不过，谁让自己技不如人呢？她知道那次战斗那人远没有发挥出他的全部实力。她想想就有些无力，既对自己实力弱小的无力，同样对自己未来的无力。
<br/>　　这两道绝色丽人自然是月神望舒与她的弟子嫦羲了。自从被元始封印，已经过了近百年了。两人现如今如同凡人一般，不能炼化灵气，所以会有饥饿的感觉。饿了两人自然要进食，而灵果便是她们的食物。
<br/>　　“老师，我们今后要怎么办啊？”嫦羲神色之间有些暗淡，幽幽开口道。
<br/>　　望舒看了一眼嫦羲，同样有些无奈，叹了一声，道：“我亦不知。”
<br/>　　顿时，两人陷入了无尽的沉默之中。
<br/>　　“哗啦——”
<br/>　　正在此时，阵法被打开，元始从阵法之中走了出来，他身上都太阴之力已经被收起来了，内敛于身，在他身上，别人感受不到丝毫的法力波动。
<br/>　　两人被惊醒，抬头看了一眼元始，顿时，三人之间，一时间变得寂静无比，空气中有一种怪异的气氛在亭中蔓延开来
<br/>　　元始淡淡地扫了她们一眼，面无表情，心中却在思索着该如何处理她们？
<br/>　　杀了？不！这两人都是洪荒之中屹立绝巅的仙子女神，杀了多浪费啊！
<br/>　　不杀？那又该如何处置？自己与她们已经结怨了，留着干什么？
<br/>　　哎！真是头疼啊！
<br/>　　思索良久，元始还是没有拿出注意。
<br/>　　“姓元的，你要如何处置我们师徒俩？”
<br/>　　就在此时，月神望舒冷冷地对着元始问道。声音极为冰冷，听不出丝毫的温度。
<br/>　　“姓元的？”元始哭笑不得，自己什么时候成了姓元的了，虽然自己名号元始，但也不是姓元啊。
<br/>　　看了两人一眼，两人虽然在看着元始，月华笼罩，面上充满冰冷之色，但元始还是从她们眼神之中看出一丝害怕。也是，谁也不想自己就这么没意义地死去。
<br/>　　嗯？罢了罢了！元始心中一叹，作出决定。挥手两道流光飞入两人体内，之后，元始头也不回，伸手撕裂虚空，踏了进去。
<br/>　　虚空之中传来元始的哈哈大笑声：
<br/>　　“记住，你俩是本座的俘虏！哈哈哈！”
<br/>　　院中两人顿时感觉身上的封印全解，一身法力神通具皆恢复，全身涌起惊人的气势，周围的花海涌动，秘境之中变得不再平静。
<br/>　　良久，秘境才恢复平静。两人对视了一眼，眼中闪过莫名之色，还隐含着一丝庆幸，庆幸自己终于得到新生。
<br/>　　莫名的是这个叫元始的竟然没有杀了自己两人，没有让自己做一些别的事，还真是奇怪。
<br/>　　“可恶！这个混蛋真是可恶！总有一天我要让他好看，还俘虏？我要让他做我的俘虏。”嫦羲对着虚空狠狠喊道。
<br/>　　“这个混蛋真混蛋！”望舒也说道。
<br/>　　同时，两人都知道，自己恐怕一生也难以报仇了，看对方的法力，便知道是高手中的高手，自己报仇的希望何等渺茫。
<br/>　　不过，两人都没有放弃，两人都是心高气傲之辈，何时受过此等窝囊气，决定从今以后，好好修炼，争取早日证道。
<br/>　　同时，两人内心深处也希望元始永远不要出现在她们面前，最好永远不要。但，那可能吗？
<br/>　　远去的元始自然不知道两人的心思，就算知道，元始也不会将之放在心上。自己一生，就是去超越别人的，永远没有被别人超远的想法。想要超越自己，你们不如洗洗睡吧。
<br/>　　自己之所以没有杀了两个女子，除了两人都是人间绝色女神的原因之外，还有就是两人之后都会有自己的缘法，之后的大劫中，两人都扮演着极为重要的角色，不能缺少。
<br/>　　嫦羲是妖帝帝俊的妻子，望舒虽然没有什么大事，但以她是太阴之主的身份，就注定不能将此事置之度外。
<br/>　　以后的星辰也会被妖族占去，自己以后想要与妖族对峙，谋算些什么，就需要望舒与嫦羲这道棋子了。
<br/>　　当然，元始绝不会承认，放过两人绝大多数原因还是因为两人都是女子，而且还是绝世倾城的寒月女神。女神吗，总有那么一点点特权的。
<br/>　　若是两人不是女神，只是男子，或者一般女子，元始绝对不会手下留情，他绝对会一巴掌拍死，以绝后患。
<br/>　　棋子？笑话，他元始还不会将日后的妖族放在眼中，东皇太一与妖帝帝俊迟早会死翘翘，剩下一些小妖，不足为惧。自己到时候也成了圣人，自己会怕他们？
<br/>　　总之，元始认为给自己找到了合适的理由，放了两位女神，至于今后会如何，那就要看今后的了。
<br/>　　今后的事，谁又说得准呢？
