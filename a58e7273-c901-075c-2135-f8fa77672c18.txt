<br/>				“伤口恶化，有黑色气息浮动，你来看下吧！”轩辕墨泽说着，示意他上前来。<br/>　　<br/>　　陌尘来到床边，见凤九已经昏睡着，只是睡得不安宁，尤其是脸色泛红渗着汗水，唇嘴却苍白无色，当下，他上前查看了下伤口，果真见那伤口处微肿，黑色气息浮动。<br/>　　<br/>　　“全身的伤口都一样这样吗？”他看向轩辕墨泽问着。<br/>　　<br/>　　闻言，轩辕墨泽道：“我解开了两处伤口，都一样。”<br/>　　<br/>　　陌尘帮凤九把了下脉，当手指触及她的脉博时，他的眉头一拧，回头看向众人，道：“你们先出去吧！”<br/>　　<br/>　　见此，关习凛等人便先行离开，让他可以静下心来为凤九治疗，只是，未见凤九的情况稳定下来，他们却是心头不安，七上八下。<br/>　　<br/>　　在天丹楼守着的火凤和老白两兽因察觉到什么一般，询问了天丹楼的人后，便迅速赶回凤府来，当它们一进凤府，见大晚上的大伙却站在主院的院中来回走着，不时的看着房间时，两兽迅速上前。<br/>　　<br/>　　“出什么事了吗？我家主人呢？”老白询问着，一边想要上前进房中，却被关习凛拉住。<br/>　　<br/>　　旁边的火凤则盯着那紧闭着的厢房，它拍着翅膀停落在桌面上，不知在想着什么。<br/>　　<br/>　　“她受了伤，情况不太好，你们先别进去，陌尘和轩辕他们在里面。”关习凛说着，一边范霖等人道：“吞云的情况跟凤九差不多，你们想办法把它送到这边过来吧！”<br/>　　<br/>　　“好。”八名凤卫首领应了一声，转身往外面走去。<br/>　　<br/>　　火凤见眼下也进不去厢房，于是便跟在八名凤卫的身后去看吞云，老白见状也连忙跟了上去。<br/>　　<br/>　　房间里，陌尘看着轩辕墨泽，道：“她的身体里有混沌青莲的气息，那一颗混沌青莲，应该就在她的身体里吧！”<br/>　　<br/>　　听到这话，轩辕墨泽也没多说，只是应了一声：“嗯。”他刚才也感觉得到了，似乎是因为她的伤口出现恶化，她体内的青莲自动散发着青莲之气，只是，那股气息很弱，一般人很难察觉到。<br/>　　<br/>　　“如果我猜得没错，今晚袭击你们的人，应该就是冲着她体内的混沌青莲来的。”陌尘轻叹一声，道：“而且，她这敌人很强，那灭世黑莲就那些人的手中，她这伤口上的黑色气息，也不是什么毒气，而是灭世黑莲的气息。”<br/>　　<br/>　　轩辕墨泽深邃的黑瞳微闪，问：“你可有办法除去她伤口上的这股黑色气息？”只要将这股气息清除，她体内的青莲就会自动修复，这些伤口要恢复就会容易很多。<br/>　　<br/>　　陌尘微微一笑，温声道：“应该要庆幸的是，今晚你们遇到的这些人，并非那名持有灭世黑莲的那个人，今晚这些人的刀剑中所蕴含的灭世黑莲气息有限，情况也不会太严重。”<br/>　　<br/>　　他的声音一顿，对轩辕墨泽道：“你把她扶起来，我运功为她清除这灭世黑莲的气息。”<br/>　　<br/>　　闻言，轩辕墨泽这才取下她额头的小布巾，再将她扶了起来，背对着陌尘。<br/>
