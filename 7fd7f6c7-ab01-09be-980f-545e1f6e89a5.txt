　　辽东的四月已经变的十分炎热，大祚荣根本没有想过这一场战争会拖的如此之久，所以他带兵出来的时候，那些骑兵还都穿着冬装。
<br/>　　可是季节这东西并不会因为你穿着冬装就不会改变，炎炎烈日下那些还穿着皮袄的靺鞨骑被晒的苦不堪言，最后不得不全都打起赤膊。
<br/>　　不过好在夏季并不缺青草，所以大祚荣只要为粮食发愁就可以，战马靠着那些新长出来的草还能坚持一段时间。
<br/>　　而就在靺鞨骑兵大营的对面，黑齿常之他们早就已经换上了夏装，又因为本土作战，不用为粮草发愁，所以小日子过的很是惬意，有些时候心情好了说不定还会弄一些粮食放到两军阵前，送给那些靺鞨。
<br/>　　大祚荣虽然明知道这是黑齿常之在破坏自己的军心，但却一点办法都没有。
<br/>　　因为他们真的快要断粮了，来时抢的那些粮食根本不够他们长时间吃的。
<br/>　　毕竟他之前的计划是在一个月之内将唐军赶出辽东，然后就可以空出手来展开攻城市，夺取原本属于大唐的城市，取得他们需要的一切。
<br/>　　可是现在一切都变了，浑河边一战他损失了近三分之一的人手，余下六万人根本不敢乱动，只能与黑齿常之僵持。
<br/>　　河边的那一仗真是把大祚荣打怕了，对唐军的战力有了更深层次的领教。
<br/>　　而那些靺鞨骑兵也同样如此，逃回来的骑兵把那些唐军几乎夸大到了天兵的程度，一个打十个甚至都不足以形容其彪悍。
<br/>　　所以靺鞨骑兵人心也开始不稳起来，大营中开始有对大祚荣不利的声音传出。
<br/>　　这更让大祚荣不敢分兵夺粮，只能在原地固守，希望能够找到机会给黑齿常之致命一击。
<br/>　　……
<br/>　　四月底，再度坚持了半个月后，大祚荣真的坚持不住了，战马虽然不缺草，但是人吃的粮食已经告窑，再不撤兵，他带出来这六万多人将会饿死在这片黑土地上。
<br/>　　而就在此时，数匹快马如飞般冲进军营，给大祚荣送来了一个让他无比震惊的消息：五千唐军自漠北绕道已经攻入靺鞨腹地，黑水河。
<br/>　　黑水河，用现在的话说就是黑龙江，是黑水靺鞨的发源地，相当于大唐长安一样的重要。
<br/>　　所以当大祚荣知道黑水河出现唐军骑兵之后，立刻意识到，这边的对峙只是一个幌子，为的只是吸引他的注意力，而唐军真正的目的，其实是断其后路，使其无家可归，最后被围死在辽东。
<br/>　　“首领，怎么回事？我听说……”这个世界上永远都不缺消息灵通之辈，大祚荣是进是退的决定还没有想好，伯咄部族长便已经来到了中军大帐。
<br/>　　“连你也听说了么？”大祚荣神色出奇的平静，就好像什么事情都没有发生过一样。
<br/>　　“首领……”伯咄部族长脸上关切的表情，但只要是个人都能看出来的，他眼中的那一层兴灾乐祸。
<br/>　　“怎么？你是来看我笑话的么？”大祚荣同样看出了伯咄部族长似乎有所图谋，脸上不由露出嘲讽的表情。
<br/>　　数月之前，在浑河一战之后，大祚荣其实便已经开始注意到伯咄部的族长。
<br/>　　那个时候虽然他们处在下风，但如果一直冲下去未必没有一丝获胜的可能。
<br/>　　可是因为伯咄部族长的一句话，军心立时就散了，所有人开始为了逃命而逃命，甚至为此还赔上了黑水部族长的一条老命。
<br/>　　事后，大祚荣每次想到当时的情景，都感觉都透着一股子阴谋的味道，只不过归来之后的伯咄部族长一直没有什么特殊的表现，他也没有办法确定当时这家伙是不是在耍什么手段。
<br/>　　而现在，狐狸的尾巴终于露出来了，在自己没有召见他的情况下，这家伙不请自来，而且还主动提出了关于唐军突袭靺鞨后方的事情，这足以说明伯咄部已经有了背叛的心思。
<br/>　　但是很显然，伯咄部族长也不是没有丝毫的准备，听到来自大祚荣的嘲讽后，竟然没有生气，只是无可奈何的笑笑，然后继续着自己的话题：“首领，误会了，我这不是关心战局么，对面的唐军对我们虎视眈眈，如果我们现在撤兵那黑齿常之必然引兵来追，如果一个应付不好，兵败如山倒，只怕我们谁都回不去了。”
<br/>　　果然是一条老狐狸，大祚荣心中暗骂一声，对于伯咄部族长不接自己话茬的举动一时也有些无奈，最后只能强忍不快问道：“那依你之见，我们该当如何呢？”
<br/>　　“我觉得我们只有两个办法，第一，那就是全军尽出，直接碾压过去，将对面的唐军杀败，然后再从容撤走；其二，那就是留下一部断后，其他族人撤回靺鞨。”伯咄部族长说道。
<br/>　　“哼，说的容易，你就那么有信心以不足七万之兵打败对面那支唐军？”大祚荣冷冷说道。
<br/>　　“首领，不试试怎么知道？”
<br/>　　“若是败了呢？还如何回援？”
<br/>　　“可是首领，就算是我们能够撤回去，若是这支唐军跟上我们怎么办？到那时候我们需要面对的就是一万五千唐军，而不是一万。”伯咄部族长的声音提高了一个八度，已经没有了刚刚客气。
<br/>　　而大祚荣却突然陷入了沉默。
<br/>　　不得不说，伯咄部族长说的还是有些道理的，如果黑齿常之不理会自己，只是在后面跟着，他的确没有办法应对。
<br/>　　想到这里，大祚荣皱起了眉头，沉声再次问道：“那么依你之见呢？哪种方法好？”
<br/>　　“我不知道，不瞒首领您说，这段时间我一直在琢磨有什么办法可以让我们在不受损失的情况下撤回靺鞨。”伯咄部族长苦笑着摇头说道：“可是思前想后却发现，似乎我们除了投降之后再也没有其他任何办法。”
<br/>　　“什么？你说投降？”一股无名之火突然充斥着大祚荣的内心。
<br/>　　若是现在投降，他怎么向已经死了的黑水部族长交待？若是现在投降，那么当初又何必从山里出来？若是现在投降他大祚荣还有什么脸面继续统领靺鞨所有部落。
<br/>　　而就在大祚荣一怒之下，想要命人将伯咄部族长拖出去五马分尸的时候，大帐外面再次走进几个人来。
<br/>　　其中一人以质问的语气说道：“可是首领，不投降又能如何呢？我们现在已经无力反击，我们的粮食不多了，我们的勇士已经没有了继续攻击的勇气，您觉得还有什么办法可以让我们面对来自大唐的反击？难道您真的认为大唐就没有能力一举屠尽我们么？”
<br/>(本章完)
