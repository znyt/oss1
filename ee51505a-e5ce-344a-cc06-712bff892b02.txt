        或许是因为，自己孤苦伶仃，无依无靠，没有家人，也没有亲人，童筱颖拎着大包小包的行李，站在站牌前，看着一行又一行的路线，错过了一趟又一趟的公交车，却不知道自己接下来要去哪里。
<br/>
<br/>        不如回云城西镇吧！
<br/>
<br/>        好像每一次在她最落魄的时候，只有外婆那里可以收留她。
<br/>
<br/>        可是……
<br/>
<br/>        那里其实只会让她觉得更加孤单……
<br/>
<br/>        毕竟，深山老林，景色虽好，但人烟稀少。
<br/>
<br/>        若是没有在遇到大哥之前，她觉得那儿确实是一块修身养性的净土，但是，现在，她已经静不下心去过那种日子了。
<br/>
<br/>        因为心里有了牵挂，所以，她好想有个家，却不知道哪里才是她的家。
<br/>
<br/>        心里像被谁塞进了一颗又一颗的石头，无端的沉重感，压得她有些喘不过气来。
<br/>
<br/>        她该何去何从？
<br/>
<br/>        正迷失在自己失意里的她，压根就没注意到身后有一个猥琐的男人正偷偷向她贴近，甚至还伸出了一只像树皮一样粗糙的大手，向她的裙底探去。
<br/>
<br/>        “找死！”
<br/>
<br/>        直到身后传来一个令她十分熟悉而低沉迷人的男性嗓音，童筱颖如梦初醒般赫然转过身去一看。
<br/>
<br/>        只见傅夜沉剑眉微蹙，冷凝着脸，紧紧地抓住了一个长得贼眉鼠眼的男人的手腕。
<br/>
<br/>        他只是微微施力，这个贼眉鼠眼的男人便露出了痛苦的表情。
<br/>
<br/>        傅夜沉此刻给人的感觉是一种超乎寻常的冷酷，黑眸里布满了血丝，就像嗜血者冷酷无情到有股咄咄逼人的寒气，向周遭散开，令身旁所有人都毛骨悚然。
<br/>
<br/>        包括她童筱颖在内，都看得不寒而栗。
<br/>
<br/>        贼眉鼠眼的男人心悸地挣扎了好一会儿，才抽回自己的手，结果还因为用力过猛，而害他自己往后跌倒在，吓得连滚带爬地跑开了。
<br/>
<br/>        见那个猥琐男跑远，傅夜沉垂眸俯视着一脸惊魂未定的童筱颖，淡漠地开口问道：“去哪？我送你。”
<br/>
<br/>        “想去黄泉路，喝孟婆汤，过奈何桥。大哥，你也送吗？”童筱颖回过神来后，不知道自己哪来的勇气，说出这种胡话来。
<br/>
<br/>        “怎么？再次被阿景抛弃了，想寻短见？”他的嘴角勾起一抹嘲讽的笑。
<br/>
<br/>        “真正抛弃我的人，难道不是大哥你吗？”童筱颖眉心紧蹙地质问，声音里透着些许哽咽。
<br/>
<br/>        “你的耳朵怎么了？”傅夜沉忽然答非所问，因为直到此刻，他才注意到她的两只耳朵上戴了助听器。
<br/>
<br/>        童筱颖心一横，偏过头去，淡淡地回答道：“生了一场病，失去了听力而已。”
<br/>
<br/>        她终究没有告诉他真相，这其实是他傅夜沉一手造成的。
<br/>
<br/>        其实，要怪就怪她自己好了，没及时去就医。
<br/>
<br/>        若是那天晚上，她自己及时去了医院的话，就不会落下这么严重的后遗症。
<br/>
<br/>        “童筱颖，你觉得我犯贱，还是你在犯贱？”他突然冷冷地问。
<br/>
<br/>        童筱颖一时间未反应过来，回头看向他，讷讷地反问：“什么？”
<br/>
<br/>        “看样子，是我在犯贱！”他突然夺走她手中的行李，另一只手抓住了她的手腕，连拖带拽地带着她往后方路边的一辆黑色越野车那里疾步走去。
<br/>
<br/>        他强行塞了她进副驾驶座的位置，自己再绕到车尾将行李放入了后备箱，紧接着，他才回到驾驶座，“砰”一声关上车门便侧头冷眼望着她不知所措的样子。
<br/>
<br/>        童筱颖此刻显得有些束手无策地坐在副驾驶座里，她黯然神伤地低下头去。
<br/>
<br/>        接下来，他要把她送哪里去？
<br/>
<br/>        车子里寂静无声，谁都没有要开口说话的意思，直到他放置在车内的手机铃声突然打破了这一刻所有的寂静。
<br/>
<br/>        童筱颖抿了抿唇，瞥了傅夜沉一眼，弱弱地说：“大哥，你的手机在响……”
<br/>
<br/>        她当他是聋子吗？
<br/>
<br/>        下一秒，她的眼前一黑，黑压压一个人头毫无征兆地袭了过来。
<br/>
<br/>        童筱颖瞬间瞪大了眼睛，还没来得及反应，唇瓣上突然一热，仿佛四片柔嫩相贴，辗转厮磨。
<br/>
<br/>        慌乱间，她抬起手来，本能地想要去推开他，却被他横过来的双手紧紧揽住后背，没来得及推开，反而更加亲密地与他的胸膛贴在了一起。
<br/>
<br/>        狂肆吐纳的气息，绵情蜜意的折磨。
<br/>
<br/>        那两片几近陌生又火热的唇瓣带着不顾一切的狂乱，紧紧地攫住了她一双唇瓣。
<br/>
<br/>        他唇齿间猛地一挑，在她震惊得没有办法回神之际，便肆意地入侵。
<br/>
<br/>        “唔——”童筱颖倒抽了一口冷气，想要往后躲开，眼泪再次在眼眶里打转。
<br/>
<br/>        然而，她只要一退，他则步步紧逼，揽在她后背的大手似乎要将她给揉碎，唇齿里的纠缠几乎深入到她喉管。
