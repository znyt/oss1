　　别人或许不清楚，但慕清霄却知道，手中的残缺地图，并不是什么藏宝图，而是标记净莲妖火所在的地图。
<br/>　　净莲妖火，异火榜上排名第三。
<br/>　　呈乳白色火莲形态，可净化万物的神秘火，有净化万物的特效，任何东西，只要被其沾上丁点。就将会被净化成一片虚无。
<br/>　　甚至于可以以人的情绪为引，进入体内，肉体灵魂斗气都净化为虚无，威力极为恐怖。
<br/>　　这种异火天地间仅此一朵，曾被净莲妖圣收服，后反噬净莲妖圣使其陨落，被封印于妖火空间多次出世，可却始终无人真正的收服。
<br/>　　双月同现，九星一体，天地潮汐，妖火降世。
<br/>　　虽然只有残缺的地图，但慕清霄迟早会吞噬净莲妖火，毕竟他可是任务在身。
<br/>　　将地图收入储存空间中后，慕清霄拿着钥匙来到石桌前，小医仙已经有些迫不及待。
<br/>　　见她一副急不可耐的样子，慕清霄将钥匙插进孔中，但是却拧不动，显然是插错了，随后又换成另一把钥匙。
<br/>　　“咔！”
<br/>　　下一秒，寂静的石室中响起轻微的声音，小医仙美眸中满是期待，呼吸都略微急促起来。
<br/>　　望着弹射而回的金属扣，慕清霄运用灵气包裹住金属盒子，看了小医仙一眼。
<br/>　　“你一个小妮子在外面，万事都需要小心些，别看佣兵团那些佣兵对你唯命是从，但看你的眼神都有些不对劲，我的话你能明白吧？”
<br/>　　闻言，小医仙点了点头，道：“这是两码事吧，快点打开吧。”
<br/>　　慕清霄摇了摇头，将石盒子打开。
<br/>　　没有发现危险，小医仙走到石桌旁，目光投向打开的石盒中，微微一愣后，旋即俏脸欣喜的伸出玉手，从中取出了一卷彩色的古朴卷轴。
<br/>　　不等慕清霄开口询问，小医仙就笑吟吟的说道：“是一本记载如何配制毒药的毒经。”
<br/>　　慕清霄并不意外，身为炼药师的他，自然清楚毒经代表着什么，在斗气大陆上，只有医师才会弄一些毒药来防身，这一类人称为毒师。
<br/>　　当然，毒师在大陆上的地位，比起炼药师要差上许多。
<br/>　　“这东西你不能跟我抢，因为天生属性的缘故，我不能成为炼药师，所以也只能依靠这些东西了。”
<br/>　　说话的同时，小医仙俏脸上浮现一缕黯然，显然，她心中最希望的是成为一名炼药师，而非一名让人又厌又惧地毒师。
<br/>　　慕清霄捏了捏她嫩滑的俏脸，语气认真，道：“既然毒经存在，就有它的用处，善恶归咎于使用它的人，而且你这么漂亮，又怎会用它去做恶。”
<br/>　　望着慕清霄淡淡的笑意，温柔的话语，小医仙一时间竟有些痴了，她是第一次听到这样的话，心间最柔软的地方被触动，一时间眼眶都有些微红起来。
<br/>　　慕清霄有捏了捏她嫩滑的脸颊，过足了手瘾，道：“难道不是吗？”
<br/>　　微不可查的点了点头，小医仙抬起头来，道：“你总是占我便宜。”
<br/>　　慕清霄干咳一声，讪讪的收回手掌，随后将钥匙插进第二个石盒中，石盒开启，在石室内月光石的照射下，其中的东西也一览无遗。
<br/>　　内部依然是摆放着一卷黑色的卷轴，伸手将黑色卷轴自盒子中取了出来，翻开一看，目光停留在卷轴侧面地小字体上。
<br/>　　“不错，是一卷玄阶高级的飞行斗技。”
<br/>　　“飞行斗技，是什么东西？”
<br/>　　生活在小小的青山镇，小医仙同样是没有听说过飞行斗技，顿时疑惑的眨了眨眼，斗技她自然听说过，比如攻击斗技，防御斗技，身法斗技等等，唯独没有听说过飞行斗技。
<br/>　　“顾名思义，飞行斗技自然是能够让人在天空中翱翔。”
<br/>　　听到慕清霄的解释，小医仙一惊，旋即满脸困惑的说道：“在天空中翱翔，不是至少需要斗灵强者，才勉强具备飞行的能力吗？”
<br/>　　在斗气大陆上，想要离地飞行，只有斗灵以上的强者，才有可能进行短距离的飞行。
<br/>　　在实力到达斗王或者斗皇后，便能用外放的斗气在背后凝聚成能量双翼，从而使得他们能够离开大地的束缚，自由自在的翱翔于天际。
<br/>　　而所谓的飞行斗技，便是一种颇为诡异的秘法，这种秘法能够让的修炼者在背后的脉络中，延伸出两条小支脉。
<br/>　　只要小支脉延伸而出，哪怕本人实力没有达到斗王级别，也能够凝化出双翼，进而破空飞行。
<br/>　　飞行是足以让任何人怦然心动的诱惑，或者说是梦想。
<br/>　　因为这个梦想，很多强者，前赴后继的对着那高不可攀的斗王级别努力冲击着，而在这种情况下，能够走捷径的飞行斗技，其整珍贵不言明了。
<br/>　　望着手中的飞行斗技，慕清霄丝毫不感兴趣，直接递给了小医仙，道：“我对斗技不感兴趣，送给你了。”
<br/>　　接过飞行斗技，小医仙小脑袋有些发懵，简直不敢置信，如此珍贵的东西，他竟白白送给自己，偷偷打量了慕清霄一眼，暗道：“真是挺帅的，喜欢他的女子肯定也很多吧？”
<br/>　　“飞行斗技对你来说也很珍贵吧，真的送给我吗？”小医仙再次柔声问道。
<br/>　　“嗯，给你就收下吧。”
<br/>　　小医仙甜甜的将斗技收入囊中，目光移向最后一个盒子，开心道：“最后一个了，时间也不早了，快些打开我们就回去吧。”
<br/>　　慕清霄将钥匙插进盒子中，还没开启，嘴角就牵起一丝古怪的笑意，道：“似乎有人跟进来了。”
<br/>　　“什么？”
<br/>　　闻言，小医仙同样是一惊，望着慕清霄，小脑袋都摇成了拨浪鼓，道：“不可能的，这里就我们两个，我没有叫人过来。”
<br/>　　或许是对慕清霄的印象极好，小医仙都不忍去怀疑他，只能急忙开口给自己辩解。
<br/>　　慕清霄望着她急切的模样，有些被她的单纯所打动，揉了揉她的小脑袋，示意她安静下来。
<br/>　　待小医仙安静下来，美眸直勾勾的望着她，慕清霄道：“我都还没说话你，你怎么急成这样？”
