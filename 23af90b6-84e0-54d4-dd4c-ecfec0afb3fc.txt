　　此时，整个火山周围，气氛都显得无比压抑。
<br/>　　天地动荡，一股股浓郁的灵气不断从四面八方汇聚而来，源源不断的自火山口灌入岩浆旋窝中。
<br/>　　远远望去，整个火山口都被包裹在浓郁的灵气中，景象极其壮观。

<br/>　　小镇口，成百上千的居民，惊愕的望着数千米外的火山。
<br/>　　其上，恐怖的灵气风暴肆虐，隐隐弥漫而出的威压，使他们都变得心惊胆颤，若非天墉城弟子暂未离去，他们早就逃之夭夭。
<br/>　　“你们觉得，火山中的东西，究竟是何等修为，为何十几年来一直没有消息？”
<br/>　　“可能是当初疏忽了吧，毕竟生活在火山中的妖魔鬼怪，实在稀少。”
<br/>　　“如此恐怖的气息，当初我在长老们身上感受过，可能......可能是练虚期！”
<br/>　　在天墉城弟子议论的同时，火山中的恐怖气息迅速扩散，瞬间将千米范围都笼罩其中。
<br/>　　在宛如深渊一般的磅礴气势下，哪怕是天墉城四名金丹期弟子，皆是感到一股深深的无力感，更何况他人。
<br/>　　至于花香镇的居民，皆是由心底蔓延出一股浓浓的恐惧感，仿佛被死亡笼罩，惊恐慌张的尖叫起来。
<br/>　　“仙人们，你们快去捉拿妖魔啊。”
<br/>　　“待会妖魔冲出火山，我们可就惨了，仙人快救救我们吧......”
<br/>　　“仙人，我们都不想死啊！”
<br/>　　站在花香镇前的四名金丹弟子与十名筑基弟子，脸上肌肉皆是狠狠的抽了抽，暗骂一声马卖批。
<br/>　　如此恐怖的气势，天墉城的各堂长老也不过如此吧？
<br/>　　他们若冲上去，跟送死有什么区别，别开玩笑了，你们不想死，我们也不想死呢！

<br/>　　火山上空，灵气肆虐，火山中的岩浆旋窝，旋转速度更加迅速，岩浆浪潮拍打在山壁上，顿时响起一阵阵沉闷的撞击声。
<br/>　　不知何时，黝黑的乌云出现在了山顶上空，原本艳阳高照，碧空如洗的蓝天，渐渐变得昏暗起来。
<br/>　　气候的变化，更是令得天墉城弟子确信，火山中的东西绝对恐怖如斯，简直就像传说中的渡劫景象啊！
<br/>　　“似乎，要出来了......”
<br/>　　火山中的气势迅速攀升，令得天墉城弟子心都提到了嗓子眼上。
<br/>　　“砰！”
<br/>　　伴随着震耳欲聋的轰鸣声，只见火山中一道岩浆柱冲天而起，旋即炸裂开来，犹如一场岩浆雨。
<br/>　　岩浆铺天盖地的喷射而出，转眼间，伴随着嘹亮的凤鸣声，一道庞大的身影携带着磅礴的气息，自火山口冲上云霄。
<br/>　　“嘭嘭嘭...!”
<br/>　　岩浆雨中参杂着碎石自天际惯纵而下，以摧枯拉朽之势，将火山下方的花田砸的满目疮痍。

<br/>　　天墉城的弟子们，在这一刻表现出了他们的正义，纷纷做起了本分的工作。

<br/>　　在成百上千道目光的注视下，遍布天际的烟雾也随风而散，天空再次变得清晰起来。
<br/>　　而在火山上空百米处，一道庞大的身影，映入他们的眼中。
<br/>　　黑色华丽的羽翼伸展而出，乌黑发亮的羽毛，狭长的凤眼中有着俾睨众生之势，翱翔于天际。

<br/>　　望着天空中的黑色身影，以及磅礴的气势，整座花香镇的居民，皆是面露敬畏，同时也一声声低喃。
<br/>　　似乎，都在祈祷神凤的保佑。
<br/>　　至于天墉城的弟子们，都一脸懵逼的望着天际。
<br/>　　存在于传说中的凤凰神鸟，他们怎会没有听说过，但是亲眼看见与道听途说，完全是两回事。
