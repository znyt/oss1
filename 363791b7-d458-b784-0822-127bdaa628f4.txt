　　这阵法，本来便是为了隔绝北邙学院的弟子，直接探索灵泉的，不过他能够隔绝灵力，全是无法隔绝的了楚风眠的剑意。
<br/>　　这一股剑意，悄然之间来到灵泉之中，顿时化为了吞天神蟒。
<br/>　　这一条吞天神蟒，足有百米之长，来到灵泉之中，顿时开始疯狂的吞噬这其中的灵力。
<br/>　　每一口灵力的吞噬，顿时便是无数灵力，被尽数吞入吞天神蟒的口中。
<br/>　　这些灵力，并没有被楚风眠炼化，而是在楚风眠的手心中，开始凝聚。
<br/>　　大约五分钟左右的时间，在楚风眠的手心中，无数灵力凝聚到了极致，便是开始凝实，化为了一块拳头大小的灵力结晶。
<br/>　　这一块拳头大小的灵力结晶，被楚风眠凝聚压缩到了极致。
<br/>　　圣泉中的灵力本来就是无比精粹，所以楚风眠不用耗费力量来提炼，只需要将这其中的灵力，压缩到极致，便是化为了这样一块拳头大小的灵力结晶。
<br/>　　这一块灵力结晶，单单从其中蕴含的灵力来说，足以抵得上十亿九鼎丹的灵力相加。
<br/>　　并且论起价值的话，更是要远超十亿九鼎丹，毕竟这灵力结晶，可是精粹无比，根本不需要耗费力气去炼化。
<br/>　　放在外面去卖，五十亿，一百亿九鼎丹只怕都有的是人要。
<br/>　　“这种灵力结晶，不管是留下以后用来修炼，还是换取财富，都是不错的选择，趁着这个机会，我还要多多凝聚才行！”
<br/>　　楚风眠心中一喜，全身心的催动着吞天神蟒，源源不断的吞噬这其中的灵力。
<br/>　　一块一块的灵力结晶，都落入到了楚风眠的手中，被楚风眠全部装入到了空戒之中。
<br/>　　同时楚风眠身上的力量，也是都在节节攀升，一滴滴远古战龙精血，也在不断的凝聚着。
<br/>　　七日时间。
<br/>　　楚风眠几乎是在毫不停息的凝炼这灵力结晶，足足是两千块灵力结晶，被他凝聚了出来。
<br/>　　这放在外面，也是一大笔的财富了，而且不同于九鼎丹，对生死境以上的武者来说，九鼎丹已经是没有任何的作用了，所以没有多少价值。
<br/>　　但是这灵力结晶的话，就算是对生死境武者，都有着大用，有着作用，便是足以确定他的价值。
<br/>　　“神力境四重。”
<br/>　　楚风眠紧闭的双眼，缓缓睁开，他身上的力量，也开始逐渐的平息了下来。
<br/>　　现在楚风眠的境界，便是一跃踏入到了神力境四重，同时楚风眠身上的远古战龙精血，也达到了惊人的五千滴。
<br/>　　五千滴远古战龙精血，已经是比起进入圣山之前的楚风眠，多了三倍。
<br/>　　论起力量来说，更是不知道强大了多少倍。
<br/>　　这一次楚风眠进入圣山，可以说是有着巨大收获。
<br/>　　“到时间了，可以离开了。”
<br/>　　楚风眠缓缓起身，将洞府内的禁制，全部毁灭之后，便是一下子踏入到了外面。
<br/>　　在洞府之外的东一齐，已经是失去了踪影，看起来已经是被人救走了。
<br/>　　毕竟东一齐出身自武道世家，在这北邙学院中，也很有可能有着他家族中的强者。
<br/>　　不过现在，楚风眠已经是丝毫不惧了，楚风眠感觉到，他现在的实力，只怕是在一些生死境高阶的强者手中，已经是足以自保了。
<br/>　　这种实力，在北邙学院，已经有着几分地位了，起码再也不会是被人拿捏的存在。
<br/>　　有着实力，楚风眠也无需在任何隐忍了，是时候展现出来了。
<br/>　　就在楚风眠思考之际，突然楚风眠感觉到，他似乎是被一股力量盯上了。
<br/>　　这一股力量，并非是武者的力量，其中的浩然，只让楚风眠想到了一点。
<br/>　　天意。
<br/>　　这是天意的力量。
<br/>　　“轰隆！”
<br/>　　天际之上，无数雷云，突然显现，这让楚风眠一下子想到了两个字。
<br/>　　“天劫！”
<br/>　　如今楚风眠踏入神力境，居然是引来了天劫。
<br/>　　这天劫，可是只有武者，踏入到生死境的一刻，因为太过逆天，才会引来天劫。
<br/>　　没想到现在楚风眠，刚刚踏入神力境，居然就引起了天意的注意，要降下天劫来，对付楚风眠。

<br/>　　楚风眠扫视了一圈周围，突然眉头一皱。

<br/>　　这里是北邙学院，一旦是楚风眠再次度过天劫，很有可能便是会引起许多北邙学院强者的注意。

<br/>　　这圣山上的雷云，也随着楚风眠遁入虚空，缓缓的消散了。

<br/>　　黑压压的雷云，一下子遍布整个虚空深处，在其中，一道道电光在雷云中，闪耀着。

<br/>　　楚风眠抬起头，看向空中，心中暗骂一声。
<br/>　　他如今踏入神力境罢了，居然是引来了万雷帝劫。
