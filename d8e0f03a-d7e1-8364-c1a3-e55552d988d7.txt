<br/>　　看着博士额头上诡异的第三只眼睛，刘哔语气平静，双臂软软的搭下来，没有一点反抗。

<br/>　　刘哔将视线重新放到博士身上，语气略微有些嘲讽。
<br/>　　“你闭嘴！”
<br/>　　博士脸色阴沉，松开了刘哔的脖子，随后狠狠一拳打在了他的下巴上。
<br/>　　瞬间，刘哔的身体向上腾飞，撞碎了楼顶的水泥，在半空中伸出一对硕大的黑色翅膀。
<br/>　　漂浮在阳光医院上空，透过刚才被撞出来的大洞，刘哔冷冷的注视着博士。
<br/>　　实验完毕，这所谓的药剂……大概也就这样吧。

<br/>　　阳光医院外的便利超市中，白小飞从货架中拿起一块面包，正欲撕开，外面却传来了一声巨响。
<br/>　　“什么东西，小鹿，快躲起来！”
<br/>　　顾不得手中的食物，白小飞一把抓起身前穿着黄色小鹿童装的孩子，抱着躲进了柜台底下。
<br/>　　在不知道发生了什么的情况下，至少先躲起来是最为明智的选择。

<br/>　　躲在白小飞怀里，小鹿声音奶声奶气的，一边说着，还一边好奇的向外张望。

<br/>　　距离相隔如此支援，白小飞根本看不清这人的相貌，也只能下意识的推断，这是一个实力强劲的尸兄。

<br/>　　事情来的突然，白小飞根本没能有半点反应的机会，瞬间和巨大物体撞在了一起，一股巨大的冲击力袭来，整个人都被撞进了身后的超市里。

<br/>　　竟然，能如此强大！
