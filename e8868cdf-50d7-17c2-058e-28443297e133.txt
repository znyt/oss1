　　“圣光十字阵！”
<br/>　　那红衣大主教一声暴喝，周围教廷的众高手当即纷纷出手。
<br/>　　轰轰轰！
<br/>　　一道白色光芒爆射而出，汇聚在一处之后，凝结成了一张巨网，直接朝下方的叶尘笼罩而去。
<br/>　　“这下就算他有三头六臂，恐怕也挡不住了吧？”
<br/>　　后方的王硕忍不住再次嘀咕了一句。
<br/>　　然而他才刚刚说完这话，就看到了难以置信的一幕场景。
<br/>　　只见，叶尘依旧站在那里，连头都没有抬一下，只是随手一掌朝上方拍出。
<br/>　　轰隆！
<br/>　　一道硕大的金色巨掌冲天而起，竟然瞬间便将数十名教廷高手，联手祭出的圣光大阵，瞬间轰破！
<br/>　　“啊！！”
<br/>　　伴随着一道道惨叫声响起，那数十名教廷高手，几乎同时被打飞了出去，就算那位红衣大主教，也被震的连连倒退，脸色煞白无比。
<br/>　　“什么！”
<br/>　　王硕和韩颖玉两人，顿时再次目瞪口呆。
<br/>　　尤其是王硕，简直无法相信眼前所发生的一切，
<br/>　　“这怎么可能！西方教廷的红衣大主教，个个都至少是神境级别的强者，甚至是人仙！数十人联手，竟然挡不住此人一击？这家伙到底是什么来头？”
<br/>　　就在王硕两人无比震撼之时，那教堂深处陡然传出一道苍老的叹息声，
<br/>　　“唉！”
<br/>　　声音落下，众人便看到，一名身穿金色教袍的白发老者，从教堂深处，踏空而来，轻轻一步迈出，便跨越数米的距离，一转眼的功夫，就已经来到众人跟前。
<br/>　　之前那些被叶尘打伤的教廷高手们，顿时全都精神一震，纷纷激动的高呼起来，
<br/>　　“是老教皇大人！”
<br/>　　“老教皇大人出手，看这小子还如何嚣张！”
<br/>　　“他就等着接受我教廷的惩罚吧！”
<br/>　　……
<br/>　　王硕当即一把拉住韩颖玉的手腕，
<br/>　　“连那位传说中的老教皇都出来了，这小子肯定必死无疑，咱们赶快趁机逃命去吧！”
<br/>　　不料，韩颖玉微微思量了片刻后，却果断摇了摇头，
<br/>　　“再等等，我觉得这少年，应该没有那么容易死掉！”
<br/>　　王硕顿时大急，
<br/>　　“小玉，你想什么呢！西方教廷的这位老教皇，据说是地仙以上的实力，便是咱们华夏的燕真人，都远远不及此人，而且这里又是他的地盘，这小子就算再厉害，还能逆天不成？”
<br/>　　王硕说着，便要强行带着韩颖玉离开。
<br/>　　然而，接下来所发生的一幕，却让所有人大跌眼镜！
<br/>　　只见，那位深不可测的老教皇，出现在众人跟前，竟然冲着那神秘少年一躬身，恭恭敬敬道：
<br/>　　“不知叶仙人驾到，有失远迎，还望恕罪！”
<br/>　　众人看到这一幕，顿时尽皆哗然，
<br/>　　要知道，眼前这位老者，可是西方教廷的真正主人，是无数教徒心中，至高无上的神！
<br/>　　如今竟然对眼前这位神秘少年如此卑躬屈膝，若非亲眼所见，实在令人难以置信。
<br/>　　“这、这怎么可能？那可是西方教廷的皇，整个欧洲大陆最尊贵的人，怎么会……”
<br/>　　王硕两眼瞪得老大，都已经有些怀疑人生了。
<br/>　　天空上，
<br/>　　叶尘冷冷瞥了一眼柯莱特，
<br/>　　“你就是西方教廷的那位老教皇柯莱特？”
<br/>　　柯莱特一双老眼，与叶尘双目相交，顿时浑身一颤，当即颤巍巍从皇座上站了起来，躬下身，用流利的华夏语，无比恭顺道：
<br/>　　“不敢！在叶仙人面前，晚辈岂敢称皇？您喊我名字即可！”
<br/>　　叶尘听到这话，不由得眉头微微一挑，
<br/>　　这个柯莱特倒是挺拿得起，放得下，自知不知叶尘的对手，果断降低自己的姿态，以他数百岁的高龄，居然以晚辈自居，一般人若是自持身份，便也不好意思对他出手了。
<br/>　　可惜，他这点伎俩，又岂能糊弄的住叶尘？
<br/>　　“神界神子卡斯特，来我华夏寻隙滋事，此事和你脱不了干系吧？”
<br/>　　叶尘也懒得废话，直接开口见山道。
<br/>　　柯莱特听到这话，心头顿时一凉，当即陪着笑脸道：
<br/>　　“叶仙人说笑了！您既然知道那卡斯特是神界之神子，那就应该清楚，我在他的面前，不过就是一个小人物，哪里能够左右他的行为？他残害华夏高手，此事我并不知情呐！”
<br/>　　叶尘冷冷一笑，
<br/>　　“我什么都还没说，你又怎么知道，卡斯特残害华夏高手？”
<br/>　　“这……”
<br/>　　柯莱特顿时无言以对。
<br/>　　叶尘的脸色骤然一寒，身上杀气陡然升起，
<br/>　　“看来此事，果然是你在背后设计！你是想为你的弟子报仇，对吧？”
<br/>　　叶尘和神界素来没有瓜葛，不可能会无缘无故对他出手，而叶尘之前曾经在狼王冰岛，斩杀了西方教廷的两名红衣大主教，难保这老教皇不会怀恨在心。
<br/>　　柯莱特听到这话，顿时吓得险些魂飞魄散，连忙摆手道：
<br/>　　“不不！叶仙人千万不要误会，我那两个徒弟不知死活，被您所杀，也怨不得别人，我怎敢对您出手？”
<br/>　　“那卡斯特想要找您的麻烦，我也曾经竭力阻止，无奈他不听我的啊！”
<br/>　　……
<br/>　　众人望着柯莱特，在叶尘面前，唯唯诺诺、卑躬屈膝的模样，一个个面面相觑，甚至都怀疑自己出现了幻觉。
<br/>　　堂堂欧洲大陆的幕后主人，竟然对眼前这个神秘少年，畏惧到如此程度，实在令人匪夷所思。
<br/>　　尤其是那些，向来视柯莱特如神灵的狂热教徒们，全都用双手捂着脑袋，根本无法接受眼前所发生的一切。
<br/>　　唯有跟随在柯莱特身后的那名红衣大主教，清楚叶尘的身份，知道眼前这个看上去不起眼的少年，具备着如何恐怖的实力。
<br/>　　然而，
<br/>　　面对柯莱特近乎乞求般的姿态，叶尘却丝毫不为所动，脸色依旧冰冷，淡淡道：
<br/>　　“柯莱特！你以为你那点小心思，能瞒得住我？”
<br/>　　柯莱特在叶尘的眼神逼视之下，顿时浑身一颤，险些跪倒在地。
<br/>　　叶尘话锋一转，接着又道：
<br/>　　“不过，你若是告诉我神界的入口，我可以考虑饶你一命！如何？”
<br/>　　柯莱特的脸色，顿时越发的难看，一脸苦涩道：
<br/>　　“叶仙人，不是我不说，实在是，我也不知道啊！”
<br/>　　叶尘听到这话，顿时两眼微微一眯，身上的杀气骤然攀升，
<br/>　　“既然如此，那你也就没有活着的价值了！死去吧！”
