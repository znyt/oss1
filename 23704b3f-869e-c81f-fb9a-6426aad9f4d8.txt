
<br/>　　<br/>　　    陈重掐着电话，棱着眼珠子看向前路，停顿了三秒之后，他皱眉说道：“靠边，停车”<br/>　　<br/>　　    “啊？”司机仅仅是稍微迟疑了一下，就很有职业节奏的迅速减速停车停靠路边，打起了双闪。<br/>　　<br/>　　    旁边的老马，诧异的问道：“怎么了？”<br/>　　<br/>　　    陈重摆了摆手，冲着电话说道：“你刚才说的是什么意······”<br/>　　<br/>　　    陈重的一句话还没有说完，他们这辆车前方大概三十几米外的地方，左侧挨着路的这一边，山顶上突然传出几声杂乱的轰隆巨响，一片碎石从山坡上滚了下来，片刻之后就冲到了路面上，直接就把前方的路给堵死了。<br/>　　<br/>　　    车里三人当场懵逼，脑袋里一片空白，从车速和位置上判断的话，如果陈重没有让司机停下车，那他们刚刚开过去的时候，那堆碎石可能直接就砸在车上了，八十码左右的车速在那个撞击力道下，瞬间就能给砸翻滚到右侧的坡道下，后面的陈重和老马都没有系上安全带，那结果可能就只有一个，车毁人亡了。<br/>　　<br/>　　    蜀中这边的高速或者国道，常有山体滑坡或者落实=石的状况发生，毁车死人的事每年都有，这并不新鲜。<br/>　　<br/>　　    电话里的王惊蛰也听到了落实的声音，他深吸了口气问道：“陈先生？”<br/>　　<br/>　　    陈重此时手心上全是汗，他“咕嘟”一声咽了口唾沫，半天之后才回了一句：“我，我在，在呢”<br/>　　<br/>　　    “在那等着，不要动！”王惊蛰直接就把电话给挂了，跟陈臣说道：“你爸出事了，我们走，先过去再说”<br/>　　<br/>　　    陈臣小脸瞬间一片煞白的问道：“他，他怎么了？”<br/>　　<br/>　　    “现在没事，正好赶上了”<br/>　　<br/>　　    这真是命里有坎，但也命不该绝了，今天陈臣要是没来找王惊蛰，或者刚才他没看见对方的面相，可能陈重此时已经滚到山坡下面去了。<br/>　　<br/>　　    很多人都听说过一句话，叫命里有贵人相助，这个贵人确确实实是真实存在的，其实每一个人的命中都会有，甚至有很大的几率是每个人都曾经遇上过这个贵人，只是可惜的是，贵人你碰上了但机会没把握住。<br/>　　<br/>　　    就我说知道的是，我老家那边有对双下岗的夫妻，下岗之后连吃饭都成问题了，后来这对夫妻在一个饭店里打工，男的帮厨女的打扫卫生，饭店老板看他俩干活挺认真日子也苦，就跟这对夫妻说，我饭店早上的时候九点后才开门，你俩要是不怕吃苦的话，就在饭店门口弄个卖早餐的摊子吧，两人一合计觉得自己干也行，然后就早上三点多起床一直干到酒店卖起了馅饼和糖饼，那条路上靠着两所学校和几个单位，早上人流量川流不息的，路过想买早餐的人太多了。<br/>　　<br/>　　    这对夫妻卖早点那时候我才上小学，后来我中学毕业了，前后也不过就五六年的时间吧，这对夫妻直接在我们那开了一家宾馆一家大酒店，其他几个城市还有连锁的店。<br/>　　<br/>　　    他们这就是遇到贵人了，那个饭店老板命中相助了他们一把，如果当时这对夫妻没把他说的话当回事听进去，那可能他们现在已经在饭店里给人打工过着朝不保夕的日子呢。<br/>　　<br/>　　    所以说，每个人的生命中都出现过一次或者几次贵人，只是非常可惜的是，得有九成的人没有意识到而已，白白的错过了机会。<br/>　　<br/>　　    “啪”陈重脸色阴沉的点了根烟，沉沉的吐了口气，老马抹了把头上的冷汗，问道：“怎么回事，你刚才怎么就忽然停车了？真险，差一点那堆石头就砸我们车上了”<br/>　　<br/>　　    陈重晃了晃手里的电话：“刚才小臣打电话给我，说我要有意外发生，然后那个叫王惊蛰年轻人把电话接了过去，告诉我马上靠边停车·····”<br/>　　<br/>　　    老马和司机瞬间呆愣，这是王惊蛰将他们从悬崖边上给拉了回来。<br/>　　<br/>　　    司机咽了口唾沫，问道：“老板，他怎么知道的？”<br/>　　<br/>　　    陈重和老马都没有吭声，他俩估计那个小年轻应该是没有表面看起来的那么简单了，看来是他们看轻了对方。<br/>　　<br/>　　    另外一头，陈重这边中途出事之后，王惊蛰和菜刀文还有陈臣就从山头上下来，上了车后就往出事的地点赶，车开起来后，王惊蛰不时的打量着陈臣的脸蛋，他发现对方左侧脸颊仍旧带着亡夫的面相。<br/>　　<br/>　　    “这一劫还没没有过去？”王惊蛰皱了皱眉，说道：“把你爸的生辰告诉我”<br/>　　<br/>　　    按理来讲，陈重躲过这一把后，陈臣的面相就该恢复如初了，因为大劫过去了，她父亲早逝的面相就该结束了，但现在还没消那就只有一个可能，接下来还得有事。<br/>　　<br/>　　    陈臣报了下陈重的生辰，王惊蛰一手拿着手机一手掐算，片刻后他再次拨通了陈重的手机。<br/>　　<br/>　　    “喂，惊蛰啊？”再次通话，陈重的语气直接就熟络亲密了不少。<br/>　　<br/>　　    “你们现在在哪？”<br/>　　<br/>　　    “下车了，在车旁站着呢······”<br/>　　<br/>　　    王惊蛰掐着手指，快速说道：“那里不行，换个地方，离你们那台车远一点，旁边要是有隔离带的话就翻过去”<br/>　　<br/>　　    这一回，陈重根本就没犹豫了，直接就从路右侧的护栏上翻了下去，随后老马和司机也跟了过来。<br/>　　<br/>　　    陈重刚刚跨过护栏的瞬间，就听见了路后方传来一阵急促的刹车车。<br/>　　<br/>　　    由于出现了山体滑坡落石了，这一条路上的车全都被堵死了，不少车辆都停了下来观望，这么一会工夫大概靠着陈重这边的车堵了能有七八辆左右。<br/>　　<br/>　　    这时候，路后方一辆加长挂车满载着货物开了过来，但由于先前没发现这边被堵住了，等开近了之后挂车司机才发现出了意外，就连忙踩下了刹车。<br/>　　<br/>　　    挂车下面的轮胎传来了一阵刺耳的动静，阵阵焦糊的味道冒出来后，下边就涌出了不少的黑烟，不到几十米的距离一辆超载的大挂车强行制动的话是根本就刹不住车，那辆挂车直接贴着护栏就冲了过来。<br/>　　<br/>　　    一阵火花闪耀！<br/>　　<br/>　　    “咣当”最后面的一辆车被撞了之后，挤压着往前冲去，站在旁边的人全都惊恐的四处逃窜躲闪。<br/>　　<br/>　　    这是碰上连环车祸了，根本就是让人防不胜防。<br/>　　<br/>　　    “咣当”<br/>　　<br/>　　    “咣当”<br/>　　<br/>　　    挂车顶着几辆车一路冲击而下，瞬间就把陈重他们那辆车给怼上了前方的石堆里，整辆车都被挤压的变形了。
