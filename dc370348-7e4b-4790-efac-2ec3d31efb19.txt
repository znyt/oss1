　　第482章
<br/>　　会议结束后，诸人各自散去，为即将到来的战斗做准备。
<br/>　　尉迟离回到他的临时住所，召开了一场小型军师会议。手下的谋士都聚在一起，分析目前的战局，看接下来这场仗究竟该如何打。
<br/>　　讨论进行得如火如荼，尉迟离一直未开口，他坐在帅案后，微微眯着眼睛，脸上时不时浮现一抹笑意。
<br/>　　“将军，是否已经有了计策？”手下谋士问。
<br/>　　“喔，还没有，你们继续商量一下，看如何才能有胜算。”
<br/>　　“将军似乎并不为目前的状况担忧？”
<br/>　　“哈，眼下当然有些困难，不过，程大雷的兵太少了，他想要将我们困死在黄沙城，几乎不可能做到。”尉迟离道：“所以，算不上什么大事，当然，也谈不上容易。”
<br/>　　“可将军似乎已经胜券在握？”
<br/>　　“喔……我不是因为这件事而高兴。”尉迟离似乎已明白手下人困惑的是什么，他眨了眨眼睛，道：“你们不觉得，大殿下他，越来越像一位帝皇了么？”
<br/>　　手下怔住，有参与今天会议的人，回忆着李乐天的表现。曾经的李乐天是个风度翩翩的人，待人谦和有礼，然而，今日的他，除了谦和之外，还展现了勇气。
<br/>　　在潜移默化之下，李乐天都收获了大家的认同。至于他的几位竞争者……李星溃逃扬州，宣告其与皇位无缘，而李行哉么，他在这次战争中的表现，实在没什么值得谈的地方。
<br/>　　如果李乐天能够顺利登上帝位，他的支持者，毫无疑问可以获得巨大的利益。至于崔相，没有投对票的他，注定成为这场赌局的淘汰者。
<br/>　　这种事，注定是几家欢喜几家忧。尉迟离心情有多愉悦，崔相的心情就有多糟糕。现在崔相一有时间就往李行哉处跑，对李行哉嘘寒问暖，生怕他的身体出一丁点状况。毫无疑问，就连李行哉的亲身父亲，对他也没这么好过。
<br/>　　李行哉的身体并无大碍，只是暂时无法下地行动，可崔相的心情实在无法好转。越看李行哉的样子，越觉得他和帝位没多大关系。崔相不禁去想，投资这个废物皇子，自己是不是真的做错了。
<br/>　　黄沙城进入紧急备战状态，在次日黎明准备展开战斗，算起来，这已经是他们进入黄沙城的第三天。
<br/>　　派出两支队伍攻打蛤蟆军的大营，一是凉州牧宋伯康率领的凉州兵，二是青州牧周唤鲸的青州兵，两家各派出一些人，合计一万人。
<br/>　　在战斗开始之前，崔相单独将宋伯康唤到一地。
<br/>　　“大人……”
<br/>　　崔相摆摆手，让宋伯康坐下，问道：“对上程大雷的蛤蟆军，两家兵力差不多，你有多少胜算？”
<br/>　　“这……”
<br/>　　算起来，宋伯康是最了解程大雷的一批人，因为蛤蟆城就在宋伯康控制的凉州境内。然而，因为距离太近，宋伯康也就越了解蛤蟆城的恐怖。
<br/>　　今日这一战，只是试探性攻击。就算在兵力相等的状况下，宋伯康也无多少胜算。
<br/>　　“属下只能尽力而为。”
<br/>　　“不是尽力而为。”崔相摇摇头：“是许败不许胜。”
<br/>　　“呃……这！”宋伯康一惊。
<br/>　　崔相笑了笑，笑容有些苦涩：“大皇子出的风头太多了，也该让品尝一下失败是怎么滋味了。”
<br/>　　宋伯康只是略微顿了顿，便悟到了其中关键。他缓慢而郑重的点点头。
<br/>　　“属下领命。”
<br/>　　城头擂鼓震震，帝国军展开了对蛤蟆军的攻击。青州兵和凉州兵同时出城，杀向蛤蟆军的大营，李乐天等人立在城头观战。
<br/>　　这一战，牵挂着很多人的心。程大雷偷袭了许多人的队伍，就连扬州五万大军也被他们击败。可这些只是听说而已。程大雷所率领的蛤蟆军究竟是否真的不可战胜，今天大家拭目以待。
<br/>　　帝国军磨刀霍霍，蛤蟆军显然也没有闲着，双方展开了猛烈的对冲。城头观战的诸人，亲眼见到了蛤蟆军的打法。
<br/>　　先用火药狂轰乱炸，火药的威力就算再不如何，也足以令帝国军人仰马翻。然后是骑兵冲撞，对着溃兵乱砍乱杀，在帝国高层人物间，他们展现了精妙的骑术，他们的骑术甚至不弱于号称骑兵无敌的戎族。
<br/>　　在骑兵对帝国军形成巨大冲击力后，便是步兵的全面冲锋，连击驽、火雷、梅花阵等展现了超强的威力。
<br/>　　“那是什么！”
<br/>　　“喔喔，那又是什么？”
<br/>　　黄沙城城头发出一声声惊呼，蛤蟆军拿出的武器、阵法令他们心惊胆战。
<br/>　　但这甚至不是蛤蟆军的全部，最令帝国军震惊的，仍是蛤蟆军展现的超强组织能力。
<br/>　　十人一组，百人一队，千人一营……每一名帝国军战士，同时都要面对三到四名蛤蟆军战士的攻击。他们配合默契，消灭一人后，立刻转向另外一人。帝国军兵力总体还是有优势的，但是在局部，帝国军却总是面对以少敌多的局面。
<br/>　　他们像一阵旋风扫过战场，所到之处，帝国军如秋收的麦子一般，成片成片的倒地。
<br/>　　如果说火药、武器、阵法帝国军因为技术限制拿不出来，但眼前这一幕，帝国军便是摸得懂里面的所有关窍，也玩不出来。
<br/>　　帝国兵源一般有：常备军，预备军，征召军，战斗力是从强到弱，数量也是由少便多，绝大部分是临时抽调过来的征召军。他们大部分是普通平民，临到打仗时，发给一件兵器，就挥舞着鞭子赶着他们上阵打仗，有时候武器都不发，用一颗脑袋扛敌人的兵器。
<br/>　　比如蛤蟆城，有三万兵力，其实按照帝国的算法，所有平民都可以拉着上战场，那么蛤蟆城的兵力应该超过十万。
<br/>　　这些临时聚起来的平民，很难说他们会有怎样的战斗力，更不用提什么配合。
<br/>　　但养活一只精锐的常备军，需要的是巨大的财力支撑。打仗打来打去，其实打得还是钱袋子。
<br/>　　如此状况下，帝国军的战败也就是理所应当。李乐天看着毫无抵抗之力的帝国军，无可奈何的选择撤兵。
<br/>(本章完)
