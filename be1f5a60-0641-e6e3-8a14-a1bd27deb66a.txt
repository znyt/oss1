　　“控火兽武魂？萧逸，长空说的可是真的？”剑堂执事立刻问道。
<br/>　　萧逸皱了皱眉头，刚想说些什么。
<br/>　　“不必问了。”顾长空从手中拿出一本手册，递给剑堂执事，道，“这是记录今年考核弟子资料的手册，执事一看便知。”
<br/>　　剑堂执事接过手册，看了两眼，顿时脸色失望起来，“竟真的是控火兽武魂。”
<br/>　　顾长空冷笑一声，道，“我说得没错吧，被誉为全大陆最低等，也是最无用的控火兽武魂。”
<br/>　　“若是让这等庸才进入内门十堂，他日我裂天剑派必将沦为他人笑柄。”
<br/>　　“萧逸，你还有什么资格进我剑堂？”
<br/>　　顾长空直视萧逸，冰冷的话音中，充满了不屑。
<br/>　　另一边，人群中的顾长风得意地笑着。
<br/>　　萧逸没有管这两人，淡淡地问道，“不知执事，萧某是否已经通过考核了？”
<br/>　　“通过了。”剑堂执事回答道。
<br/>　　萧逸继续道，“那三项考核，可有限定要有厉害的武魂才能通过？”
<br/>　　“没有。”剑堂执事摇了摇头。
<br/>　　萧逸淡然道，“那我是不是控火兽，和能不能加入剑堂，有何关系？”
<br/>　　考核中，确实没有对武魂强制要求。
<br/>　　要通过裂天剑派的考核，必须要有足够的天赋和实力。
<br/>　　可是，若武魂品阶不高，根本不可能拥有足够的天赋和实力，也就不存在通过考核了。
<br/>　　这是常识，自然没必要在考核中加入规定。
<br/>　　只不过，萧逸本身就是打破常规的人，任何常识，到了他这里，都会变得让人惊讶和震撼。
<br/>　　剑堂执事的脸色变得有些难看起来，叹了口气，道，“确实没有关系，按照规定，萧逸你能加入剑堂。”
<br/>　　“只不过。”剑堂执事劝道，“控火兽武魂，资质太低了，你就算进入内门，日后的成就也不会大。”
<br/>　　“对你而言，外门与内门，根本没有区别。修炼资源再多，到了你手上，也只会浪费。”
<br/>　　“可惜，你只将玄界碑领悟到七成。否则，若你能领悟到九成、乃至全部领悟的话，拥有一位破玄境武者的全部修炼知识，就算你武魂资质低，日后最低成就也能达到破玄境。”
<br/>　　“这样，你进入内门才算值得。”
<br/>　　“当然了，这只是我的观点。你已然通过考核，是否要加入内门，决定权在你手上。”
<br/>　　萧逸点点头，随即心头一愣。
<br/>　　“七成？谁说我只领悟了七成的？”萧逸心里想着，刚准备说些什么。
<br/>　　顾长空却再次抢先说道，“不，决定权，在执事你手上。”
<br/>　　顾长空看向执事们，道，“第三轮考核，是否通过，全凭执事你判断与认可。”
<br/>　　“你说过了，萧逸便是过了；你说不过，萧逸便不算通过考核。”
<br/>　　顾长空边说着，心头也是一惊，看向萧逸的目光中充满了惊讶和怨毒。
<br/>　　要知道，裂天剑派有史以来，参加此项考核最厉害的，也不过是领悟了八成。
<br/>　　而他自己，当年也不过是领悟了五成左右。
<br/>　　参悟玄界碑，极其困难，毕竟那些修炼知识，都远不是先天境武者能理解的。
<br/>　　所以这个考核，裂天剑派才会如此看重，也是用来查看弟子天赋的最直接手段。
<br/>　　参加这项考核的人，就算能通过，也大多只能领悟一、二成，三成已经非常了不起了。
<br/>　　像今年通过的二十人，几乎都只是领悟了三成罢了，却已经让十位执事非常满意。
<br/>　　能达到四成的，每年也就那么一两个。
<br/>　　能达到五成的，已是凤毛麟角，十年不出一个。也就今年，出了林劲四人。
<br/>　　达到七成，更是数十年没有见过了。
<br/>　　这也是为什么刚才十位执事对萧逸如此期待，对萧逸的表现如此兴奋。
<br/>　　言归正传。
<br/>　　这时，十位执事愣了愣，顿时明白过来顾长空的意思。
<br/>　　剑堂执事说道，“是否认定为通过考核，确实是我们判断的。但按照规定，萧逸也确实通过了。我并不想坏规矩，所以....”
<br/>　　顾长空的脸色阴沉起来，打断道，“执事的意思是要一意孤行了，是否要我亲自回剑堂问问所有弟子的意见，或者让师傅亲自来做个判断。”
<br/>　　“那萧逸资质低下，控火兽武魂拥有者，也被认定为一辈子没有前途的武者。执事，长空还是要劝你三思阿。”
<br/>　　“这...”剑堂执事顿时踌躇起来。
<br/>　　论地位，内门弟子不如内门执事。
<br/>　　但，顾长空除了是剑堂首席外，还是剑堂长老亲传弟子。日后一旦突破破玄境，可是要承袭长老之位的。
<br/>　　所以，每一堂的首席，都是重点培养对象，拥有一定的实权。
<br/>　　剑堂执事也要考虑，若如今得罪顾长空，日后自己要是遭受报复的话，后果能不能承受得住。
<br/>　　“罢了。”剑堂执事脸色一凛，道，“第三轮考核，萧逸失败。”
<br/>　　剑堂执事很清楚，顾长空乃是剑堂长老最青睐与疼爱的弟子，就算今日自己坏了规矩，也不会有人说什么，反而还会支持顾长空。
<br/>　　而且，萧逸一个控火兽武魂拥有者，到哪里都无法说理。
<br/>　　“你...”萧逸原本想说的话，因此而咽了下去。
<br/>　　剑堂执事沉默着，没有看萧逸直视的愤怒目光。
<br/>　　“你什么你。”顾长空冷声道，“如此资质低下，让你浑水摸鱼来到第三个考核，已是大大的不应该。”
<br/>　　“你能成为外门弟子，已是天大的幸运，难道还有资格埋怨不成？”
<br/>　　“咔咔。”萧逸的拳头捏得劈啪作响，但却蓦然一松，笑了起来。
<br/>　　只不过，他的笑意中，带着漠然和冰冷。
<br/>　　“愤怒吗？”顾长空冷声道，“好，我来此的两个目的，如今完成一个。接下来，也该讨个说法了。你夺我顾家灵器，证明你心思贪婪，为人见利忘义。”
<br/>　　“你敲诈顾长风的修炼物品，抢夺同门弟子修炼资源，证明你心术不正。”
<br/>　　“如此卑劣之人，作为剑堂首席，我不得不给你点儿教训。”
<br/>　　顾长空一副大义凛然的样子，实则只是在给萧逸安上莫须有的罪名。
<br/>　　另一边的顾长风满脸得意之色，在一众武者内，更是自觉高人一等。
<br/>　　这边，萧逸脸色不变，沉声道，“你什么意思？”
<br/>　　顾长风冷笑道，“裂天剑派，乃是武道圣地，不容你这样的小人撒野，我自然要给你点儿惩罚。”
<br/>　　话音刚落，顾长风身影一闪。
<br/>　　“嗯？”萧逸一惊，“好快的速度，起码在洞玄七重以上。”
<br/>　　下一秒。
<br/>　　“噗。”萧逸猛地吐出一口鲜血，直接被击飞。
<br/>　　剑堂首席，岂是浪得虚名。
<br/>　　而且，顾长空入内门已有十年，自是修为高深，实力过人。
<br/>　　仅仅一拳，萧逸已然脸色煞白，只觉体内五脏六腑如遭雷击。
<br/>　　顾长空显然是下了死手，要不是自己肉体强悍，刚才那一拳，足够废了自己。
<br/>　　“萧逸。”一旁的林劲四人顿时大惊，随即便是暴怒。
<br/>　　“顾长空，我草你祖宗。”林劲手中玄铁重刀凝聚而出，愤怒至极。
<br/>　　铁牛愤怒地吼了一生，如同愤怒的蛮牛。
<br/>　　柳嫣然不语，但眼中的杀意，却在顷刻间凝聚。
<br/>　　秦飞扬也做好了出手的准备。
<br/>　　四人神色一冷，当即准备出手。
<br/>　　另一边，顾长风高声道，“哥，这四个家伙也没少帮萧逸对付我，不能轻饶。”
<br/>　　“哼。”顾长空冷哼一声，“一群蝼蚁，也想对我出手？找死罢了。”
<br/>　　“俺不管你什么剑堂首席，伤我萧逸兄弟，俺跟你没完。”铁牛瓮声说着，立刻出手。
<br/>　　林劲三人，也瞬间出手。
<br/>　　嗖，忽然，一道快到了极致的身影瞬间在场上飞奔，并挡在了他们四人身前。
<br/>　　一手捉住林劲，一手捉住铁牛。
<br/>　　人影，正是萧逸。
<br/>　　“萧逸。”林劲四人一惊。
<br/>　　萧逸没有说话，脸色异常难看，只是凝重地摇了摇头。
<br/>　　“可是。”林劲想说些什么。
<br/>　　但他却发现，自己被萧逸捉着的手，丝毫反抗不得，甚至动弹半分都做不到。
<br/>　　铁牛同样如此，一向以力量见长的他，完全无法挣脱开萧逸的手。
<br/>　　“好强，这才是萧逸真正的实力吗？”两人心中一惊。
<br/>　　萧逸看着他们，再次凝重地摇了摇头，而后松手，淡淡地走到顾长空面前。
<br/>　　顾长空不屑地冷笑道，“怎么，你还想报复我不成？我刚才只是对你略施小戒，你若...”
<br/>　　啪的一声，萧逸从乾坤袋中扔出一堆东西，放到地面，打断了顾长空的话语。
<br/>　　“这是顾长风的东西，我也不入内门便是。”萧逸淡淡地说罢，而后没再说什么，也没再理会其他人的目光，转身离去。
<br/>　　“哼，算你识趣。”顾长空不屑地笑笑，没有再管萧逸。
<br/>　　裂天剑派，禁止同门相残，他打伤萧逸就罢了，剑堂首席，没人敢说他什么。
<br/>　　但若是当着这么多人的面杀了萧逸，哪怕他是剑堂长老的徒弟，也要受重罚的。
<br/>　　“萧逸。”林劲四人苦涩地说了一声，紧紧地盯着萧逸离开的背影。
<br/>　　那道背影，并不宽阔，但一直给人很宽广，很靠谱的感觉。
<br/>　　可现在，却是那么的萧瑟，那么的无奈。
<br/>　　少年的身影，在落日的余晖下，被拉得颀长，显得很是悲凉。
<br/>　　三项考核，皆是最出色，却无法进入内门，这是何等地不公，对这个孤傲的少年又是何等的打击。
<br/>　　但，这就是这个世界的生存法则，拳头比不过别人，任何道理都是空话，任何不公，都只能咽下。
<br/>(本章完)
