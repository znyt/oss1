　　“这个司令部并没有提及。”
<br/>　　那名force-recon队员摇了摇头，回答道：“司令部那边只是让我们在事态进一步扩大之前立刻返回。刚才的追击过程中造成了平民的伤亡，现在瓦斯汀牢牢的抓住了这一点对比洛多将军进行质问，尽管比洛多将军已经展开了危机公关，可是收效甚微，现在事态的趋势正在变得愈发严重，国内已经动用各种手段封锁消息，不过消息究竟能封锁多久目前还不知晓，司令部要求我们必须在事态不可控之前完成撤离！”
<br/>　　“可是这中间还有中……还有那些人，为什么矛头指向我们？”维尔纳特有些失态的质问。
<br/>　　“因为瓦斯汀跟那些人背后的政府进行了联系，并且获得了支持！”force-recon队员回答道：“而且这件事情的根本就是布克尔特因为私仇绑架了他们国家的公民，他们国家特种部队来这里是为了营救被绑架公民，在瓦斯汀的操作下，他们的入境行为已经完全合法化，反观我们，是因为跟布克尔特的交易才进入这个国家，成了非法入境，如果比洛多为了保全自己，说是我们逼迫他合作，那我们的处境就更加被动了，所以必须在此之前从这里撤离。”
<br/>　　最终，force-recon的入境没有被定义为“非法”，对外说法中，他们成了被布克尔特聘请的雇佣兵组织，反正他们的行动是不会得到美国政府承认，所以被定义成雇佣兵还是恐怖分子都无关紧要。对于五角大楼里那些人而言，他们只要撇清自己的关系就可以了。
<br/>　　战术中队和“龙牙”的四个人，莱昂、布谷、天蛇、玄龟修整了一晚上，瓦斯汀派出车队他们接入了他的府邸，妥善进行安排。
<br/>　　瓦斯汀现在是把罗昊、齐凡他们这些人当成了贵人，如果没有他们这些人在这里闹上一闹，他根本就无缘于这次大选，而现在大选的天平可以说已经是完全向他发生倾斜。
<br/>　　跟瓦斯汀交谈的过程中，罗昊发现，这个手中没有兵权，主要也是跟国家农业发展、经济发展打交道的男人更加圆滑，更加懂得擅于把握机会。
<br/>　　罗昊觉得，跟中饱私囊的巴坦伦和只顾强军的比洛多相比，瓦斯汀这样的人或许更适合做这个国家的总统，因为他切切实实是在为这个国家的农民发展做考虑。
<br/>　　比洛多虽然也是将打击这个国家中的贩毒集团作为目标，可是对于打击掉贩毒集团后，那些被贩毒集团强迫种植古柯的农民应该要怎么安排，比洛多却没有详细的计划。而瓦斯汀却是做出了全面的计划，甚至连那些农民最开始的补助计划，瓦斯汀都做了详细的规划。
<br/>　　第二天一早，危机公关失败的比洛多召开了新闻发布会，自觉大势已去，在新闻发布会上宣布退出此次大选。
<br/>　　原本，不论是在跟巴坦伦竞争，还是在跟比洛多竞争中都不被看好，甚至自己都做好了要辞职退休的瓦斯汀成了最后的大赢家，现在这个国家中真的不会再有人可以跟瓦斯汀有一争之力了。
<br/>　　有了瓦斯汀这个未来总统的帮忙，战术中队的回国就不再是个麻烦。中方直接派出了一架专机来接战术中队回国，莱昂、布谷他们四人跟云凌风联系后，暂时也跟专机回国，然后再返回“龙牙”总部。
<br/>　　返回途中，担心再出什么幺蛾子，由瓦斯汀派出两架战机护送中方专机护送出境，在太平洋上空由中方航母所起飞的战机进行接替。
<br/>　　机场上，罗昊有些意外地看到了比洛多和他的女儿塔尼特，比洛多看到手臂上缠着绷带，脸上贴着创可贴的罗昊，还有其他身上带着伤的战术中队队员，脸上的神情略显尴尬。
<br/>　　不过，最后他还是主动跟罗昊伸出了手，说道：“罗队长，发生这样的事情我很抱歉。”
<br/>　　“你感到抱歉并不是因为你对我们所做的这些事情，而是对你所做的这些事情没有达到你想要的结果。”罗昊毫不客气的戳穿了比洛多的道歉，说道：“每个人都要为自己做出的选择付出代价，你也不例外，并不是任何情况下都能左右逢源的。另外，你的两亿美金军火订单，瓦斯汀先生已经提高到了三亿，我想没有人会跟钱过不去，我们也很愿意跟瓦斯汀先生完成这笔订单。瓦斯汀先生，希望我们之间能合作愉快。”
<br/>　　“罗队长，我们之间的合作一定会非常愉快。”瓦斯汀说道：“而且我听说你们国家的高铁技术非常发达，领先于世界，我觉得我有必要在跟你们国家签订一份高铁援建的合同，当然我们是会按照标准支付费用。”
<br/>　　“瓦斯汀先生，我只是一个军人，如果您有这方面要求的话可以直接跟我们国家相关领导协商，但是我相信他们都会非常愿意跟您和您的国家进行合作。”罗昊在登机前，跟瓦斯汀来了一个大大的拥抱，说道：“愿我们两国友谊长存！”
<br/>　　“友谊长存！”
<br/>　　瓦斯汀笑得无比真诚，也无比开心！
<br/>　　一子错满盘皆落索。
<br/>　　这句话，或许是现在最能形容比洛多的一句话，明明手中抓了一把好牌，可偏偏自己把能赢下的牌送给了对手。
<br/>　　有两国战机交替护送的情况下，战术中队的回国之程并没有受到什么波折。从登上飞机开始，布谷嘴里就在念叨着等到回国后，要去吃涮羊肉和烤鸭。
<br/>　　“真是可惜，涮羊肉不能打包，不然我一定带回去给小布丁吃。”
<br/>　　在战场上，冷静沉稳，面对敌人毫不留情的布谷，这时眼中却是露出溺爱之色，说道：“小布丁还没吃过涮羊肉呢。”
<br/>　　“小布丁是谁？”司徒空眨巴着眼睛对布谷问道。
<br/>　　布谷从口袋中取出一张照片，递给司徒空回答道：“是我妹妹，可爱吧？”
<br/>　　司徒空看到小布丁的照片，立刻跑到布谷面前，抓住了他的手，表情真挚的说道：“大舅子！！”
