　　剑阵之内，萧逸疾速前行。
<br/>　　每多前行一里，攻来的利剑，便会多一把。
<br/>　　待得数百里距离时，已是数百利剑宛若剑气游龙般吞噬而来。
<br/>　　利剑上之气息，或狂猛异常，或阴冷瘆人，或鬼魅疾速…
<br/>　　这一本本利剑，本身，便带着剑碑主人身前的剑道气息。
<br/>　　萧逸前行的速度，渐渐缓慢了下来。
<br/>　　前行，越发困难，但萧逸却越发喜意。
<br/>　　因为，他此刻体内气泉，早已溢出；元力，溢在了小世界之中。
<br/>　　这代表着，单凭元力和修为，他已达圣王境巅峰。
<br/>　　而且，现今才仅仅前行数百里，得了数百份元力。
<br/>　　若是真的走到尽头，他的小世界元力绝对能溢满。
<br/>　　那时，便是他冲击圣皇境之时。
<br/>　　或许，还能有更大的收获也不一定。
<br/>　　萧逸笑了笑，手中连连打出剑气。
<br/>　　不过，恰在此时，身后一道身影疾速而来。
<br/>　　“嗯？”萧逸皱了皱眉，看向身后，竟是桌万宗和子风师弟二人也在快速闯阵。
<br/>　　萧逸没有理会，自顾前行。
<br/>　　身后，两百里距离范围。
<br/>　　桌万宗面露得意之色。
<br/>　　身怀气宗宗主的剑气，实力大幅度飙升。
<br/>　　而子风师弟，身上也同样有一道剑气在护身。
<br/>　　作为林霄独子，他自然也有林霄的护身剑气。
<br/>　　“桌万宗，我劝你不要有什么别的心思。”林子风警告了一声。
<br/>　　“那可由不得你。”桌万宗冷笑一声。
<br/>　　“林子风，我不管你，但我劝你也少管我的闲事。”
<br/>　　话音落下，桌万宗疾速前行。
<br/>　　不过数分钟，二人便追上了萧逸的脚步。
<br/>　　而此时，萧逸已来到八百里距离。
<br/>　　此时，已是八百把利剑自成剑阵而来。
<br/>　　萧逸当先闯阵，也是率先激发这一把把利剑。
<br/>　　故，这八百把利剑，几乎过半都在朝他攻击。
<br/>　　桌万宗、子风师弟二人，一并闯阵，却仅仅是受到不足一半的利剑攻击。
<br/>　　萧逸皱眉看了眼桌万宗，却不理会。
<br/>　　嗖…嗖…嗖…
<br/>　　半个时辰后，萧逸已前行至一千里距离。
<br/>　　一个时辰后，萧逸前行至接近两千里的距离。
<br/>　　至此，他已经得了近两千份元力馈赠。
<br/>　　体内小世界的精纯元力，几乎是以肉眼可见的速度不断上涨着。
<br/>　　而同时，桌万宗亦紧紧跟随在萧逸身后。
<br/>　　虽然他面对且对付的利剑，数量不多，不足萧逸的一半。
<br/>　　但他也是闯阵者，也算是通过了考验。
<br/>　　故他得到的元力，同样是近两千份。
<br/>　　在剑阵的考验判定中，这算是萧逸三人在共同破阵。
<br/>　　自然的，三人得到的元力馈赠是一样的。
<br/>　　前方，萧逸手中剑气纵横，与一把把利剑交锋。
<br/>　　此时，他其实已知晓桌万宗的心思。
<br/>　　桌万宗拿他打头阵，自己跟在身后，不必应付太多压力变能轻松闯阵，且得到好处。
<br/>　　不过也无所谓了，他终究是要闯阵，而且他得到的元力馈赠也不会少分毫。
<br/>　　再者，子风师弟也在身后。
<br/>　　不过，这时，桌万宗却是得意冷笑一声，“萧逸小贼，看什么看？”
<br/>　　“乖乖给本公子在前面打头阵。”
<br/>　　萧逸闻言，皱了皱眉。
<br/>　　桌万宗，将萧逸的脸色看在眼中，越发得意，“皱眉？哈哈哈哈。”
<br/>　　“自己在前面奋力拼杀，本公子轻松就能在后面捞好处，你很不爽吧？”
<br/>　　“不过，你没得选了。”
<br/>　　从当初萧逸上台打扰比拼时，桌万宗心头便已对萧逸愤恨不已。
<br/>　　更别说萧逸还在比武台上一招败他，险些让他贻笑大方。
<br/>　　这一次，他不仅能‘利用’萧逸，还能让萧逸无可奈何，只能心头愤怒。
<br/>　　他自然是感到无比得意的，仿佛心头出了一口恶气般。
<br/>　　“聒噪。”萧逸双眼一眯，前行的速度，陡然大增。
<br/>　　他不介意桌万宗在他身后捞好处，但他没兴趣听这些刺耳之言。
<br/>　　嗖…嗖…嗖…
<br/>　　萧逸手中剑气愈发激扬，前行速度越发疾速。
<br/>　　身后桌万宗脸色微变，“小贼，我可不会让你拉开距离。”
<br/>　　“你跑不掉的。”
<br/>　　若是他跟在萧逸身后，剑阵会自动判定为他与萧逸一并闯阵。
<br/>　　可若是拉开距离太大了，则会被判定为各自闯阵。
<br/>　　剑阵之内，自会有等量的剑影拦他去路。
<br/>　　剑帝碑空间内的手段，玄奥莫测。
<br/>　　这万剑之阵，也非等闲剑阵。
<br/>　　嗖…桌万宗指尖一凝，一道剑气轰然打出。
<br/>　　前方，萧逸疾速前行的身影，猛地一滞。
<br/>　　萧逸的反应，极快。
<br/>　　堪堪转身，数道剑气打出。
<br/>　　不过，他打出的剑气，却瞬间被轰来的剑气打散，剑气直直轰到了他的胸膛上。
<br/>　　瞪…瞪…瞪…
<br/>　　萧逸连退十数步。
<br/>　　别忘了，他周遭还有两千把神兵利器在对他攻击。
<br/>　　萧逸脸色一变，亏得他一身战斗经验丰富，身体一侧，一个翻身，堪堪躲过了攻击。
<br/>　　“气宗宗主的剑气？”萧逸皱眉看了眼身后的桌万宗。
<br/>　　刚才朝他打来的那道剑气，威力绝对达到绝世强者层次。
<br/>　　毫无疑问，那是气宗宗主的剑气。
<br/>　　不过，终归不是气宗宗主亲自打出，而出桌万宗这个圣皇境八重打出。
<br/>　　否则，刚才那道剑气，瞬间就能让萧逸重伤。
<br/>　　“桌万宗，你别逼我杀人。”萧逸身上剑意滔天，挡下了周遭神兵利器，冰冷的双眸，直视桌万宗。
<br/>　　“杀我？就凭你？”桌万宗不屑一笑。
<br/>　　“我告诉你，乖乖给本公子在前面打头阵。”
<br/>　　“否则，我让你再无机会闯阵。”
<br/>　　他现今有气宗宗主的剑气护身，自觉不惧萧逸。
<br/>　　咔…咔…咔…
<br/>　　桌万宗身旁，林子风已然拳头握得劈啪作响。
<br/>　　“桌万宗，我最后警告你一次，收起你的心思。”
<br/>　　林子风杀意凛然。
<br/>　　“林子风，我也最后说一次，少管闲事。”桌万宗冷声道。
<br/>　　“这剑帝碑，是我们剑域的剑帝碑。”
<br/>　　“他萧逸一个外人，能来此得好处，已是他天大福气。”
<br/>　　“现在不过是替我二人打头阵罢了，也值得你这般动怒？”
<br/>　　说罢，桌万宗看向萧逸，“记住，乖乖前行，这第二层空间的好处，你便能分一杯羹。”
<br/>　　“否则，我一道剑气便能让你这一路前行前功尽弃。”
<br/>　　“你找死。”林子风勃然大怒。
<br/>　　“子风，不必与他多言。”远处，萧逸说了一声。
<br/>　　“萧逸师兄。”林子风脸色一急。
<br/>　　萧逸笑笑，“我先行一步，你忘了我有什么了？”
<br/>　　“嗯？”林子风皱了皱眉，半晌，面露喜色，点了点头。
<br/>　　锵…萧逸手中，冷焱剑凭空而现。
<br/>　　轰…几乎是冷焱剑出现的瞬间，周遭神兵利器，顷刻威力大减。
<br/>　　“冷焱剑，万剑折服？”桌万宗脸色一变，“不好，萧逸，你这是作弊。”
