　　越来越多的武者，如今都纷纷的来到了这天羽神域之中来。
<br/>　　这些武者的实力，虽然不如在天羽神域之中，已经是修炼了数年的那些武者。
<br/>　　但是这可依然是一股不可小视的势力。
<br/>　　像是盘纯，他的实力，跟鲜血魔尊接近，但是论起资质，盘纯却是要远远超越鲜血魔尊。
<br/>　　这也就意味着，只要是让盘纯，在这天羽神域之中，得到些奇遇，他的实力便是顿时可以超越鲜血魔尊，甚至是不亚于许多老牌强者。
<br/>　　甚至是小魔主，他的资质还在盘纯之上，只要是得到一些资源，都可以一飞冲天的人物。
<br/>　　而在这天羽神域中，最不缺少的，便是资源。
<br/>　　像是鲜血魔尊这样，资质平平的武者，都足以达到如今的实力，那些真正的天才，只怕都会成长到更加可怕的地步。
<br/>　　“恩？”
<br/>　　小魔主站在一地，一动不动，突然似乎是想到了些什么一样，便是直接带着九魔城的弟子，向着群山走了过去。
<br/>　　盘纯跟那些圣王宗弟子，也是一样，纷纷都是进入到了群山之中。
<br/>　　天羽神域，三年的沉寂中，如今再度变的热闹起来。
<br/>　　不过如今，楚风眠却是没有理会这些人，他如今正是变化成为了杀狂魔尊的样子，跟鲜血魔尊一起，向着九魔城势力的所在走去。
<br/>　　在这天羽神域之中，便是以势力之分，分为了无数势力，像是九魔城便是在其中独树一派。
<br/>　　所有的九魔城弟子，都加入到了其中，在其中一同联手对敌，也会安全许多。
<br/>　　就像是在外面，七大宗门，甚至是各大势力的弟子，也都会选择抱团一样。
<br/>　　像是楚风眠这样，独自一人的倒是少数，楚风眠值得信任的寒月漓，如今在来到归墟天后，也是不知道去了哪里。
<br/>　　归墟天之大，不亚于任何一域，想要从中寻找一个人难度不比大海捞针要轻松多少。
<br/>　　“少主，前方就是我们九魔城弟子的聚集地。”
<br/>　　一直是来到一座山峰面前，鲜血魔尊突然开口道。
<br/>　　就在前方，是一座极为高达的山峰，山峰之上，建有无数宫殿。
<br/>　　这些宫殿看起来，都是极为古老，显然便是曾经云羽天宫，建立起来的宫殿。
<br/>　　临近这宫殿周围，楚风眠便是感觉到，在山峰边缘的一股魔意，在其中，大约有着七八道气息。
<br/>　　其中最为强大的那一道气息，比起楚风眠在归墟天中所见到过的任何武者，都要强大。
<br/>　　这一道气息的主人，应该就是那地魔门弟子中的最强者，锁魂魔尊。
<br/>　　“少主小心，锁魂魔尊一身魔功，已经大成，更是足以刺杀魂魄，他手中的天级灵器九幽锁，据说足以上锁九幽，任何武者被九幽锁锁住，魂魄都会被锁魂魔尊控制。”
<br/>　　鲜血魔尊出言提醒道。
<br/>　　锁魂之术，在无数魔功之中，都足以称得上是极为诡异的一种，防不胜防。
<br/>　　“锁魂之术？”
<br/>　　楚风眠目光一闪，这锁魂之术，他自然也是清楚，厉害无比，一般武者没有防备，几乎是瞬间，都会被对方控制。
<br/>　　不过好在，楚风眠两世为人，他的魂魄比起天下间任何人的魂魄，都要坚韧，这锁魂之术虽然可怕，但是楚风眠只要是防备的好，也可以轻松应对。
<br/>　　“鲜血，杀狂，你们两个怎么回来了？”
<br/>　　就在两人交谈之际，一道身影突然从这山峰中走下来，看到楚风眠跟鲜血魔尊二人，打了个招呼。
<br/>　　这是一尊一身黑衣男子，一双眼睛，深邃到极点，谁跟此人对视，似乎都会被他的眼睛所迷惑一样。
<br/>　　“这是迷瞳魔尊，他的一双眼睛乃是无上魔眼，谁若是跟他对视，便是很有可能会被直接控制。”
<br/>　　鲜血魔尊忌惮的对着楚风眠小声传音道。
<br/>　　“迷瞳，不想死的话，收起你的无上魔眼。”
<br/>　　楚风眠听完这鲜血魔尊的话，顿时冷哼一声道。
<br/>　　这一股霸气，跟杀狂魔尊一模一样。
<br/>　　杀狂魔尊的个性，便是狂傲之际，要是楚风眠表现的太多温和，反倒是不想杀狂魔尊的样子。
<br/>　　“杀狂兄勿怪，这双魔眼有的时候也不受我控制，我这就将他压制下来。”
<br/>　　迷瞳魔尊听到楚风眠的话，目光闪烁一下，顿时陪笑着说道。
<br/>　　他的双眼，逐渐便是化为了平常的样子，这楚风眠的脸色才舒缓了一些。
<br/>　　“鲜血，你不是说你碰到了一个新人吗？怎么？人在哪里？难道是被你杀了？还有疯魔到哪去了？”
<br/>　　迷瞳魔尊看着鲜血魔尊开口道。
<br/>　　“这件事，我正准备给锁魂师兄禀告，那新人的实力，极为强大，以我们三人，居然都不是他的对手，疯魔甚至更是死在了此人的手中。”
<br/>　　鲜血魔尊脸色有些难堪的开口道。
<br/>　　“我要将这个消息，速速禀告给锁魂师兄。”
<br/>　　“什么？疯魔死了？一个新人，居然能够击败你们三人联手？”
<br/>　　迷瞳魔尊脸上露出几分惊讶，似乎是有些不可置信的开口道。
<br/>　　“一个新人，怎么可能有着如此实力？”
<br/>　　迷瞳魔尊眼神一转，开口道。
<br/>　　“疯魔居然都死了，这件事事关重大，我陪你们一起去见锁魂师兄。”
<br/>　　“好。”
<br/>　　鲜血魔尊看了一眼楚风眠的颜色，看到楚风眠暗自点了点头，便是开口道。
<br/>　　“我们一起去。”
<br/>　　说着迷瞳魔尊便再旁边，跟鲜血魔尊跟楚风眠一起，向着山峰上走去。
<br/>　　这山峰上，有着十多座宫殿，每一名九魔城弟子，都在一处宫殿中。
<br/>　　而这其中最为庞大雄伟的宫殿，便是就属于那锁魂魔尊的宫殿。
<br/>　　“锁魂师兄，鲜血，迷瞳，杀狂求见。”
<br/>　　鲜血魔尊在宫殿外，恭敬的开口道。
<br/>　　锁魂师兄的地位，在如今所有九魔城弟子中，都是高高在上的人物，没人敢触犯他的威严。
<br/>　　像是他们九魔城弟子，都是以对方名号相称，唯有在魂锁魔尊面前，必须要加上师兄二字。
<br/>　　“进来吧。”
<br/>　　从宫殿中，突然传来了一道嘶哑的声音，这声音极为刺耳，令人听到都不由的有些头昏脑胀。
