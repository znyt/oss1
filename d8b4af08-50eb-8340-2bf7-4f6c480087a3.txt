长生不老术？
<br/>
<br/>        夏夕绾轻拧了秀眉，这世上哪有什么长生不老，以前古代那些君王为了长生不老，迷信一些道士所谓的仙丸，还乘舟去什么蓬莱仙境，最后不但失败了，还落得个不得善终的下场。
<br/>
<br/>        这个夜老怎么痴迷这个？
<br/>
<br/>        夏夕绾想起刚才跟夜老的对视，夜老的双眼里藏着几分疯狂的暗色，让她十分的不喜，而且她敏感的察觉到夜老对她的冰冷和不善，大概是因为他的孙女夜明珠的原因。
<br/>
<br/>        夏夕绾心里有所警觉，今天的选拔恐怕没有那么顺利，包括她想要的科学院之行，怕也是困难丛丛。
<br/>
<br/>        不过，妈咪的第二个宝箱就在科学院里，她一定要进入科学院拿到宝箱，谁也不能阻挡她的脚步。
<br/>
<br/>        这时前方走来了几个熟人，是厉嫣然和T大的几个女生。
<br/>
<br/>        那几个女生将厉嫣然公主般的簇拥在中间，
<br/>
<br/>        哇，嫣然，你真的要跟陆总订婚了？
<br/>
<br/>        陆总看着好高冷啊，还是嫣然你有办法，将陆总给拿下来了，我们太羡慕你了！
<br/>
<br/>        范恬她们也听到了几个女生叽叽喳喳的声音，范恬顺口嘀咕了一句，“这个厉嫣然要跟陆总订婚了？”
<br/>
<br/>        “应该是的吧，最近帝都都在传陆厉联姻，这个不奇怪。”
<br/>
<br/>        “陆总看着生人勿进，一身禁欲的，没想到眼光不过如此，喜欢厉嫣然这种小白花。”
<br/>
<br/>        这时厉嫣然看到了夏夕绾，她迅速走上前，一脸的骄傲和喜悦，“夏夕绾，你们在谈论什么呢，是不是在谈论我和寒霆哥哥的婚期，没错，我们要订婚了，订婚日子都选好了，就在三天后。”
<br/>
<br/>        三天后？
<br/>
<br/>        夏夕绾觉得自己大病一场已经好全了，但是听到他们的订婚日期她的心还是狠狠的抽痛了一下，她没有想到他们的订婚日子这么急。
<br/>
<br/>        “寒霆哥哥说想快点跟我订婚，正好三天后就是黄道吉日，夏夕绾，我会给你寄邀请函的，欢迎你来参加我和寒霆哥哥的订婚宴哦。”
<br/>
<br/>        厉嫣然看着夏夕绾微微泛白的脸色，心里就觉得畅快淋漓，自从夏夕绾出现在了帝都，她就抢走了她所有耀眼的光芒，让自己变得黯淡失色。
<br/>
<br/>        现在她跟陆寒霆订婚，另一个目的也是想将夏夕绾给打回原形，一个不知道从哪个角落里冒出来的野花野草也配跟她这个天之骄女争？
<br/>
<br/>        夏夕绾并没有表露什么，她澄亮的翦瞳看着厉嫣然，“那就先恭喜你了，恭喜我已经当面说了，所以邀请函就免了。”
<br/>
<br/>        说完，夏夕绾离开。
<br/>
<br/>        看着夏夕绾纤柔的身影消失在了视线里，厉嫣然咬了一下牙，夏夕绾不冷不淡的态度让她感觉自己的一拳都砸在了棉花上。
<br/>
<br/>        不过很快她就勾起了红唇，夏夕绾应该是去准备科学院的选拔了吧，但是好可惜啊，她注定与科学院无缘了。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾强迫自己忘掉陆寒霆和厉嫣然订婚的事情，她早已经知道陆寒霆讨厌她到了极点，那天在帝吧里就是最好的证明，她的陆先生已经不见了，现在只是陆寒霆而已，她没有必要再为了他伤心。
