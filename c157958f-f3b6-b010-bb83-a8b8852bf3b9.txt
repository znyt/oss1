　　如果是在十年前，罗昊现在看到的局面根本不可能会出现。
<br/>　　那个时候，他们国家的大使可没有现在的地位，仅凭一句“我是中方大使馆大使”就能将他人震慑住是不可能的；那个时候，中方驻他国的大使存在感很低，即便是发声抗议，也会被人忽视。
<br/>　　弱国无外交！！
<br/>　　这句话又一次浮现在罗昊的脑海中，或许，今日之国家更符合常任理事国的地位。
<br/>　　“既然没我什么事了，我去看看小熙去。”
<br/>　　罗昊推开车门，跳下汽车。
<br/>　　刚下车，两名行动队队员就拦住了他，两支M4A1战术步枪指着他。
<br/>　　“别拿着东西指着我。”罗昊冷冷的扫了眼那两个行动队队员，说道：“注意你们的态度，你这样的态度会让你们会让你们丢掉性命，让开！”
<br/>　　两个拦在罗昊面前的行动队队员最终还是让开了路，罗昊身上所散发出来的那种慑人的气势，令得两个行动队队员感觉到恐惧，是的，就是恐惧，这种恐惧让两个行动队队员觉得自己面对的不是一个人，而是一头非洲草原上的雄狮，稍有不慎，就会被吃得肉渣都不剩。
<br/>　　罗昊从两名行动队队员面前走过，走到救护车车厢前，打开门迈步走了上去。
<br/>　　云熙从医院离开时挂着葡萄糖，这个时候，葡萄糖也快打完了，如果再晚一点发现的的话，血液就会倒流进点滴袋中。
<br/>　　罗昊替云熙拔了点滴埋针，然后安静的坐在救护车中，安静的陪着这个还在昏迷中的女人。罗昊发现，能像现在这样安静的陪着云熙的时间真的很少，他们更多的时候就连通话的机会都没有。
<br/>　　相对于在救护车车厢中的安静，医院大门口却是剑拔弩张，Guram的第一行动小队和中方大使馆还在对峙着，这种对峙已经持续了超过四个小时。
<br/>　　目前不仅是K国安全局，就连总统府中的卡尔也是一筹莫展，事情已经向他不可控的方向发展，而那个来自洛克希德公司代表的电话再也没有想起过。
<br/>　　中方外交部就发生在K国的事情，连续发表声明，语气也是一次比一次严厉，从最初的告诫到随后警告，每一次声明都像是一记重拳落在K国政府的心头，让他们整个人都跟着打颤。
<br/>　　K国国会此时也是乱成了一锅粥，几个国会大佬都快把自己头顶上仅剩不多的头发都给揪下来了。
<br/>　　“卡尔在干什么！？”
<br/>　　一名国会大佬拍着桌子，怒声道：“他想要干什么！？我们国家现在已经够乱了，他难道还不嫌乱，还要再添一点乱吗！？还有科尔，是谁给他的胆子让他去拦中方大使馆车队的！？”
<br/>　　“现在情况对我们很不利，中方外交部部长已经连续发表声明，从告诫变成了警告，如果我们再不有所表态，情况恐怕会彻底失控。”另一名国会大佬开口说道：“我建议，立刻启动对卡尔总统的弹劾预案，我认为他已经不再适合担任K国总统！”
<br/>　　当天下午三点，K国国会在总统缺席的情况下，全票通过了对总统卡尔的弹劾预案。
<br/>　　总统府邸，卡尔一次又一次的拨打着那个给自己承诺的电话，但是电话始终都没有被接通。
<br/>　　跟K国有着十个小时时差的美国。
<br/>　　一栋有花园、游泳池、总面积达上千平米的豪华别墅。宽敞的游泳池在周围灯光的映照下，碧蓝色的池水显得格外美丽。
<br/>　　别墅二楼面朝游泳池的书房中，书桌上的手机正在震动着，但是坐在书桌背后，背对着书桌的洛菲克却并没有去拿起手机进行接听。
<br/>　　黑暗中，洛菲克那张留着一大蓬胡子的脸一片惨白，这种没有任何血色的惨白在这黑暗中格外慎人，他睁大着眼睛看着外面美丽的游泳池，只是他的眼睛却是没有生命光彩的灰白色。洛菲克西装下那间全手工制作，纽扣采用施华洛世奇水晶打造的衬衫被从他脖子上流出的血染红，血流淌到他脚下，最后被他脚下那块手工地毯给吸收，白色的地毯上一大片深红色的血迹。
<br/>　　在洛菲克的脖子上，插着一张扑克，扑克牌面是黑桃2。
<br/>　　黑暗中，一个人影从沙发上站起来，走到书桌前将还在震动的手机挂断，看着已经没有呼吸的洛菲克，说道：“你以为你有着很大的权利，其实你不过就是一张黑桃2。早知道，你这么会搞事情，当时从飞机上一脚把你踹下去的时候，就不应该给你降落伞。有些时候，机会只有一次，浪费了就没有了。”
<br/>　　蛰龙拿起桌子上那张红桃A，夹在手指间，收回到自己手中那叠扑克中，对着洛菲克的尸体说道：“比大小，赢不赢不是牌说了算，而是我说了算，因为我是庄！”
<br/>　　蛰龙将那张插在洛菲克脖子上的黑桃2拔了出来，在洛菲克身上擦掉血迹，收起扑克，放回口袋，转身离开了书房。
<br/>　　在蛰龙打开书房门的时候，洛菲克书桌上的手机再次震动起来，这次蛰龙没有再去把电话按断，这个电话注定不会有人再接听，等到警察发现洛菲克尸体，他跟卡尔之间的通话记录也会被人发现，但是这些通话记录注定不会被曝光，最终它们会跟这部手机一样被粉碎，然后消失。
<br/>　　……
<br/>　　K国，总统府邸。
<br/>　　卡尔被人带出了总统府邸，他不再属于这里了，他再也没有机会回到这里了。
<br/>　　一步天堂，一步地狱。
<br/>　　很遗憾，卡尔这一步没能迈入天堂，而是跨入了地狱，等待他的将会是终身监禁。
<br/>　　卡尔再被人从总统府邸带走后，科尔也是被免职接受调查，医院中的第一行动小队被勒令撤离，罗昊、赵天宇、云熙三个人跟随大使馆的车队前往中方大使馆，他们会在中方大使馆的安排下，在最短时间里启程回国。
<br/>　　而K国代总统也是立刻召开新闻发布会，发表讲话，态度显得格外端正。他们招惹不起，根本招惹不起的常任理事国。
