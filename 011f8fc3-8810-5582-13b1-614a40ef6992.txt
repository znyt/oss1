　　“老大是天选之子，是叶法善钦定的对付太阴山的人员，只有他才有机会进入太阴山。他们不能废了老大。”
<br/>　　老郭沉吟道：“听你这么说的话，还真是这样，不过，最大的问题是，你没有证据来证明。”
<br/>　　瓜瓜瞪大眼睛看着他，说道：“这个要什么证据啊。我就是证据啊！”
<br/>　　“你是谁？是法术界一代宗师，还是阴司大佬？你说的话，他们为什么要信你，就算他们相信，找你要证据，你有怎么能证明？二当家你太单纯了，你是小师弟的门人，关系在这摆着，他们完全可以说你是想保住小师弟，故意编造的谎言，这不能行的。”
<br/>　　大家仔细一想，也的确是像老郭说的这样，一时间都没了主意。
<br/>　　“那怎么办啊，”瓜瓜失望地说道，“不能让他们剥夺老大的位置啊！”
<br/>　　“废话，茅山掌教如果被开除，这面子丢大了，最重要的是，一旦他被开除，就等于被盖棺定论，证明他犯了错，茅山历史上还没有出过这样的例子，茅山在法术界的威望也会一落千丈，小师弟绝对不允许这种事情发生的。”
<br/>　　老郭深吸一口气，道：“我再去跟小苏商量一下吧，回头去见见几大宗派的掌教，探探他们的意思，劝说一下，目前只能先这样，如果不行再想办法。”
<br/>　　大家也都赞同这样，一起商量起来，老郭在法术界地位太低，跟一些宗师级的说不上话，苏钦章是代理掌教，身份是够的，就是资历太浅，实力也不行，那些老家伙也看不上他，小青小白决定一起去拜访各大山门，美华要留在老郭这里，捉鬼联盟之中，只有他们两个名义上还是阴神，跟他们地位对等，也算能说上几句话。
<br/>　　老郭嘱咐了他们一番，讨论结束后，谢雨晴问了瓜瓜一个问题：“你说什么天命所归，是不是说，少阳注定了有一天会去什么太阴山？”
<br/>　　“这……我不知道，反正我不告诉他。我不想他去。”
<br/>　　小青接过话头说道：“老大如果去，我们一定都跟着，不过，我们也是不希望他去，毕竟基本上就是有去无回了。”
<br/>　　一直没作声的雪琪说道：“也有一种可能，你们想想，如果少阳注定要去斗无极鬼王，那这就是他的使命，就算瓜瓜不说，他早晚也会知道……”
<br/>　　谢雨晴道：“我不管什么使命！”
<br/>　　“听我说完，我是说，既然他使命去……如果这是命运的话，说不定他真能做到呢？命运总不可能让他去白白送死吧？”
<br/>　　大伙怔住。
<br/>　　老郭叹了口气说道：“你这话也有道理，不过三分天命，七分人力，如果连结局也注定了，那还要人干什么，直接演化就是了。反正不管怎么说，我们尽量不要让他知道这件事，将来如果他通过别的途径知道的话，到时候再说。”
<br/>　　谢雨晴道：“如果真有那一天，不管用什么法子，我都会阻止他的。”
<br/>　　讨论结束之后，小青和小白一起离去，打算先去龙虎山见张无生，毕竟这几大门派之中，龙虎山跟茅山的关系最近，张无生跟叶少阳的关系实际还是不错的，可以先去探一下他的口风再说。
<br/>　　美华因为老郭的要求留下来，直接钻进了神龛上的香炉里，虽然她在人间的时候，喜欢像人一样躺在床上休息，不过她对房间和床的要求很好，老郭这里到处脏兮兮的，她看着就不舒服，干脆进神龛里清修去了。
<br/>　　瓜瓜从老郭这里离开之后，又去了青冥界。
<br/>　　空界的界河两岸，因为是空间裂缝的所在，已经成为了尸族和空界联盟的必争之地，在经过几次战斗之后，如今是空界联盟和尸族各自占据了一边，严阵以待。
<br/>　　空界联盟为了方便人间和鬼域来的盟友，又开辟了好几处空间裂缝，瓜瓜走的就是其中一个，出口就在青丘山上。
<br/>　　为了团结在一起，抵抗尸族入侵，青丘山跟阐教势力已经和解，之前星宿海一战就好像没有发生过，现在双方一致对外，青丘山现在管事的是阿紫阿黄和几大护法，号召空界的所有妖族一起，在青丘山成立了一个临时的指挥所，几大妖王坐镇，一起监控前线，调兵遣将。
<br/>　　瓜瓜和小青等捉鬼联盟的成员，在这里享受着最高的待遇，这也是遵照小九的“遗言”，把瓜瓜他们当成了自己人，瓜瓜因为是捉鬼联盟的二号人物，在这里被群妖称为二爷，每一个见到他都恭恭敬敬的，瓜瓜自己也觉得挺装逼，不过内心还是有些落寞的。
<br/>　　他并不想出风头，只想跟在叶少阳身边，当他的小跟班，听他的差遣，什么职位、名分，对他里说都没有意义。
<br/>　　从空间传送点出来，瓜瓜一个人上到山顶，进入了妖宫之中。阿黄听侍从说瓜瓜回来了，立刻出来跟他相见。
<br/>　　“有没有主人消息？”一见面，阿黄立刻就问道。
<br/>　　瓜瓜摇了摇头。
<br/>　　阿黄暗自叹了一声，道：“二爷，后山来了一个神秘人，指明要找你，已经等了很久了。”
<br/>　　“找我？谁呀？”瓜瓜很好奇。
<br/>　　“没通报姓名，不过是从尸族占据的地方过来的，有可能是尸族的人……但是他对我们很友好，只说有事找你面谈，所以我也没为难，让他在后山下的小屋里等着了，你快去吧，不过要小心点。”
<br/>　　瓜瓜道了谢，出了妖宫，一路下山过去，心中充满了疑惑，尸族那边来的人……找自己干什么？
<br/>　　一路来到山脚下，找到了阿黄说的那个小屋，里头黑乎乎的，瓜瓜保持警惕，走了进去。
<br/>　　屋里站着一个人，身穿黑衣，脸上带着面纱，一动不动地站在角落里。瓜瓜好奇地打量过去，刚要开口，对方突然伸手朝他抓过来。
<br/>　　瓜瓜保持着警惕，反应也是极快，抬手阻挡。
<br/>　　他现在的实力，已经很强了，就算与青丘山上那些普通妖王相比也不落下风，因此没有退走，而是选择了出手去挡，这样免得对方一击不成，选择逃走。
