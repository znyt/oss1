　　第171章
<br/>　　大战过后，来不及庆祝，战士需要休息补充体力，为将来的更大的战斗做准备。
<br/>　　充当后勤的老弱妇孺却不能休息，要连夜准备物资，轮班睡觉。
<br/>　　程大雷却也要商议即将发生的事情，与徐神机、刘悲、黄三元等人聊到很晚。
<br/>　　很晚时，程大雷来到苏樱房间，轻叩屋门。
<br/>　　“睡了么？”
<br/>　　小蝶将门打开，气鼓鼓的看了程大雷一眼。程大雷走进屋子后，见苏樱坐在那里，暗自垂泪，见程大雷进来，忙擦干净眼泪。
<br/>　　程大雷摸了摸鼻子，在苏樱对面坐下。
<br/>　　苏樱恢复平静，令小蝶与程大雷倒水，慢慢道：“今天你这一战打得很漂亮。”
<br/>　　“还可以吧。”一场小小的胜利还不足以令程大雷冲昏头脑，况且今日只是个开始，直到现在程大雷还没有与戎族正面交锋的勇气，自然也没这个实力。
<br/>　　“能守得住么？”苏樱问。
<br/>　　“守着试试呗。”程大雷苦笑一声。
<br/>　　“这么晚了，你怎么快来了，是有什么事么？”苏樱看了一眼窗外，发现不是『太晚』，而是天快亮了。
<br/>　　“我来……是想与你谈些事情。”程大雷欲言又止。
<br/>　　“什么？”
<br/>　　“你先出去一下。”程大雷对小蝶道。
<br/>　　“你想干什么？”小蝶向来不是很怕程大雷，但此刻当触及到程大雷的眼神时，却忽地闭上嘴巴。
<br/>　　毕竟是统五千人的一寨之主，今日又沾染上千条人命，程大雷说话做事，早和从前不同。
<br/>　　小蝶离开房间，门在他门后重重关上。
<br/>　　苏樱看了程大雷一眼：“他们还有多久杀过来？”
<br/>　　“大概三两天吧。”程大雷有口无心道：“戎族的行军速度是很快的。”
<br/>　　“那么……你要与我谈什么？”
<br/>　　程大雷深吸口气，吞吞吐吐道：“这个……你是不是愿意……现在……嫁给我？”
<br/>　　苏樱愣了愣，眉头皱起，难以置信道：“现在？”
<br/>　　“我知道啊，是很唐突的，但我的确很需要你嫁给我？”程大雷道：“当然，你如果不愿意，咱们还可以再聊。”
<br/>　　苏樱愣了好大一会儿，慢悠悠道：“你是太紧张的缘故嘛，想……放松一下？”
<br/>　　程大雷满头黑线，真正的原因倒是不好给苏樱说，不过他现在需要苏樱嫁给自己是真的。
<br/>　　山寨要面临戎族大军，任何一丝力量都是弥足珍贵。程大雷能借用最多的力量，自然就是山贼之心的系统。
<br/>　　现在他差一个顶级人物没办法把山寨升到三级，虽然一直在积累恐惧值，但距离百万还差点。
<br/>　　想来想去，就剩下一个抢夺压寨夫人的任务没有完成，关键是系统也没给下一环任务，程大雷不知该怎么做。
<br/>　　但根据常理判断，『抢夺压寨夫人』这个任务，自然是到洞房花烛夜为一个节点，不然难道要生了孩子才算完成，那就有些扯了。
<br/>　　所以程大雷想提前完成这个任务，能换点奖励是点。
<br/>　　而且，程大雷也能感觉到苏樱对自己的情意，毕竟他也不傻，若小男小女情投意合，同床共枕也是顺水推舟，水到渠成的事，程大雷没什么好为难的。
<br/>　　问题就出在，苏樱那边情投了，而程大雷这边意未合。
<br/>　　简单明了一句话：他不喜欢苏樱。
<br/>　　当然，爱美之心人皆有之，程大雷也不是坐怀不乱的柳下惠，苏樱也绝对称得上漂亮标致，而她也的确到了成婚的年纪。
<br/>　　程大雷已没有道理不喜欢苏樱，这时代三妻四妾也算平常，戎族那边更乱，什么父妻子继的事情都有。就算程大雷日后三妻四妾，苏樱顶多吃点闲醋，也不会有什么事。
<br/>　　问题出在程大雷这里。
<br/>　　他来到这个世界，始终觉得和这个世界有层隔膜在，对于这个世界而言，他是冷漠的。
<br/>　　他喜欢苏樱的漂亮，也会对这个世界的美女馋得流口水，却无法投入心里那点真感情。
<br/>　　而现在他想娶苏樱，不是因为喜欢，甚至不是因为贪图对方美色，而是为了打仗！
<br/>　　别人知道怎么说，程大雷不知道，程大雷心里自己觉得，作为一个男人来讲，这样做有点低格。
<br/>　　两人对坐良久，程大雷话也说不太明白，这件事也不好说明白。最后倒是把自己闹了个大红脸。
<br/>　　“好啦，你算我没说！”程大雷咬咬牙站了起来，向门外走去：“你早点歇着吧，这件事过后是活是死还不知道呢。”
<br/>　　“我以为……”身后响起苏樱的声音，令程大雷扶在门上的手顿住了：“……我早已经是你的人了。”
<br/>　　程大雷回头，却见一盏红烛，佳人如玉，那一袭大红袍子，一如嫁衣。
<br/>　　『碧玉破瓜时，郎为情颠倒
<br/>　　感君不羞赧，回身就郎抱』
<br/>　　几番枕上联双玉，共赴多少巫山，赏几片风雨，便难以一一言尽。
<br/>　　门外，小蝶低头听着屋里动静，突然啐了一口，红着脸跑远了。
<br/>　　古时女子成婚前，总有老仆姆妈拿着绣像，指着画上的小人给待嫁的闺女看，毕竟那个时代的资讯不像如今这么发达。不过苏樱少了这一节，她和李如的关系自然也没好到谈这些事情的地步，所以在闺中之事上，她根本一窍不通。
<br/>　　“一窍不通。”程大雷琢磨着这个词，总觉得有几分邪恶。
<br/>　　好在程大雷自己是懂的，在饭老师、小泽老师的指导下，他已积累了很多知识，当然没有实践过。此时，他方才知道纸上得来终觉浅，绝知此事要躬行的深刻含义。
<br/>　　古人诚不我欺呐！
<br/>　　温柔乡是英雄冢，其中滋味太容易消磨英雄意气。
<br/>　　古人又诚不我欺！
<br/>　　现在该看看自己的任务完成没有，程大雷查看脑海中的系统。
<br/>　　嘟，奖励半日风雨。
<br/>　　看着新到系统提示，程大雷陷入深深的沉默中。既没有提示任务完成，也没有提示任务失败，反而来了一条很奇怪的系统奖励。
<br/>　　半日风雨……这奖励算怎么来了，难道半日就有半日风雨，一日就有一日风雨……系统这奖励有些太扯了吧。
<br/>　　“喂……”程大雷推推苏樱的胳膊。
<br/>　　“怎么了？”
<br/>　　唉，纸上得来终觉浅，绝知此事要躬行呐。
<br/>
