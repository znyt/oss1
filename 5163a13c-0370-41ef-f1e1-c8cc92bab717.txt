　　药王宗的万毒窟，向来号称是有进无出，即便是药王宗的高层，都不敢深入其中。
<br/>　　而且，
<br/>　　他们之前明明听到，叶尘和五彩神蛤打斗的声音。
<br/>　　本以为叶尘早已经死在了里面，没想到居然还能活着出来。
<br/>　　另外一边，
<br/>　　叶尘缓步走入众人的视野之中，身后还跟着两人，赫然正是苏蔓和祝小白。
<br/>　　原来，就在刚刚，叶尘第一时间找到了祝小白，便将他也带了过来。
<br/>　　“小白！”
<br/>　　祝明飞看到自己的弟弟，趁着药王宗众人震惊的时候，已经身体一晃，来到三人跟前。
<br/>　　祝小白同样大喜过望，
<br/>　　“哥！”
<br/>　　两兄弟重逢，自然都格外欢喜。
<br/>　　而此时，
<br/>　　云天青也反应了过来，当即吼道：
<br/>　　“祝明飞！看到了吧，此人便是那个世俗界的凶徒，杀了我药王宗数十名弟子，罪大恶极！”
<br/>　　“只要你杀了此人，本宗主便不再追究你和你弟弟的罪过，如何？”
<br/>　　祝明飞眉头一皱，也转而望向叶尘，脸上露出一抹犹疑之色。
<br/>　　凭他和祝小白两人，根本不可能是药王宗高手们的对手。
<br/>　　而且眼前这个少年，虽然看上去极为不简单，可毕竟太年轻了，对付起来应该相对容易一些。
<br/>　　更何况，周围还有其他四大宗门的宗主，云天青应该不至于会出尔反尔。
<br/>　　眼看祝明飞已经动了心思，一旁的祝小白顿时急了，
<br/>　　“哥！你别听他胡说八道！叶大哥是好人，之前就是他救了我！”
<br/>　　“而且你想想，他们药王宗灭掉了我们火神宗，怎么可能还会让我们活着？”
<br/>　　祝明飞听到祝小白这话，顿时心头一震，脸上露出一抹惭愧之色，
<br/>　　“小弟，你说的对！是为兄糊涂了！”
<br/>　　说到这里，祝明飞转而望向叶尘，道：
<br/>　　“这位兄弟，谢谢你救了我弟弟，待会我拼死给你们杀出一条血路，你带着他们赶紧走，我弟弟小白，就拜托你照顾了！”
<br/>　　祝明飞说完，脸上又露出一抹决然之色，显然已经做好了随时拼命的准备。
<br/>　　然而，就在祝明飞，准备对云天青等人出手的时候，一旁的叶尘却已经先他一步，走了出去，
<br/>　　“你照顾好他们两个人，药王宗的这帮杂碎，由我来解决！”
<br/>　　说出这话的同时，叶尘脚下猛然一顿，直接纵身跃起，朝云天青等人冲了过去。
<br/>　　“小兄弟，你别冲动……”
<br/>　　祝明飞顿时大急，想要阻止已经来不及了。
<br/>　　一旁的祝小白道：
<br/>　　“哥，你放心好了，叶大哥很厉害的！”
<br/>　　祝明飞眉头紧皱，摇头叹息道：
<br/>　　“你不懂，药王宗作为五大宗门之一，光是元婴境之上的高手，就有十几位！”
<br/>　　“药王宗宗主云天青，和四大长老，更是元婴境后期的实力，另外还有那位，在天榜排名第三的云千秋，更是深不可测！”
<br/>　　“哎！就凭我们几个，想要对抗整个药王宗，无异于痴人说梦，今日怕是难逃此劫了！”
<br/>　　祝明飞说完这话，祝小白也是小脸煞白，倒是一旁的苏蔓忍不住道：
<br/>　　“喂，你不要长他人志气，灭自己威风嘛，我老弟可是很厉害的！”
<br/>　　祝明飞瞥了一眼苏蔓，发现她只有炼气初期的修为，顿时再次摇了摇头，也懒得再继续向她解释了。
<br/>　　然而就在这时，
<br/>　　天空上忽然一道雷音响起，
<br/>　　“药王宗的废物们！你们一起上吧！我叶狂仙今日，便要踏灭你们药王宗！”
<br/>　　轰！
<br/>　　叶尘这话一出口，众人顿时尽皆哗然，
<br/>　　“这小子！好狂的口气啊！”
<br/>　　“叶狂仙？此人便是在世俗界，公开建立修真门派，广招弟子的那个叶狂仙吗？”
<br/>　　“哼！不过区区一个世俗界的修真者罢了，竟然敢口出如此狂言！”
<br/>　　……
<br/>　　众人议论纷纷，脸色也全都露出嘲讽之色。
<br/>　　毕竟，在蓬莱仙界之人的眼中，世俗界属于低等世界，不过都是一群蝼蚁罢了。
<br/>　　更何况，眼前的这个少年，又是如此的年轻，修为又能高到哪里去？
<br/>　　纵然在世俗界能称王称霸，来到蓬莱仙界，屁都不是！
<br/>　　药王宗的一名长老，更是直接从人群中跳出，直奔叶尘杀了过来，
<br/>　　“狂妄的小子！本长老这便让你见识一下，我药王宗的厉害！”
<br/>　　轰！
<br/>　　那名药王宗长老冲出人群之后，速度快如闪电，瞬间便冲到叶尘跟前，然后狠狠一掌拍落下去。
<br/>　　周围众人之中，除了云天青等人，见识过叶尘的厉害之外，其他人并不知道叶尘的实力，
<br/>　　“这位好像是药王宗的七长老夏侯深吧？”
<br/>　　“据说此人的修为，早已经达到元婴中期，在药王宗里面，实力也能排进前十！”
<br/>　　“此人出手，这小子必死无疑！”
<br/>　　……
<br/>　　众人全都不看好叶尘，唯独云天青等人，反应过来之后，纷纷脸色大变。
<br/>　　“夏侯长老！快回来！”
<br/>　　云天青大吼了一声，可惜为时已晚。
<br/>　　“不自量力！”
<br/>　　叶尘连看都没有看那夏侯长老一眼，直接随手一剑挥出！
<br/>　　轰隆隆！
<br/>　　下方的祝明飞，猛然感受到，传来一道令人心悸的气息，慌忙抬头一看，顿时两眼瞪得老大！
<br/>　　只见此时，
<br/>　　那个比他还要年轻许多的少年，随手一剑挥出，那凌厉无比的剑芒，竟然达到惊人的千米长度！
<br/>　　那恐怖的毁灭气息，仿佛要将整个天穹撕裂一般！
<br/>　　“什么！”
<br/>　　不但祝明飞目瞪口呆，其余四大宗门的高手们，也全都脸色大变。
<br/>　　而那位夏侯长老，更是脸色一下子惨白无比，
<br/>　　直到他生命中的最后一刻，他才忽然明白过来，宗主为何要阻止他出手。
<br/>　　可惜等他明白过来的时候，已经为时晚矣。
<br/>　　轰！
<br/>　　没有任何悬念，叶尘这恐怖的一剑斩落之后，
<br/>　　夏侯长老瞬间被轰成了渣渣，甚至连一丝残骸都没有剩下。
<br/>　　秒杀！
<br/>　　全场顿时一片寂静。
<br/>　　……
<br/>　　“这……他好像连金丹境都没有达到吧？怎么会有如此强大的实力？”
<br/>　　祝明飞已经彻底看傻了眼，简直无法相信眼前所看到的一切。
<br/>　　苏蔓下巴微微一扬，傲然道：
<br/>　　“我说过，我老弟很厉害的，一定可以将这些人打的哭爹喊娘！”
<br/>　　祝明飞先是一滞，继而却再次摇了摇头，低声喃喃道：
<br/>　　“不可能！哪怕他的实力，可以媲美云千秋，可是凭他一人之力，也不可能对抗整个药王宗……”
