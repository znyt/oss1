　　第456章
<br/>　　程大雷看到李行哉拦住杀虎口的出路，顿时气得三尸神暴跳，五灵豪气腾空。什么最狠，就是绝人退路最狠，别人上房你抽梯，别人过河你断桥。
<br/>　　李行哉呀李行哉，我往日待你可是不错，你今天可是把事情做绝了。
<br/>　　李行哉横枪立马，拦在杀虎口前，见到程大雷出现，心中是长长的松了口气。
<br/>　　程大雷这妖孽果然没死，至于起义军死多少人，李行哉才不会关心，人都死绝了他才开心。
<br/>　　幸好，自己换防及时，堵住杀虎口的都是相府的人。可以找机会放程大雷一条出路。当然，眼前人多眼杂，他也不能表现得太明显。
<br/>　　心中转了个念头，李行哉持枪喝道：“呔，姓程的狗贼，本王子在此恭候多时了，看你往哪里走。”
<br/>　　“呔，李行哉，你当真是狼心狗肺，六亲不认，难道不记得当初怎么吃我的喝我的，现在吃干抹净不认账了。”
<br/>　　“姓程的，闭上你的鸟嘴，本殿下什么身份，难道能与你这山贼蛇鼠一窝，来来来，今日就是今日，快快到我枪前送死。”李行哉。
<br/>　　瞧，是当真不念旧情了。程大雷本来还喜欢李行哉这废物王子，但果然，最狠帝王心呐。身上不愧是明帝的血，一脉相称，秉承某些『为国为民，送你去死』之类的屁话。
<br/>　　“李行哉，你吃我一招！”
<br/>　　程大雷握着大旗攻击，李行哉还枪相迎，一招画龙点睛，打的是程大雷软肋。
<br/>　　程大雷没有软肋。
<br/>　　他用斧头，和林少羽的花枪不同，拼一个势沉力猛而已。你只要力气比他大，出招比他快，就能胜过他。但这两点，李行哉几乎不可能做到。
<br/>　　程大雷半程收力，轻轻敲在李行哉的枪杆上，李行哉虎口发麻，双臂差点断开。
<br/>　　李行哉嘴角咧开，气得只想骂娘，压低声音道：“还不快走，更待何时？”
<br/>　　程大雷没有杀李行哉的心，给他一个教训的心情还是有的。此时听到这话不由地一楞：“你要放我走！”
<br/>　　“再不走就走不了啦。”
<br/>　　李行哉推开程大雷，握着枪大吼：“姓程的，快再与我大战三百回合，分个胜负。”
<br/>　　程大雷心灵神会，双方打在一起，当真是枪来旗去，难解难分。
<br/>　　有兵丁在一旁观战，皱眉不解道：“只听说程大雷如何如何了得，今日看来，也不怎么样嘛？”
<br/>　　“嘿，你懂什么。”一旁还有人解释：“高手过招，都是化繁为简，返璞归真，都这样，都这样……”
<br/>　　二人打了一阵，程大雷收旗退开，口中怪叫：“啊呀呀，好生厉害，竟然胜你不得。”
<br/>　　一众士兵目瞪口呆，六皇子殿下竟然打败了程大雷。
<br/>　　李行哉横枪在胸前，轻轻吐出一口气：“吁，我只出了半成力，你就挡不住了。”
<br/>　　“呔，姓程的，你那里走！”
<br/>　　程大雷身后一声大喝，是大皇子李乐天冲了过来。
<br/>　　“快走吧，我大哥脾气可不是很好。”李行哉道。
<br/>　　程大雷点点头，将手中大旗挥动，口中大喝道：“弟兄们，与我冲杀出去。”
<br/>　　两军相交相杀，伤亡十分惨重。
<br/>　　李乐天领着士兵扑来，口中大喝：“六弟，快快拦住他们！”
<br/>　　“大哥放心，大哥放心。”李行哉横马拦在李乐天身前，道：“大哥，我早想到啦，在这里堵死，大哥，我刚才还与程大雷交手，堪堪将他击败。”
<br/>　　“六弟，你快躲开，我要杀了那逆贼。”
<br/>　　“跑不了，跑不了！”李行哉兴奋道：“刚才我们打得可是难解难分，大哥再不来，我就能取他性命。”
<br/>　　“六弟！”
<br/>　　李乐天恶狠狠的看了他一眼，横马冲过，咬住了起义军的尾巴。
<br/>　　一时间杀虎口内人满为患，互相厮杀，李行哉扑了过来，要与程大雷缠斗。
<br/>　　他为程大雷分担大部分压力，周围的士兵顾忌他的存在，冲不到程大雷身边。
<br/>　　程大雷已能看到杀虎口外的天光，再冲杀一阵，便有机会迫出去。
<br/>　　只要能救走林少羽，这次行动虽然说凶险些，但总算是成功了。
<br/>　　“喂，你可以走，他却不能走。”
<br/>　　李行哉一枪拦住了林少羽，林少羽回身相击，虽然这一枪没对他造成威胁，可他却的确是被留下了。
<br/>　　程大雷回头恶狠狠吼了一声：“我不为他，我为何要来此地。”
<br/>　　“我不杀他，如何为父王报仇！”
<br/>　　程大雷咬牙切齿，却是无话可说。杀父之仇不共戴天，自己没资格要李行哉放了林少羽。
<br/>　　也就这一愣神的光景，程大雷和林少羽便被乱军隔开，杀虎口内的帝国军越聚越多，林少羽的情况已经极其不妙。
<br/>　　程大雷一咬牙，回身扑了过来：“你不放他，我自己救他走！”
<br/>　　林少羽一枪将身边的兵丁迫开，虎死雄风在，他就算负伤，普通兵丁也不能迫近他。
<br/>　　“林少羽，你与我留下性命。”李乐天冲了过来。
<br/>　　林少羽与李乐天缠斗在一起，他回身见到程大雷冲来，忽然一咬牙，道：“大当家，你莫要管我，自己去吧。”
<br/>　　回身一枪避开李乐天，目光环视身周，见程大雷没有顾自己的话，仍旧拼命压来。
<br/>　　“大当家，林少羽一条命，受不起您这恩情。”林少羽喝了一声。
<br/>　　程大雷猛然悟到什么，口中大喝：“少羽，你别做傻事！”
<br/>　　但已经来不及了，林少羽哇呀吐出一口血，盯着周围的兵丁恶狠狠道：“你们不是要用我这条命全你们功名么，好，我这条命，便送给你们。”
<br/>　　说罢仰天大笑，此时此刻，李乐天一枪逼来，用的是离龙吐火的架门。林少羽吐出一口血，用自己的胸膛撞了过去。
<br/>　　今日，李家盘龙枪破了林家的破浪枪。
<br/>　　程大雷眼前一黑，直欲从马上跌下来。
<br/>　　只见林少羽惨死之后，身周的帝国军都陷入疯狂中。他们开始哄抢，林少羽的衣物，兵器，尤其是尸体，都成了他们哄抢的对象。
<br/>　　短短片刻，林少羽已经被千刀万剐，尸身分成千断万断。
<br/>　　程大雷大吼一声，从眼底流出两行血泪。
<br/>(本章完)
