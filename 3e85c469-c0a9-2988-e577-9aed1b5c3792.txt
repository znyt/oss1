　　此时此刻，
<br/>　　叶尘也意识到了，云千秋似乎要动用某种底牌，不由得眉头紧皱。
<br/>　　“给我破！”
<br/>　　叶尘抡起紫琼剑，猛然一件劈出，想要破开眼前的毒网牢笼。
<br/>　　刺啦！
<br/>　　毒网并不怎么坚固，被叶尘一剑便看出了一道裂缝。
<br/>　　然而，
<br/>　　还没等叶尘高兴起来呢，那道裂缝竟然却又再次迅速合拢！
<br/>　　滋滋滋~
<br/>　　而且那些网丝，飞速的增加，竟然将紫琼剑也包裹在了其中！
<br/>　　继而，
<br/>　　一道道褐色液体，沿着那些毒网，蔓延到紫琼剑之上，然后朝叶尘侵蚀而来。
<br/>　　“哼！”
<br/>　　叶尘见状，顿时冷哼一声，
<br/>　　“三昧真火！”
<br/>　　轰！
<br/>　　体内的三昧真火，瞬间便自叶尘的掌心升腾出来，沿着紫琼剑蔓延而去。
<br/>　　滋滋滋~
<br/>　　那些褐色毒液，遇到三昧真火，瞬间便焚烧殆尽，而去那些缠绕在紫琼剑上的毒网，也顿时犹如老鼠见到猫一般，纷纷退缩了回去。
<br/>　　之前被毒丝困住的紫琼剑，也终于恢复了自由。
<br/>　　“咦！”
<br/>　　云千秋看到这一幕，口中顿时发出一道惊呼，
<br/>　　“三昧真火！小子，你还真是让人意外啊！”
<br/>　　“可惜，今天就算是三昧真火，也救不了你！”
<br/>　　说完这话，云千秋双手猛然一合，双眼先是缓缓闭上，继而又猛然一瞪，口中一声暴喝，
<br/>　　“呔！”
<br/>　　轰！
<br/>　　云千秋的眉心之中，顿时发出一团黑芒，直奔叶尘爆射而去。
<br/>　　叶尘见状，脸色也顿时大变，
<br/>　　“居然是神魂之毒！”
<br/>　　所谓的神魂之毒，是施术者以牺牲自己的元神为代价，通过各种秘术和毒物的培养，所修炼出来的一种毒功。
<br/>　　此毒术和寻常毒素不同，必须以神魂之力为载体，通过神魂攻击的方式，来攻击对方。
<br/>　　而相应的，被攻击也只能以神魂力量来阻挡，普通的物理攻击无效。
<br/>　　因为是神魂之力，三昧真火同样也会失去作用。
<br/>　　轰！
<br/>　　叶尘不及多想，眉心之中也当即发出一道白芒，和那道黑芒相撞在了一处。
<br/>　　两人的神魂之力交织在一起，叶尘的神魂之力，瞬间便受到了侵蚀。
<br/>　　“喝！”
<br/>　　云千秋再次一声低喝，开始继续释放痛苦之源。
<br/>　　滋滋滋~~
<br/>　　一眨眼的功夫，整个巨网之中，已经完全被毒力所占领。
<br/>　　叶尘虽然竭力抵抗，可是那毒力无孔不入，而且在眼下这个封闭空间之中，连躲都没有地方去躲。
<br/>　　“啊！！”
<br/>　　叶尘一个不慎，被一道毒力所侵蚀，顿时发出一声痛苦的嘶吼。
<br/>　　云千秋见状，顿时大喜过望，当即又催动更多的痛苦之源，朝叶尘侵蚀而去。
<br/>　　轰！
<br/>　　被痛苦之源所侵蚀之后，叶尘只感觉脑袋之中，顿时犹如要爆炸了一般！
<br/>　　恍恍惚惚之间，前世所经历的种种痛苦，犹如潮水一般，自他的记忆深处翻涌而出。
<br/>　　“啊！！”
<br/>　　叶尘再次一声狂吼，双眸几乎瞬间变成了赤红色，头发更是根根倒立了起来！
<br/>　　“叶无伤！穆天华！天狼星主！清虚道尊！琼华仙帝！……”
<br/>　　那些前世让叶尘刻骨铭心的仇人，顿时开始一个个，全都浮现在叶尘的脑海之中……
<br/>　　轰！
<br/>　　满腔的仇恨和痛苦，转化成了滔天的杀意！
<br/>　　就连外面围观的众人，也忽然感觉到一股难以想象的寒意，仿佛一下子坠入了冰窖之中。
<br/>　　“什么！”
<br/>　　看到这一幕，就连云千秋也吓了一大跳。
<br/>　　在他想来，对方不过就是十几岁的少年，所经历过的痛苦，应该十分有限，所以他一出手便没有留情，势要将叶尘心中的痛苦，完完全全的激发出来。
<br/>　　可是他怎么也没有想到，被痛苦之源激发出怒火的叶尘，竟然会如此的可怕！
<br/>　　而更可怕的是，寻常人在如此巨大的痛苦打击之下，只怕精神早就已经崩溃了。
<br/>　　而叶尘则不然，他的意志力，竟然也惊人的强悍，丝毫不像是一个少年，倒像是活了几百岁的老怪物！
<br/>　　“不好！根据先辈们留下的经验，痛苦之源若是无法击溃对方的精神，反而会让对方坠入魔道，实力大增！”
<br/>　　云千秋望着那道气息犹如地狱修罗般的少年，猛然想到了一件事，顿时脸色剧变。
<br/>　　“不行！必须阻止他！”
<br/>　　就在云千秋准备收回痛苦之源的时候，一直处于癫狂状态下的叶尘，猛然抬起头来，赤红的双眸，死死盯着云千秋，声音竟然也变得无比沙哑，
<br/>　　“我要你们——统统全部去死！”
<br/>　　轰！
<br/>　　已经处于暴走状态下的叶尘，抡起手中的紫琼剑，便直接朝云千秋冲了过去。
<br/>　　紫琼剑上，竟然也隐隐出现了一道妖异的血芒！
<br/>　　云千秋顿时大骇，慌忙双手一合，
<br/>　　漫天毒网当即聚集在一处，挡在了叶尘的面前。
<br/>　　彭！
<br/>　　叶尘一剑斩落下来，那些毒网瞬间便被斩为两半。
<br/>　　而那毒网牢笼，也被叶尘这一剑，轻松斩破。

<br/>　　叶尘身体犹如鬼魅，缩地成寸跨越数千米的距离，再次来到云千秋的跟前。
<br/>　　“什么！”
<br/>　　云千秋见叶尘一下子变得如此强悍，老脸都要吓绿了，慌忙从乾坤镯中，祭出了一堆法宝和傀儡。
<br/>　　嘭嘭嘭！
<br/>　　然而这些法宝傀儡，在叶尘的面前，却犹如西瓜白菜一般，根本不堪一击。
<br/>　　不过，等最后一剑法宝，被叶尘斩破之后，剑芒的威力也变弱了许多。

<br/>　　周围围观众人，顿时一个个全部目瞪口呆。
<br/>　　他们忽然震惊的发现，眼前这个世俗界的神秘小子，简直就像是打不死的蟑螂，而且越打越强！

<br/>　　“快！用护宗大阵！”
<br/>　　下方上千名药王宗弟子闻言，当即纷纷出手，众人的功力通过大阵汇聚在一处，竟然化成了一个长达上千米的青色巨掌，朝叶尘狠狠抓来。

<br/>　　几乎已经快要入魔的叶尘，嘴角忽然微微一扬，赤红的双眸中，透着一抹浓浓的邪恶之色，一字一顿开口道：
<br/>　　“今天，你们谁也别想活着离开！”
