钱教授哼了一声，“听说你现在可是一个名人，名气比那些当红明星都要大，而且嫁给了陆氏集团的总裁，一跃成了豪门少夫人，我在圣黎院就听说了你的大名，今天特意过来看看你的。”
<br/>
<br/>        像钱教授这样搞医学的是不会看八卦娱乐新闻的，一定是夏妍妍在钱教授面前说了她很多的坏话，夏夕绾澄亮的翦瞳坦诚的看着钱教授，没有说话。
<br/>
<br/>        周萍咳嗽了一声，“夏夕绾，你来给钱教授倒杯茶吧。”
<br/>
<br/>        “好。”夏夕绾走上前，给钱教授倒茶。
<br/>
<br/>        钱教授看着夏夕绾，“你今年多大了？”
<br/>
<br/>        夏夕绾，“刚过20岁的生日。”
<br/>
<br/>        “20岁？好人家20岁的女孩子还在读书深造，你怎么不读书了，你学的什么专业，师从何人？”
<br/>
<br/>        一般20岁的女孩子的确在读大学，像叶翎现在虽然在娱乐圈，但她还是北电的一名大二学生。
<br/>
<br/>        这时夏妍妍迅速笑道，“导师，绾绾她只是高中毕业，并没有什么学历文凭专业之类的，更没有老师…”
<br/>
<br/>        其实在医学界，是很讲究老师这一块的，得高望重的老师就是自己的门面和招牌，像夏妍妍就师从钱教授，钱教授一来，她整个人容光焕发。
<br/>
<br/>        “什么？”钱教授一皱眉，迅速看向了周萍，“周主任，夏夕绾只有高中毕业，这样的学历你们都能录取了，什么时候枢密的门槛变得这么低了？”
<br/>
<br/>        周萍跟钱教授一样，都十分看重一个人的学历文凭还有从医经验，但夏夕绾一个都没有，背景不清不白的。
<br/>
<br/>        周萍有点尴尬的笑了笑，她总不能说夏夕绾是我们校长空降的吧，而且校长对这个夏夕绾可狗腿子了，还自荐成夏夕绾的老师！
<br/>
<br/>        “钱教授，喝茶喝茶。”周萍岔开话题。
<br/>
<br/>        钱教授却十分不满，“夏夕绾，一个女孩子一定要注重自己的品行，学历和修养，而不是整天搞些歪门邪道，乌烟瘴气的东西，妍妍是我的得意门生，她在医学上十分有天赋，才气斐然，你可能一辈子都够不上妍妍的高度，但你也不能自暴自弃，这样自甘堕落。”
<br/>
<br/>        钱教授是一个文人，骂起人来都是文绉绉的，她对夏夕绾的鄙视轻蔑和讨厌已经跃然纸上了。
<br/>
<br/>        夏妍妍很得意，前些日子她还一脸病态，今天已经唇红齿白了，她骄傲的守在钱教授的身边，居高临下的看着夏夕绾。
<br/>
<br/>        就算夏夕绾再厉害又如何，不过还是一个空架子，一个医学废才怎么跟她这个医学天才斗呢？
<br/>
<br/>        等她将夏夕绾的名声都搞臭了，到时陆寒霆自然会厌弃她，跟她离婚的！
<br/>
<br/>        夏夕绾给钱教授倒了一杯茶，然后不卑不亢的笑道，“钱教授，我受教了，以后我会努力的。”
<br/>
<br/>        钱教授和夏妍妍都觉得拳头砸在了棉花上，这个夏夕绾软硬不吃，滑的很。
<br/>
<br/>        “周主任，我今天过来是想向枢密借一本大罗心经的，我听说大罗心经就在你们的图书馆里，不如让夏夕绾现在去找给我吧？”
