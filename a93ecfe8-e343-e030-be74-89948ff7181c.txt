　　第291章
<br/>　　程大雷能够挣开套索，挣不开这漫天的黑布，像铁牛陷进泥沼里，浑身力气撕扯不开。
<br/>　　程大雷拔出了剑，只要给他一个呼吸的时间，他就有把握斩开黑幕，逃出生天。而一旦脱开这黑布……比快剑，程大雷还没有怯过谁。
<br/>　　咚！
<br/>　　一根烧火棍敲在程大雷脑袋上，程大雷身子晃了三晃，噗通一声跌倒在地。
<br/>　　黑布扯开，露出樊梨花一张俏脸，几个手下站在周围。看着地上死狗一般的程大雷，樊梨花嘴角翘起，在他屁股上踹了一脚。
<br/>　　“绑了，带走。”
<br/>　　高飞豹几个还守在外面，歇马等着程大雷回来。
<br/>　　“这么久了，大当家怎么还不回来，该不会出什么事吧？”有人问。
<br/>　　“大当家那么大本事，能有什么事。”高飞豹道：“估计是拉肚子吧。”
<br/>　　“可这去的时间也忒长了吧。”
<br/>　　“你放心啦，要说做山贼，大当家是这一行的祖宗。你没听他刚才说，当山贼的还能被别人抢了。”
<br/>　　当高飞豹走进山林时，发现林子里空空荡荡，已经看不见程大雷的身影。
<br/>　　“队长，队长，你看哪里。”
<br/>　　手下指着一棵大树，高飞豹目光看过去，只见树上用匕首扎着一张纸条。
<br/>　　上面六个字：落玉寨，樊梨花。
<br/>　　“哎呀我的妈。”高飞豹突然大叫一声：“大当家真的被贼劫了。”
<br/>　　……
<br/>　　等程大雷睁开眼睛时，看到的是绿油油的草地。他挣了两下，这才发现自己被搭在马上，身体被浸湿的麻绳紧紧缠着，强撑着抬起头来，看到马上一张脸笑盈盈的看着自己。
<br/>　　“嗨，又见面了。”樊梨花的拳头重重砸在他的腰上。
<br/>　　程大雷脑袋又垂了下去，心里一句买买提，脸上则扯出一个笑容。
<br/>　　“我还当是谁，原来是樊家姑娘，好久不见，就算你想我了，说上一声，我还不是早请早到，晚请晚到，何必用这个方式请我来。”
<br/>　　“狗贼，死到临头，你还敢口花花，我真要把你的心挖出来，看看你心里都有什么。”樊梨花擒住程大雷，此刻正是得意，听到程大雷这话也不生气。
<br/>　　“还能有什么，心里装得可不都是樊家姑娘你。”程大雷笑着：“不如樊家姑娘帮我翻个身，我好好看你一……”
<br/>　　一柄刀押在程大雷肩膀上，程大雷识趣的闭上嘴巴。
<br/>　　再往前行，程大雷看到马进入一山谷，谷口石碑上写着三个大字：落玉寨。
<br/>　　进入山谷，眼前顿时豁然开朗，山谷内一片翠绿草原，几匹马悠哉悠哉的吃草。
<br/>　　程大雷却是没想到，琴川还有如此一片洞天，与其说这是山寨，倒不如说这是一座马场。
<br/>　　有女子在湖边洗衣，看见樊梨花时微笑着致意，见到程大雷时也不怒也不怯，笑盈盈的看着他。程大雷如果不是伸不出手来，真像给她们好好打个招呼。
<br/>　　此刻樊梨花翻身下马，程大雷也被丢了下来，差点将程大雷脖子砸断。
<br/>　　程大雷身上绑了一圈圈绳子，此刻被人抬着往前走。正这个时候，一大汉快步过来，道：“寨主，你真把这贼人擒了回来，这狗贼要如何处置？”
<br/>　　“我已经给他手下留了信，让他的人拿银子来赎人。”
<br/>　　“这样是不是太便宜他了，不如把他交给莫将军，莫将军绝不会亏待我们。”大汉道。
<br/>　　“他手下的本事都不弱，真如果把他交给姓莫的，我们山寨挡得住么？”樊梨花哼了一声。
<br/>　　程大雷暗暗点头，看来这樊梨花也是有些见识的，并不是胸大无脑。
<br/>　　“喔，原来樊家姑娘不舍得杀我，我就说么，我们两个不打不成交，床头吵架床尾交，一见钟情，一世至尊，一念永……啊！”
<br/>　　樊梨花一巴掌拍在程大雷下巴上，心中恨恨，刚才的话不该让程大雷听见的。
<br/>　　程大雷吐出一口吐沫，妈的，流血了。
<br/>　　他被压到落玉寨的聚义厅，只见樊梨花高坐第一把交椅，两侧有男有女，有老有少，想来应该是落玉寨的几位当家主事之人。
<br/>　　至于程大雷自己，则是像一根柱子一样杵在大厅中央。
<br/>　　“呔，狗贼，见到我们寨主还不跪下。”刚才见过面的那汉子道，程大雷多看了他一眼，知道他的名字叫做阴万良。
<br/>　　立刻就有几个人过来，要压程大雷跪下。樊梨花摆摆手，道：“他也是条好汉，莫要辱他。”
<br/>　　程大雷看了樊梨花一眼，递给她一个『我就知道你疼我』的眼神，樊梨花心头泛起燥火，真的很想一刀杀了程大雷。
<br/>　　“姓程的，没想到吧，你也会落到我手中。”樊梨花微微倾身，趾高气扬的看着程大雷。
<br/>　　“哎，这有什么没想到的，第一次见到你我便在你的眼里陷落。你的眼睛就像一汪清澈的泉水，我好像溺死在你其中。”程大雷用上了咏叹调。
<br/>　　樊梨花一张脸腾地红了，倒不是单单因为程大雷这句话，而是程大雷这话是当着山寨几位主事说的。
<br/>　　“来人，掌嘴！”樊梨花一张脸从趾高气扬，变成满脸寒霜。
<br/>　　程大雷被押下去关了起来，关他的笼子就在山谷的湖边，周围窝着几匹马。
<br/>　　程大雷揉着发胀的脸颊，心中郁郁，早知道自己就不占嘴上的便宜了。要是自己一见到樊梨花就高举双手投降，说愿意与她一起共建美好家园，齐心协力将山寨带到正确的道路上……也不知她信不信。
<br/>　　反正，徐神机会带人来救自己的，不过就是早几日，晚几日的事情而已。所以程大雷知道自己并没有生命危险，无非就是受两日苦，他心中也并不在意。
<br/>　　问题的关键在于，这樊梨花明明是自己召唤出来，该是自己的人才对，怎么会把自己抓起来。
<br/>　　很显然，这是触发支线任务了啊，只有完成任务才能得到樊梨花……的忠心，当然不是得到她这个人啦。
<br/>　　只不过这次的支线任务明显玩得有些大啊，程大雷枕着木笼的门，嗅着马粪的味道，脑海里悠悠的想着。
<br/>　　系统已经给出任务了，关键是该如何破局呢。
