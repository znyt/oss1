>　　    林羡鱼瞬间无言。<br/>　　<br/>　　    丁武又低下脑袋，再低头的时候，两行泪水从眼中滑落掉在了地上。<br/>　　<br/>　　    林羡鱼忽然觉得，一个会哭的男人，也许没有那么可怕。<br/>　　<br/>　　    然后，她就对丁武放下了包裹的很严实的戒备心，尽管他的手上不久之前还沾染着一个人的鲜血。<br/>　　<br/>　　    有的时候，女人的心弦被拨动其实很简单，不一定非得是钻石，豪车和驴包才可以。<br/>　　<br/>　　    有的时候，可能一个男人最柔软的一面就会瞬间击垮女人心里的堡垒。<br/>　　<br/>　　    丁武的两面性是很让人迷惑的，前一刻他毫不犹豫的用一把刀子捅进了一个人的胸口，鲜血喷了他一身都没有皱下眉头。<br/>　　<br/>　　    但下一刻，丁武不过是讲了个曾经的故事，就忽然之间泪两行了。<br/>　　<br/>　　    一个矛盾的男人？<br/>　　<br/>　　    厢房里，王惊蛰喷着吐沫星子，手指着桌面掷地有声的说道：“我跟你们说，别往我身上扣屎盆子啊，你们说的东西我真没有·······”<br/>　　<br/>　　    “聊天归聊天，解决问题归解决问题，别动不动就碰瓷啊·······”
