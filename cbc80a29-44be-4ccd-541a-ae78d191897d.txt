　　“轰！”
<br/>　　只见虚空一震，波纹散开，一阵轰动，伏羲借助河图、洛书，终于推演出天机之道，轰的一声，有八卦图于虚空之中显现。
<br/>　　巨大的八卦图，有阴阳五行，有天、地、山、水、风、火、雷、泽，有日月星辰，乾坤风云，俱在其中。它包罗万象，蕴含大千。道韵流转之间，显现于虚空，遮天蔽日般的浩大八卦图，将天上的昊阳都比了下去。
<br/>　　“先天八卦者，乾、坤、坎、震、离、兑、艮、潩，乾为天，坤为地，震为雷，坎为水，艮为山，潩为风，离为火，兑为泽。包含天地万物，分裂八方，中绘阴阳，转换天地。”
<br/>　　“转轮皇者，此时不破，更待何时！”
<br/>　　只听伏羲一声大喝，他全身一震，轰隆隆间，无尽灵气汇聚而来，道韵缭绕，道则飞扬，先天八卦图悬浮虚空，缓缓旋转，一股天地浩荡的大道显现，大气磅礴。
<br/>　　紧接着，一道缩小无数倍的八卦图，从他体内飞了出来，那是伏羲的转轮——八卦轮！
<br/>　　“轰隆隆！”
<br/>　　九天之上，随着伏羲的突破，无尽雷云汇聚而来，却是修士突破时必须要渡的天劫。成则鱼跃龙门，败则一朝身死。
<br/>　　“去！”
<br/>　　只听伏羲一声大喝，虚空之中的巨大先天八卦图瞬间飞上九天，将那浩大劫雷包围起来，生生的给磨灭了。
<br/>　　八卦横空，光辉照耀天地之中，接着，一道浩荡的声音传遍天下，响彻云霄：
<br/>　　“一朝轮回前尘尽，两世伏羲天地变！
<br/>　　太极两仪阴阳道，五行八卦易天途！”
<br/>　　伏羲一朝悟道先天八卦，异象横空，遮天蔽日，惊天动地，突破转轮皇者境，气势磅礴，席卷乾坤，撼动天穹。
<br/>　　巨大的动静，一时之间，引起了人族，和三界各路大神们的注意，霎时间，无数神念横空，降临虚空。
<br/>　　“嗯？伏羲突破了转轮皇者境，可以让他接任人族共主之位，管理人族了。”人族之中，众多人族高层略带笑意道。
<br/>　　人族之中，武祖玄天早有规定，想要成为人族共主，必须要将修为修炼到转轮皇者境，否则就算贡献再大，也不能成为人族共主。
<br/>　　伏羲之前的贡献也早已达到要求，但他的修为不到，所以一直都是部落的首领，并没有成为整个人族的首领，虽然他的名声很大，但终究不是人族共主。
<br/>　　如今，伏羲一朝顿悟，成就皇者，却是可以执掌人族了。
<br/>　　“来人，传人族各首领前来人祖殿，商议人族大事。”一道威严的声音从首阳山最高峰上的武宫之上传了出来。
<br/>　　“是，武祖！”
<br/>　　“伏羲要证道天皇了，人族又要多一位大能？”大神通者们神念颤动，看着河边的伏羲，惊声道。
<br/>　　“天皇大道，人族大帝之位，不远矣！”娲皇宫高台之上，女娲看着伏羲，脸上露出甜美的笑容，那笑容很美，美不胜收，天地为之失色，可惜无人有缘见到。
<br/>　　除了女娲之外，其他圣人，三清，接引，准提都是眉头一挑，看向伏羲，神色各有不同，有兴奋，有淡然，有嫉妒，有……
<br/>　　“见过两位道友！”伏羲睁开眼睛后，看向虚空中的玄都与龟灵，拱手拜道。
<br/>　　“见过道友，恭喜道友突破转轮皇者，证道天皇不远矣。”
<br/>　　玄都同样拱手回道，他还是比较羡慕伏羲这等人的，修炼数百近千年，便能突破准圣，而自己却已经修炼上万年了，还在大罗巅峰徘徊着，想要突破，还差一点。气运浓厚，就是好，做什么都是顺，天地同助之。可惜，不是什么人都有这样的资格。
<br/>　　“见过道友。”
<br/>　　龟灵圣母则是脸色比较复杂的看着伏羲，毕竟之前还是师徒关系，虽说这样的师徒关系之中带着许多功利算计，不太纯，但毕竟是师徒，如今徒弟都超过了自己，她的心情不复杂才怪。
<br/>　　同时，看到伏羲这幅态度，她便猜测是伏羲觉醒了前世的记忆，伏羲的前世，可是妖族羲皇，准圣后期高手，死的时候她才刚刚出世呢，人家能叫你一声道友，那就是看得起你了。
<br/>　　其实，她猜错了，伏羲还没有觉醒洪荒伏羲的记忆，他一直带着的是神州伏羲的记忆，他一直都明白龟灵来收他做徒弟是怎样的目的，不过，之前他实力弱小，一直没有多说，如今自己已经突破了准圣，当然不会再称呼龟灵老师了。
<br/>　　三人一番交谈后，龟灵依旧随着伏羲回到人族，毕竟伏羲还没有证道天皇，她的任务就不算完。而玄都则是去洪荒其他人族部落探亲去了，毕竟，他之前也是人族，如今也算是衣锦还乡了。
<br/>　　伏羲回到人族，立即就被人族高层叫了过去，一番商议，伏羲继任人族共主，统领整个人族部落，如之前人族四祖一般，只不过如今人族四祖已经功成身退，隐世修炼了，就相当于人族的守护者之类的人物。
<br/>　　伏羲继位人族共主，声望达到顶峰，在人族无人不知，无人不晓。他开始用自己的手段整合人族，为人族编订制度，制定人族纲常，修订礼仪，带领人族走向鼎盛。
<br/>　　同时，伏羲也将自己所悟的先天八卦，传给一些天资聪慧的人族，毕竟，他创先天八卦的初衷，便是为人族趋利避害，断定吉凶祸福，演算天机未来。若是不传给人族，就他一人，又怎么能够拯救整个人族？
<br/>　　伏羲继位第五年，他娶了一位妻子，那妻子便是他的妹妹女娲。人族女娲这些年一直陪着伏羲，帮助人族制定许多制度。
<br/>　　造笙簧，制礼乐；定婚姻，配男女。
<br/>　　便是人族女娲将人族的婚姻制度建立，此乃天地人三婚之中的人婚，自此之后，人族有了婚姻。
<br/>　　混沌天外，娲皇宫中。
<br/>　　女娲看着人族女娲露出幸福的笑容，她自己脸上也随着露出笑容，只是，这个笑容多少有些辛酸苦辣。
<br/>　　她知道自己不能一直陪着哥哥伏羲，也欠了哥哥许多东西，于是她便斩去自己对哥哥伏羲的情感之身，随着伏羲一同轮回。如今轮回归来，伏羲与那道情感之身，也结为了夫妻，这也算是一个美满的结局了。
<br/>　　如今的自己，也算是放下了心中的执念，一心苦悟大道了。
<br/>　　其他人不明白她为什么这么做，也只有她自己明白。如今人族那道轮回的感情之身，也被她斩去了与她的联系，成为一个独立的人。
<br/>　　伏羲不再是之前的伏羲，女娲也不再是之前的女娲，之前的伏羲，已经死了，他一直在圣人女娲的心中，而如今的伏羲，却也不再是之前那个她的哥哥，妖族伏羲了。
<br/>　　这一刻，女娲好似明白了什么，又仿佛有更多的不明白。她的内心仿佛打破了禁锢，心境在这一刻提升了许多，好似少了一分牵盼，多了一分领悟。
<br/>　　紧接着，一道恐怖的威压升起，女娲突破了，成了圣人三重天的圣人。
<br/>　　昆仑山，元始若有所感，抬头看了一眼娲皇宫方向，又看了看人族之中的女娲，好似明白了什么。不过，他没有更多的关注女娲，而是将目光投向一个人族部落山顶的一个小孩。那小孩只有三四岁的模样，但脸上却充满坚毅。
<br/>　　元始对这个小孩的兴趣好似超过女娲，对于他来说，女娲突破到三重天，在他眼里还是不够看，而这个小孩却不同，因为小孩的名字，他注定不平凡，
<br/>　　这个小孩姓姜，号烈山氏，属于姜姓部落，他父亲乃是部落首领，为帝魁。
